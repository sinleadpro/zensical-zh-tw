# 設定電子票券

如何於後台建立並設定電子票券商品。
{ .subtitle }

[:lucide-lock:{ title="適用方案" }](conventions.md#適用方案) | PLUS / 企業
{ .doc-badge }

![](docs/assets/images/ec-e-ticket-hero.zh-tw.png){ .hero-page }

## 建立電子票券商品

### 操作路徑

CYBERBIZ 管理後台 > 商品 > 電子票券列表 > 新增票券

### 設定項目

1. 設定基本資訊（商品名稱、上架狀態）
2. 上傳票券圖片
3. 設定票種名稱與售價

> 電子票券不支援一般商品的款式規格（如顏色、尺寸），系統會以「票種名稱」作為唯一款式單位。

1. 登入 CYBERBIZ 管理後台，前往 **商品 > 電子票券列表 > 新增票券**，進入 **新增票券** 頁面。
2. 依序填寫相關資訊：基本設定、票券圖片、款式管理。 

	![](docs/assets/images/ec-e-ticket-type.zh-tw.png)

3. 點擊 **儲存**，套用變更。
> 電子票券商品將無法像一般商品一樣編輯款式規格（顏色、尺寸）。系統會自動預設一組名為 *票種名稱* 的款式規格。  

	!!! example "票種範例"
		- 活動票券：如 *早鳥票*、*團體票* 等，以區分不同票種。
		- 商品兌換券：如 *灌籃高手兌換組*、*七龍珠兌換組* 等，以分類不同的商品兌換模組。

4. 新增的電子票券會出現在 **電子票券列表** 頁面。


## 訂單與付款限制

- 電子票券需獨立結帳
- 僅支援信用卡一次付清


## 購買電子票券（顧客流程）

1. 顧客於前台選擇電子票券。
> 電子票券商品頁面會顯示票券注意事項，提醒顧客使用辦法。

	![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定05.png){ .screenshot }

2. 若購物車同時包含一般商品，系統將要求拆分結帳。

	![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定06.png){ .screenshot }

3. 勾選 **同意電子票券條款**。
> 消費者需勾選 *同意電子票券條款* 選項，才能進行結帳。（購物車內的 **配送時段** 可忽略）。

	![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定07.png){ .screenshot }

4. 完成結帳。

### 查看電子票券
顧客完成付款後，可在以下兩個地方查看電子票券 QR Code（核銷碼）。

- 我的帳戶：我的帳戶 > 電子票券訂單查詢
- Email：系統寄送 QR Code 至會員信箱

=== "電子票券訂單查詢"

	1. 網頁前台點選 **我的帳戶 > 電子票券訂單查詢**。    
	2. 點擊 **顯示票券 QR code**，即可獲得 QR Code。
	    
	<div class="grid cards" markdown>
	
	- ![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定08.png)
	- ![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定09.png)
	
	</div>
	
=== "電子票券確認信"

	!!! note "需設定會員電子郵件"
	    為確保電子票券能成功寄送至會員信箱，請先確認 *顧客註冊設定* 中的「電子郵件」已設為必填。若會員未提供 Email，系統將無法寄送 QR Code。
	
	1. 登入 CYBERBIZ 後台，前往 **管理中心 > 顧客註冊設定**。  
	2.  **電子郵件**：請勾選「必填」。若會員未填寫 Email，系統將無法寄送 QR Code。
	
	 ![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券設定39.png){ .screenshot }
	
	![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定10.png){ title="信箱畫面範例" }

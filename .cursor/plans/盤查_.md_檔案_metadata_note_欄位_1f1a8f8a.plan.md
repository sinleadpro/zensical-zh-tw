1. 撰寫並執行 Python 腳本，遍歷 `docs` 資料夾下所有 `.md` 檔案。
2. 解析每個檔案的 YAML frontmatter，檢查 `note` 或 `notes` 欄位。
3. 篩選出內容不為空的檔案，記錄其編號、檔案名稱、檔案路徑及 note 內容。
4. 將結果儲存為 `note_stats.csv`。
5. 在回覆中以 Markdown 表格呈現統計結果。
6. 依據資料夾（ec, pos, wms, storepal, resources）進行階段性檢查以確保無遺漏。

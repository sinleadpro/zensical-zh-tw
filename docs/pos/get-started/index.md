## 建置前準備

=== "官網預設置"

    <div class="grid cards" markdown>

    - :lucide-plug:{ .lg }
    __①付款__
    POS系統費用已付款

    - :lucide-plug:{ .lg }
    [__②帳號權限管理__]()
    官網商品是否已建置，並確認每隻商品皆有填寫SKU碼（大量補填SKU教學）。

    - :lucide-plug:{ .lg }
    __③付款__
    確認官網商品庫存，當庫存未設定為∞時將會歸0。

    - :lucide-plug:{ .lg }
    [__④帳號權限管理__]()
    向開店顧問提出開通需求，開通後後台及出現「POS功能」。

    </div>

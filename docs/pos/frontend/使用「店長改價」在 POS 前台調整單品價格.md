# 使用「店長改價」在 POS 前台調整單品價格

在 POS 前台針對單一商品調整價格，並設定改價原因與權限管理。
{ .subtitle }

{ .doc-badge }

{ .hero-page }

## 店長改價說明

**店長改價** 是門市人員針對「單一商品品項」提供的彈性折扣方式。當該商品不適用現有的行銷活動，或有特殊議價需求時，即可使用此功能進行價格調整。

以下為 POS 前台店長改價的詳細說明與操作教學：

## 前台操作步驟

1. **進入結帳頁面：** 在 POS 前台將商品加入購物車後，點擊欲調整價格的 **單一商品項目**。
2. **開啟改價功能：** 點選後會展開該商品的折扣明細與「店長改價」欄位。
3. **選擇折價形式：** 您可以依據需求選擇以下三種方式之一來調整售價：
    - **折扣固定金額：** 例如直接減去 100 元。
    - **最終售價：** 直接輸入最後要賣給消費者的價格。
    - **百分比（折數）：** 例如輸入 90 代表打 9 折。
4. **設定改價備註：** 為了便於後續對帳，必須輸入改價原因。系統會自動記錄最新的 5 筆備註供您快速選取。
5. **確認完成：** 按下「確認」後，該商品的價格即會變動，隨後即可繼續完成後續收款流程。

## 權限設定教學

「店長改價」功能預設僅開放給具備 **「店長」** 身分的帳號。若要讓「店員」也能執行改價，必須由管理員於後台下放權限：

- **後台路徑：** **POS 功能 > POS 權限管理**。
- **設定流程：**
    1. 點選「設定身分權限」。
    2. 選擇欲開啟功能的員工身分（如店員）。
    3. 勾選並開啟 **店長改價** 功能並儲存。

!!! note "若未於權限管理設定，亦可在「修改 POS 設定」中設定 **POS 權限密碼**，店員結帳時輸入該密碼即可使用折價功能。"

## 常見問題

??? quote "為什麼我在單品項目找不到「店長改價」的欄位？"

	這通常有兩個原因：

	1. **權限不足**：請確認目前登入的帳號具備「店長」權限，或已在後台開啟該身分的改價功能。
	2. **組合商品限制**：若該商品屬於「固定組合商品」的一部分，系統可能不支援針對單一子項目改價，建議確認該品項是否為獨立商品。

??? quote "如果該商品已經有「全店折扣」或「滿額優惠」，還能使用店長改價嗎？"

	可以。店長改價屬於「手動單品折扣」，其優先級通常最高。系統會先計算店長改價後的金額，再依序套用其他的行銷活動。但請注意，頻繁疊加折扣可能會影響您的毛利分析。

??? quote "改價後的金額如果是負數或零，系統會允許結帳嗎？"

	系統不允許最終售價為「負數」。若您輸入的折扣金額大於商品原價，系統通常會跳出提示，或將售價自動限制為 $0 元（視系統版本設定而定）。

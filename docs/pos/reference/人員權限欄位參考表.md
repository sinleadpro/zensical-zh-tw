# 人員權限欄位參考表
本參考表收錄 POS 系統支援的權限項目，管理員可依此對照各勾選項目的前台操作權限與後台管理範圍，實現精確的門市分權控管。

## 權限類別與影響範圍

### 1. 收銀與訂單管理

此類權限直接影響前台結帳流程與資金安全。

| 權限名稱 | 前台操作範圍 | 後台管理範圍 |
| :--- | :--- | :--- |
| **前台結帳** | 核心收銀：掃碼、收款、掛單 | 新增會員、查詢會員消費紀錄 |
| **會員修改** | 修改會員姓名、手機、Email 等資料 |  |
| **店長改價** | 針對「單品」手動調整售價 |  |
| **整筆訂單折扣** | 針對「整筆訂單」套用折扣 |  |
| **開啟訂單** | 瀏覽並繼續執行「掛單」中的訂單 |  |
| **關帳** | 執行門市日結作業，列印日結單 | 瀏覽歷史關帳紀錄 |
| **零用金設定** | 設定初始預留金、零用金支出紀錄 |  |
| **訂單查詢** | 搜尋歷史訂單明細與發票資訊 | 瀏覽後台訂單列表 |
| **訂單取消/換貨** | 執行退貨、換貨與作廢發票作業 |  |
| **訂單結案** | 將未付款或異常訂單強制結案 |  |

### 2. 門市人員管理

此類權限涉及顧客資料維護與店員日常作業。

| 權限名稱 | 前台操作範圍 | 後台管理範圍 |
| :--- | :--- | :--- |
| **上下班打卡** |  | 瀏覽自店打卡紀錄 |
| **員工列表** | 查看同店之店員名冊 | 瀏覽自店員工基本資料 |
| **排班** | 查看並設定門市人員排班表 | 瀏覽並管理後台排班系統 |
| **公告系統** |  | 瀏覽並修改 POS 系統公告 |

### 3. 庫存與數據報表

此類權限管控貨物異動與營收數據瀏覽權限。

| 權限名稱 | 前台操作範圍 | 後台管理範圍 |
| :--- | :--- | :--- |
| **調倉單** | 執行店間庫存移轉 | 瀏覽/審核調倉紀錄 |
| **進/出倉單** | 執行入庫驗收、報廢或退貨給供應商 | 瀏覽/管理進出貨單 |
| **POS 盤點** | 執行門市實體商品盤點作業 |  |
| **庫存調整盤點** | 手動修正系統上的帳面庫存數量 | 瀏覽盤點紀錄 |
| **庫存報表** | 即時查詢本店及跨店剩餘庫存 | 下載後台庫存統計表 |
| **業績報表** | 查看門市業績總額 | 下載 POS 業績統計報表 |
| **圖表分析** | 查看營收趨勢、時段客流等分析圖表 | 瀏覽後台數據儀表板 |

### 4. 系統設定與進階工具

此類權限僅建議開放給高層管理或技術人員。

| 權限名稱 | 前台操作範圍 | 後台管理範圍 |
| :--- | :--- | :--- |
| **POS 設定** |  | 修改門市店名、地址、發票與錢箱密碼等核心設定 |
| **電子標籤費用** |  | 瀏覽與設定電子標籤系統整合相關費用 |
| **切換離線模式** | 當網路不穩時，強制進入或離開離線結帳狀態 | | 
| **同步失敗紀錄** | 查看並重新嘗試上傳離線期間失敗的數據 | 


## 角色權限設定模板

您可以根據人員職責，快速建立身分：

=== "店長 / 負責人"

    **核心定位**：全權負責、帳務核對、系統配置。

    **安全責任**：負責保管「店長授權密碼」，防止未經授權的改價與開啟錢箱。

    - [x] **全選所有權限**
    

=== "正職店員"

    **核心定位**：流暢結帳、基本維護、庫存異動。

    - [x] 前台結帳、整筆訂單折扣
    - [x] 會員註冊/查詢/修改
    - [x] 庫存報表、庫存查詢
    - [x] 進/出/調倉單
    - [ ] 店長改價（建議設為需輸入店長密碼）
    - [ ] POS 設定

=== "工讀生 / 兼職"

    **核心定位**：純結帳作業、會員註冊。

    - [x] 前台結帳
    - [x] 會員註冊/查詢
    - [ ] 庫存盤點與調整
    - [ ] 訂單取消/換貨
    - [ ] 業績與圖表分析



## 資安最佳實務建議

1.  **瀏覽即操作**：在 POS 系統中，「可瀏覽」往往等同於「具備操作權限」。例如賦予「員工列表」瀏覽權限的人員，通常也能查看該店所有員工的代表色與基本資料。
2.  **改價授權分離**：即使將「店長改價」勾選給正職員工，也請在「POS 設定」中啟用密碼保護。這樣員工執行改價時仍需向店長索取密碼，確保每筆折扣都有被追蹤。
3.  **定期清理帳號**：每當人員異動，請立即執行「停權」，並重新檢視「指派使用者權限」清單。



## 常見問題

??? quote "為什麼勾選了「庫存查詢」，店員還是看不到其他店的庫存？"
    請確認該員工帳號是否有被限制「僅能瀏覽特定商店」。若員工僅被指派至 A 店，即使有權限，也只能查詢 A 店及其相關連動門市。

??? quote "可以設定某個身分「只能看報表不能結帳」嗎？"
    可以。只需在該身分中取消勾選「前台結帳」，僅勾選「圖表分析」與「訂單查詢」即可。這通常適用於財務或行政人員。

??? quote "可以讓工讀生只能「進貨」不能「調貨」嗎？"
    可以。只需在權限列表中僅勾選「進/出倉單」，並取消勾選「調倉單」即可。

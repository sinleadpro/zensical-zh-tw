# 平板裝置使用說明
商家除了使用桌上型電腦，亦可搭配 iPad 或 Android 平板打造行動化結帳櫃檯。本文說明平板裝置的規格建議與使用限制。
{ .subtitle }

[:lucide-tag:{ title="適用方案" }](../../resources/conventions#適用方案) | 進階 PLUS / 高手 PLUS / 企業
{ .doc-badge }

!!! tip "應用情境"
    - **行動收銀**：在大型賣場或展場中，店員可手持平板在排隊人群中直接完成結帳。
    - **櫃檯美化**：使用 iPad 搭配簡約支架，提升門市的時尚感與專業形象。
    - **空間優化**：針對空間狹小的櫃位（如百貨美食街、快閃店），平板電腦是省空間的首選。


## 裝置規格建議

為確保 POS 前台運作流暢，請選購符合以下最低規格的平板裝置。

### 1. 作業系統與版本

- **iPad**：iPadOS 13 以上。
- **Android**：Android 9.0 以上。

### 2. 螢幕與解析度

- **螢幕尺寸**：建議 **10 吋** 以上，以維持操作按鈕的點擊精確度。
- **解析度**：
    - 4:3 比例：1024 x 768 以上。
    - 16:10 比例：1920 x 1200 以上。

!!! warning "Windows 平板：硬體配置建議"
    若您採用 Windows 系統平板進行結帳，為確保系統穩定性與功能完整度，請務必留意：
    
    - **優先選用有線設備**：強烈建議搭配 **有線刷卡機** 與 **有線發票機**，方可透過 [POS 驅動程式]() 實現硬體間的完整串聯。
    - **硬體支援**：唯有安裝 [POS 驅動程式]() 模式下，系統才支援 **客顯螢幕**、**標籤印表機** 以及 **離線結帳模式**。



## 將 POS 系統加到主畫面

為了優化平板使用體驗並獲得全螢幕結帳效果，建議將雲端 POS 網址設為主畫面捷徑。

=== "iPad (Safari)"

    1. 使用內建的 **Safari** 瀏覽器開啟雲端 POS 登入網址。
    2. 點選畫面右上角（或下方）的 :lucide-upload: **分享** 圖示。
    3. 在選單中選擇 **加入主畫面**。
    4. 點擊 **新增** 。
    5. POS 網站捷徑即會出現在桌面。

=== "Android (Chrome)"

    1. 使用 **Chrome** 瀏覽器開啟雲端 POS 登入網址。
    2. 點選右上角的 **更多 (⋮)** 選項。
    3. 選擇 **加到主畫面**。
    4. 依提示點選 **確定** > **自動新增**。
    5. POS 網站捷徑即會出現在桌面。
    

設定完成後，透過主畫面捷徑進入系統，將自動開啟全螢幕模式。

## 常見問題

??? quote "為什麼我的平板 POS 解析度看起來怪怪的？"
    請檢查平板的顯示設定是否有縮放比例調整。若螢幕解析度低於 1024 x 768，部分按鈕可能會被擠壓或隱藏。

??? quote "Android/IOS 平板如何列印電子發票？"
    平板模式無法透過 USB 連接一般熱感式發票機。您必須申請 **MyPay 金流方案** 並使用商米無線刷卡發票機進行列印。詳細設定請參閱 [商米刷卡機安裝教學](商米無線刷卡機安裝教學.md)。

# 列印發票明細
您可於 POS 子機設定中啟用 「自動開立發票」 功能，並依門市需求配置發票或交易明細的列印偏好。
{ .subtitle }

[:lucide-tag:{ title="適用方案" }](../../resources/conventions#適用方案) | 進階 PLUS / 高手 PLUS / 企業
{ .doc-badge }

!!! tip "應用情境"
    - **快速結帳需求**：結帳後自動產出發票與明細，門市人員無需手動點選，加速顧客疏導。
    - **憑證完整性**：確保每筆交易完成後同步產出消費清單（明細），方便顧客對帳。

## 使用須知

- **設備限制**：**MYPAY 發票機** 不支援自動列印相關設定。
- **格式特性**：若發票含有 **統一編號 (統編)**，明細內容將會與發票連動列印，不會進行截斷處理。

## 操作流程


1. 登入 CYBERBIZ 管理後台，前往 **POS 功能 > 所有 POS 商店**。
2. 點選欲設定的 **POS 店名**。
3. 下滑至 **POS 系統列表** 區塊，針對欲調整的子機點選 **修改 POS 設定**。
    ![](../../assets/images/POS-後台-所有POS商店-修改POS子機設定01.png){ .screenshot }
4. 根據門市需求開啟或關閉以下功能開關：

    - **自動開發票**：顧客結帳完成後，系統立即自動開立電子發票。
    - **自動列印發票設定**：結帳成功後，發票機自動印出紙本發票。
    - **自動列印明細**：交易完成時，同步自動產出消費清單（明細）。

    ![](../../assets/images/POS-後台-所有POS商店-修改POS子機設定02.png){ .screenshot }

5. 點擊 **儲存** 完成設定。

# 核銷電子票券
提供多種核銷電子票券的方式，包括電腦輸入、手機掃碼、後台查詢

## 操作流程

### 電腦輸入
- [ ] 登入 CYBERBIZ 電商後台，前往「商品」→「核銷票券頁」。
- [ ] 將游標放在核銷碼欄位，使用掃碼槍掃描 QR Code，或手動輸入核銷碼。
- [ ] 輸入「核銷數量」，點擊「確認核銷」。
- [ ] 彈窗跳出，再次確認票券名稱、數量，確認無誤後點擊「確認核銷」。
- [ ] 核銷成功。

![[電子票券核銷-電腦輸入.mp4]]
### 手機掃 QR Code
- [ ] 開啟手機相機，掃描 QR Code。
- [ ] 使用瀏覽器開啟網頁。
- [ ] 若尚未登入，將跳轉至登入頁面。
- [ ] 登入後，會跳轉至「核銷票券頁」，並自動帶入核銷碼。
- [ ] 輸入「核銷數量」，點擊 **確認核銷**。
- [ ] 彈窗跳出，再次確認票券名稱、數量，確認無誤後點擊 **確認核銷**。
- [ ] 核銷成功。

![影片-電子票券手機掃碼核銷](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/%E9%9B%BB%E5%AD%90%E7%A5%A8%E5%88%B8%E5%BD%B1%E7%89%8702.mp4)

### 後台「票券核銷列表」
若消費者忘記攜帶 QR Code 或因突發狀況無法出示，可透過姓名或電話查詢核銷碼。

- [ ] 後台路徑：「商品」→「票券核銷列表」。
- [ ] 在「選擇欲搜尋類別」中，可使用以下欄位篩選票券：
- 票券名稱
- 核銷碼
- 訂單收件人姓名
- 訂單收件人電話
- 訂單編號
- [ ] 篩選出票券後，點擊「前往核銷」，進入「核銷票券頁」，核銷碼將自動帶入。
- [ ] 輸入「核銷數量」，點擊「確認核銷」。
- [ ] 彈窗跳出，再次確認票券名稱、數量，確認無誤後點擊「確認核銷」。
- [ ] 核銷成功。

![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券設定12.png)

### 後台「核銷票券頁」手動核銷
- [ ] 登入 CYBERBIZ 電商後台，前往「商品」→「核銷票券頁」。
- [ ] 手動輸入核銷碼，點選「查看票券」，輸入「核銷數量」，點擊「確認核銷」。
- [ ] 核銷成功。

![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券設定40.png)

## 說明概述
啟用商品評論功能，獲取真實的顧客意見，以建立品牌信譽。設定與管理商品評論，了解商品評論的相關流程，從顧客發表評論，後台編輯、審核上架，及刪除等相關操作。

=== ":material-information-outline: 使用須知"

    - [ ] 需先洽[[#申請開通|客服人員開通]]商品評論功能。

=== ":material-lightbulb-outline: 使用情境"

    - 提升商品信任度：顧客可參考其他買家的真實評論，增加購買信心。  
    - 優化品牌形象：積極管理與回應評論，展現品牌對顧客意見的重視。  
    - 促進商品銷售：正向評論能成為強大的購買誘因，提升商品轉換率。  
    - 收集顧客回饋：透過評論了解顧客對商品的看法，作為商品優化與服務改進的依據。

### 步驟一：設定商品評論功能

=== ":material-numeric-1-circle: 申請開通"

    - [ ] 登入 CYBERBIZ 電商後台，點擊右下角「線上客服 :octicons-comment-discussion-24:」聯絡客服人員。

    ![[EC-線上客服.png]]

=== ":material-numeric-2-circle: 配置設定"

    - [ ] 登入 CYBERBIZ 電商後台，前往「網站外觀」→「管理商品評論」。
    - [ ] 依照需求變更設定。
    - [ ] 點擊「更新」儲存變更。

    ![](https://www.cyberbiz.io/support/wp-content/uploads/商品評論01.png){ .screenshot }

    | 項目 | 功能說明 | 類型 |
    |------|----------|------|
    | **預設審核留言** | 設定顧客留言後，商家是否需要手動審核。 | :material-toggle-switch: |
    | **預設隱藏部分姓名** | 設定顧客留言時，是否部分隱藏其姓名。 | :material-toggle-switch: |
    | **Google reCAPTCHA** | 安全驗證機制，防止垃圾訊息與機器人留言。設定請參考[[設定留言區 reCAPTCHA]]。 | :material-toggle-switch: |
    | **審核後贈送紅利點數** | 設定商家審核通過顧客評論後，是否自動贈送紅利點數。 | :material-form-textbox: |

=== ":material-numeric-3-circle: 檢視留言板"

    所有商品的留言板皆可在此細部設定，按下「檢視」可進行編輯。

    ![](https://www.cyberbiz.io/support/wp-content/uploads/商品評論02.png){ .screenshot }

=== ":material-numeric-4-circle: 設定留言板"

    - [ ] 可各別設定留言板。
    - [ ] 點擊「垃圾桶 :material-trash-can-outline:」可刪除商品評論。

    ![](https://www.cyberbiz.io/support/wp-content/uploads/商品評論03.png){ .screenshot }

### 步驟二：顧客發表商品評論

!!! note "顧客需登入會員後，方可於商品頁面發表評論。"

- [ ] 顧客在商品頁面點選「發表評論」按鈕。
- [ ] 系統將跳出彈跳視窗，顧客可在此填寫星級評價與評論內容。

![](https://www.cyberbiz.io/support/wp-content/uploads/商品評論04.png){ .screenshot }

### 步驟三：商家審核商品評論

商家可在後台審核顧客提交的商品評論。

=== ":material-numeric-1-circle: 前往後台"
    - [ ] 登入 CYBERBIZ 電商後台，前往「會員」→「商品待審核評論」。

=== ":material-numeric-2-circle: 選擇評論"
      - [ ] 點選「待審核」，顯示待審核留言。
      - [ ] 勾選欲操作的留言。

=== ":material-numeric-3-circle: 審核動作"
    - `審核` 將評論顯示於前台，通過審核後即可公開。
    - `刪除` 刪除該評論。
    - `編輯` 修改評論內容，進行調整或修正。

![](https://www.cyberbiz.io/support/wp-content/uploads/商品評論05.png){ .screenshot }

### 步驟四：隱藏商品評論功能

若您暫時不希望顯示商品評論功能，可透過樣版編輯器隱藏。

!!! danger "若選擇自行移除程式碼，CYBERBIZ 將不提供恢復功能協助。"

=== "拖拉版型"

    === ":material-numeric-1-circle: 前往樣板編輯器"
        - [ ] 登入 CYBERBIZ 電商後台，前往「網站外觀」→「套版主題管理」→「選擇操作 : CSS/HTML編輯器」。

    === ":material-numeric-2-circle: 新增區塊文件"
        - [ ] 在「區塊」選單中點選「新增區塊」。
        - [ ] 將新增區塊命名為 `board_comments.liquid`。

    === ":material-numeric-3-circle: 文件內容留白"
        - [ ] 點擊打開新增的 `board_comments.liquid` 的區塊檔案。
        - [ ] 將檔案內容（右方區域）留白即可隱藏商品評論功能。點擊 :material-trash-can-outline: 刪除 `board_comments.liquid` 文件即可恢復商品評論功能。

    ![[ec-樣板編輯器-board_comments.liquid.png]]

=== "一般版型"

    === ":material-numeric-1-circle: 前往樣板編輯器"
        - [ ] 登入 CYBERBIZ 電商後台，前往「網站外觀」→「套版主題管理」→「選擇操作 : CSS/HTML編輯器」。

    === ":material-numeric-2-circle: 編輯樣板文件"
        - [ ] 搜尋並打開 `product.liquid` 文件，找到 `shop.plugins.board_comments` 這段程式碼。
        - [ ] 以 HTML 註解符號 `<!--` 與 `-->` 包覆整段程式碼（註解程式碼[^註解程式碼]），即可停用並隱藏商品評論功能，無需刪除程式碼。

    === ":material-numeric-3-circle: 儲存變更"
        - [ ] 點擊「儲存」套用更新。

    ![](https://www.cyberbiz.io/support/wp-content/uploads/商品評論07.png){ .screenshot }

## 常見問題

??? question "顧客發表評論是否需要登入會員？"
    是的，顧客必須登入會員後才能發表商品評論。

??? question "如果自行移除樣版編輯器中的程式碼，CYBERBIZ 會提供恢復功能嗎？"
    若您自行移除樣版編輯器中的程式碼，CYBERBIZ 不提供恢復功能等操作。請務必自行保留程式碼備份。

??? question "如何設定 Facebook 分享功能？"
    若要使用 Facebook 分享功能，需至「第三方整合」→「臉書Facebook設定」→「設定 應用程式ID (APP ID) 及 應用程式密鑰 (App Secret)」填寫完畢，方可使用。

# 批次修改商品資訊
透過 Excel 批次匯出、編輯與上傳，一次更新多筆商品的描述、溫層、配送方式與銷售通路
{ .page-subtitle }

## 說明概述

=== ":material-lightbulb-outline: 使用情境"

    - 節省操作時間：一次性更新多筆商品描述、溫層、配送方式及銷售通路，減少手動修改。
    - 快速調整資訊：促銷或季節性商品可即時批次修改描述、通路與物流設定。
    - 統一管理屬性：批次設定商品溫層與配送方式，避免單筆錯誤，確保物流與儲存流程正確。
    - 精準控制通路：快速設定或修改商品可銷售的通路，方便跨通路管理。

---

## 操作流程
### 步驟一：匯出商品 Excel 表格

=== "匯出部分商品"

    === ":material-numeric-1-circle: 選取商品"

        - [ ] 登入 CYBERBIZ 電商後台，前往 商品 → 所有商品。
        - [ ] 在商品列表中勾選欲修改的商品品項。

        ??? tip "快速選取符合條件的特定商品"
            利用[[搜尋商品|商品篩選器]]，快速篩選出符合特定條件的商品，再點擊 `已選取_商品` 中的勾選框 :material-checkbox-blank-outline:，一次選定符合篩選條件的所有商品。

    === ":material-numeric-2-circle: 匯出商品"
        - [ ] 點擊 `Select` 操作選單 :material-arrow-down-drop-circle-outline: 。
        - [ ] 點擊 `匯出商品`，系統將自動下載 Excel 檔案至您的電腦。

        ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/批次修改商品資料1.png){ .screenshot }

=== "匯出全部商品"

    === ":material-numeric-1-circle: 選取商品"

        - [ ] 登入 CYBERBIZ 電商後台，前往 商品 → 所有商品。
        - [ ] 點擊 `已選取__個商品` 欄位，選擇 `選取全部商品`。

    === ":material-numeric-2-circle: 匯出商品"
        - [ ] 點擊 `Select` 操作選單 :material-arrow-down-drop-circle-outline: 。
        - [ ] 點擊 `匯出商品`，系統將自動下載 Excel 檔案至您的電腦。

        ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/批次修改商品資料2.png){ .screenshot }

### 步驟二：編輯 Excel 檔案
開啟下載的 Excel 檔案，在對應欄位輸入或修改商品資訊。

!!! warning "`商品 id` 及 `商品款式 id` 欄位必須帶有系統數值，才能更新既有商品。若這些欄位為空，系統將會新增商品。"

=== "商品描述"

    - [ ] 商品描述頁面包含「商品介紹」、「規格說明」、「運送方式」三個區塊。請分別在 Excel 表格的「商品介紹」、「規格說明」、「運送方式」欄位中輸入對應文字。
    - [ ] 您可至後台路徑：「商品」→「所有商品」→ 點擊特定商品 → 點擊「商品描述」頁籤，查看商品描述的設定頁面。

    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/批次修改商品資料7.png){ .screenshot }  
    
=== "商品通路"

    - [ ] 請輸入商品的出貨通路名稱。若輸入商店原先沒有的通路，系統將視為 **新增通路**。
    - [ ] 留空表示 **適用全通路**。

    !!! warning "通路新增後將 **無法刪除**" 

=== "商品溫層"
    - [ ] 請輸入 `常溫`、`冷藏` 或 `冷凍`。
    - [ ] 留空表示 **預設為常溫**。
    - [ ] 可複選多溫層，輸入時請使用**英文逗號**分隔，例如：`常溫,冷藏,冷凍`。

=== "商品運送名稱"
    - [ ] 請輸入配送物流名稱。請勿輸入後台未設定的物流名稱。
    - [ ] 可複選多物流配送，輸入時請使用**英文逗號**分隔，例如：`黑貓,宅配通`。
    - [ ] 留空表示**適用全部配送方式**。

    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/批次修改商品資料3.png){ .screenshot }

### 步驟三：上傳 Excel 檔案
- [ ] 登入 CYBERBIZ 電商後台，前往 商品 → Excel 大量匯入商品。
- [ ] 上傳已編輯完成的 Excel 檔案。

![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/批次修改商品資料4.png){ .screenshot }

### 步驟四：等待匯入排程通知

=== ":material-progress-check: 資料匯入處理中"
    - [ ] 若輸入格式無誤，您將會收到「資料匯入處理中」的 EMAIL 通知。

    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/批次修改商品資料5.png){ .screenshot }

=== ":material-check-circle-outline: 商品資料匯入成功" 

    - [ ] 當匯入作業完成後，您會再收到「商品 資料匯入成功」的 EMAIL 通知，表示批次修改商品內容已完成。

    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/批次修改商品資料6.png){ .screenshot }

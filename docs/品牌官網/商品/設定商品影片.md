## 操作流程
### 步驟一：上傳商品影片至後台

![[EC-新增商品-商品影片.png]]

=== "影片規格"
    
    為確保影片能順利上傳、處理及於前台正常播放，請遵循以下技術規格：

    | 規格項目     | 限制 / 說明             |
    | -------- | ------------------- |
    | **解析度**  | 最大支援 1280 × 1280 像素 |
    | **建議比例** | 9:16[^影片比例]            |
    | **影片格式** | 僅支援 MP4 格式          |
    | **影片長度** | 最長 60 秒             |
    | **檔案大小** | 最大 30 MB[^影片大小]        |
    | **音訊支援** | 不支援音訊輸出[^影片音訊限制]         |

=== "上傳方式"

    === ":material-package-variant-closed-plus: 新增商品時上傳影片"

        - [ ] 登入 CYBERBIZ 電商後台，前往 商品 → 新增商品。
        - [ ] 填寫[[新增單一商品#基本設定|商品基本資訊]]。
        - [ ] 在商品影片區塊點擊 **上傳** 按鈕，選擇本機影片檔案。
        - [ ] 確認影片與商品資訊無誤後，點擊 **儲存**。
        - [ ] 系統將處理影片上傳，請耐心等待完成。

    === ":material-movie-plus-outline: 為既有商品新增影片"

        - [ ] 登入 CYBERBIZ 電商後台，前往 商品 → 所有商品。
        - [ ] 在商品列表中，點擊欲編輯的商品名稱，進入 商品編輯頁面。
        - [ ] 在商品影片區塊點擊 **上傳** 按鈕，選擇本機影片檔案。
        - [ ] 確認影片正確後，點擊 **儲存**。
        - [ ] 系統將處理影片上傳，請耐心等待完成。

### 步驟二： 影片彈窗顯示設定

統一設定影片在商品彈窗中的顯示位置，以提升前台展示效果。

- [ ] 登入 CYBERBIZ 官網後台，前往「網站外觀」 →「套版主題管理」→「網站設定」→「全站設定」。
- [ ] 在「商品影片設定」區塊選擇影片顯示位置
- [ ] `置於商品圖片之前`：影片會在商品圖片之前播放。    
- [ ] `置於商品圖片之後`：影片會在商品圖片之後播放。

![[圖示-影片彈窗顯示設定.gif]]{ .screenshot }

### 步驟三： 媒體庫影片管理

- [ ] 登入 CYBERBIZ 電商後台，前往「商品」→「媒體庫」。
- [ ] 集中管理所有影片檔案，可搜尋或刪除。
- [ ] 影片上傳後，當您編輯商品頁面時，即可選擇「從媒體庫選擇影片」。

## 常見問題
??? question "為什麼我的商品影片沒有聲音？"
    CYBERBIZ 商品影片功能目前不支援音訊輸出，影片將以靜音方式播放。

??? question "影片上傳失敗怎麼辦？"
    - 確認影片檔案**大小是否超過 30 MB**  
    - 確認影片**格式為 MP4**  
    - 若仍無法上傳，請聯絡客服支援。

??? question "商品影片只能在拖拉版型顯示嗎？"
    是的，商品影片功能**僅支援拖拉版型**，其他版型無法顯示影片。

??? question "我可以調整影片在彈窗中的顯示順序嗎？"
    可以，上傳後可在後台**設定彈窗影片顯示順序**為第一格或最後一格。

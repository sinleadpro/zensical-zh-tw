# 設定商品到貨通知 :material-lock-outline:
發送 Email 通知顧客追蹤商品已到貨。
{ .page-subtitle } 

[:material-tag-text:](){ .badge-icon title="適用方案" }[進階 | 進階 PLUS | 高手 | 高手 PLUS | 企業](){ .badge-text }


## 說明概述
=== ":material-information-outline: 使用須知"
    - 每位會員僅會收到一次補貨通知[^補貨通知次數]。若商品再次售罄，會員需重新點擊「已售完，貨到通知我」按鈕才能再次登記。
    - 此功能支援版型版本號 3.47.0 及以上。請確認您的版型版本號是否符合要求。
	 
=== ":material-lightbulb-outline: 使用情境"

	- 提升顧客滿意度：當熱門商品缺貨時，顧客可登記補貨通知，商品到貨後即時收到通知，避免錯失購買機會。
	- 促進銷售轉換：有效追蹤顧客對缺貨商品的興趣，並在商品補貨時精準觸達潛在買家，提高銷售轉換率。
	- 優化庫存管理：透過顧客登記數據，商家可更精準地評估商品需求，優化庫存補貨策略。

---

## 設定指南
商品到貨提醒功能的觸發條件如下：

- [x] 商品需開啟「管理庫存」:material-toggle-switch:，且庫存為 `0`。同時，庫存不足時需設定為 `停止銷售` ，前台商品頁面才會顯示「已售完，貨到通知我」按鈕。
- [x]  商品庫存狀態需設定為「當無庫存時，款式為可點選狀態」。客戶點選商品款式後，商品才會顯示「貨到通知」選項。

## 操作流程
### 步驟一：開啟商品到貨提醒功能 
=== ":material-numeric-1-circle: 設定庫存管理與停止銷售"
    - [ ] 前往後台路徑：「商品」 → 「所有商品」→ 選擇您要設定的商品 → 「款式管理」。
    - [ ] 確保已開啟「管理庫存」:material-toggle-switch:，並將「庫存不足」時的設定調整為「停止銷售」。

    [![商品貨到通知01](https://www.cyberbiz.io/support/wp-content/uploads/商品貨到通知01.png)](https://www.cyberbiz.io/support/wp-content/uploads/商品貨到通知01.png)

=== ":material-numeric-2-circle: 設定無庫存時商品款式狀態"

	=== "拖拉版型"
	    - [ ] 前往後台路徑：「網站外觀」 → 「套版主題管理」 → 「網站設定」 → 「商品頁面」 → 「基本設定」。
	    - [ ] 勾選「無庫存時，商品款式狀態設定」中的「當無庫存時，款式為可點選狀態」。
	
	    ![商品貨到通知02](https://www.cyberbiz.io/support/wp-content/uploads/商品貨到通知02-1.png)

	=== "一般版型"
	    - [ ] 前往後台路徑：「網站外觀」→ 「套版主題管理」 → 「網站設定」 → 「商品群組與商品頁設定」。
	    - [ ] 勾選「顯示商品無庫存狀態」中的「當無庫存時，款式為可點選狀態」。
	
	    ![商品貨到通知02](https://www.cyberbiz.io/support/wp-content/uploads/商品貨到通知02.png)

=== ":material-numeric-3-circle: 前台顯示貨到通知選項" 
	
	- [ ] 會員點擊「已售完，貨到通知我」按鈕後，按鈕會變更為「已登記補貨通知」。
	- [ ] 當商品庫存從 0 更新為 > 0 或無限庫存時，系統將自動發送補貨通知信給已登記的會員。
	
	<div class="grid cards borderless two-columns" markdown>
	
	- ![商品貨到通知03](https://www.cyberbiz.io/support/wp-content/uploads/商品貨到通知03.png)
	- ![商品貨到通知04](https://www.cyberbiz.io/support/wp-content/uploads/商品貨到通知04.png)
	
	</div>

### 步驟二：Email 樣板設定
![[ec-訊息推播-商品貨到通知.png]]

自訂商品貨到通知的 Email 樣板內容。
=== ":material-numeric-1-circle: 進入操作頁面"
	 - [ ] 登入 CYBERBIZ 電商後台，前往「訊息推播」 → 「Email 通知樣板」 → 「顧客相關」 → 「商品貨到通知」。
=== ":material-numeric-2-circle: 更改通知內容"
	- [ ] 在「電子郵件內文類型」區域，選擇「HTML 樣板內文」或 「純文字」。
	- [ ] 在「電子郵件內文」區域中更改電子郵件內文。

	!!! danger "請勿任意修改 `{{}}` 內的文字，此為系統自動帶入的變數，更動可能導致功能異常。"
	
=== ":material-numeric-3-circle: 查看預覽"
	- [ ] 編輯完成後，點擊「:material-file-search-outline: 預覽」 查看修改內容。 
=== ":material-numeric-4-circle: 儲存變更"
	- [ ] 確認無誤後，點擊「儲存」按鈕。

# 設定條件分類群組
依商品條件建立動態分類
{ .page-subtitle } 

---

## 說明概述
=== ":material-lightbulb-outline: 使用情境"

	- 高效商品管理：依據商品價格、庫存、關鍵字等條件，自動將符合規則的商品歸類到特定群組，大幅提升管理效率。
	- 精準行銷活動：針對特定條件分類的商品進行行銷活動，例如 [[docs/ec/product/operation/設定單品限時折扣群組|單品限時折扣]]。
	- 優化搜尋引擎排名：透過設定分類描述與關鍵字，優化 SEO 表現，讓商品更容易被搜尋引擎找到。
	- 多層級條件篩選：支援設定多層篩選與搜尋條件，方便精準管理複雜的商品結構。

---

## 操作流程
### 步驟一：建立商品條件分類
=== ":material-numeric-1-circle: 進入操作頁面"
	![[ec-商品條件分類-新增商品分類.png]]
	
	- [ ] 登入 CYBERBIZ 電商後台，前往「商品」→「商品條件分類」。
	- [ ] 點擊「新增商品分類」進入編輯頁面。

=== ":material-numeric-2-circle: 填寫分類設定"
	![[ec-商品條件分類-分類設定.png]]
	
	- [ ] 輸入「分類名稱」、「分類網址」，並填寫「分類描述」。

=== ":material-numeric-3-circle: 儲存變更"
	![[ec-商品條件分類-儲存.png]]
	
	- [ ] 點選「儲存」以建立新的商品條件分類。

### 步驟二：設定商品篩選規則
=== ":material-numeric-1-circle: 選擇條件分類"
	![[ec-商品條件分類-選擇條件分類.png]]
	
	- [ ] 在條件分類列表中，點選欲加入商品的條件分類，進入編輯頁面。

=== ":material-numeric-2-circle: 設定篩選規則"
	![[ec-商品條件分類-設定商品篩選規則.png]]

	- [ ] 點擊「加入商品」分頁，開始設定商品篩選規則。
	- 篩選結果：選擇「符合所有條件」或「符合任一條件」邏輯。
	- 新增商品規則：依據商品名稱、類型、廠商、價格、定價、庫存現貨、款式名稱、商品標籤等條件進行關鍵字查找與篩選。

=== ":material-numeric-3-circle: 儲存變更"
	![[ec-商品條件分類-篩選商品列表.png]]
	
	- [ ] 點擊「儲存」，符合條件的商品將自動顯示於下方列表，表示商品已成功加入該分類。

	!!! warning "設定商品排序時，若商品名稱包含系統不支援的標點符號，可能導致排序功能無法正常運作。"

### 步驟三：編輯 SEO 設定
- [ ] 在商品條件分類頁面，下滑找到「SEO 設定」欄位。
- [ ] 編輯網頁資訊，優化搜尋引擎，以提升網站搜尋排名。

![商品群組介面說明](https://www.cyberbiz.io/support/wp-content/uploads/商品條件分類03.png){ .screenshot }

### 步驟四：前台畫面顯示
- [ ] 完成設定後，商品條件分類將會顯示於商店前台。

![商品群組介面說明](https://www.cyberbiz.co/support/wp-content/uploads/2020/03/智慧群組09.png){ .screenshot }

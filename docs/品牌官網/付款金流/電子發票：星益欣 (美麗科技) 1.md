# 星益欣電子發票設定指南

:lucide-lock:{ title="適用方案" } | 專業 PLUS · 進階(PLUS) · 高手(PLUS) · 企業  


本文件教您如何在 CYBERBIZ 後台設定星益欣電子發票，涵蓋方案購買、服務串接、多站台共用、對帳發票設定及帳戶續購。

## 使用須知

=== "前置條件"
	- [x] 商家方案：需企業版或支援電子發票串接的方案
	- [x] 已購買或準備購買星益欣方案
	- [x] 已取得統一編號，並可使用商品評論功能（若尚未開通，請聯絡客服）

=== "適用範圍"
	- [x] 多個 EC 官網發票開立
	- [x] 線下 POS 系統發票開立
	- [x] 企業版 EC 官網請款發票開立
	- [x] 其他通路（第三方平台）發票開立

=== "注意事項"
	- 「美麗科技」已與「星益欣」合併，本文以「星益欣」指稱。
	- 設定請依 [星益欣官網公告 :lucide-external-link:](https://www.wixtar.com/product/e-invoice) 為準，電話 : 02-2711-9528 #240。
	- 匯入字軌請參考 [匯入字軌教學文件](https://www.cyberbiz.io/helpcenter/wp-content/uploads/美麗科技-增加字軌數量_匯入教學使用手冊-20230511.pdf)。

=== "費用"
	- 教學僅提供操作流程說明，實際費用請依後台顯示為準。

## 同統編多通路發票開立情境

* 系統依統編建立帳戶，僅能開立相同統編發票。
* 支援以下開立情境：

| 開立情境                  | 發票開立位置       | 注意事項 |
| :------------------------ | :---------------- | :------ |
| 多個 EC 官網              | CYBERBIZ EC 後台  | 請參閱 [多站台共用帳戶](#多-ec-站台共用帳戶) 並完成串接。 |
| 線下 POS 系統             | CYBERBIZ EC 後台  | 支援 2 台 POS 機開立，多台請參考 [POS 教學文件](https://www.cyberbiz.io/support/?p=46379)。 |
| 企業版 EC 官網請款發票    | CYBERBIZ EC 後台  | 由 CYBERBIZ 代開請款發票，請參考 [串接啟用對帳發票](#串接啟用對帳發票)。 |
| 其他通路（第三方平台）     | 星益欣後台         | 請自行操作星益欣後台，CYBERBIZ 不提供第三方操作教學。 |

* **注意**：若需開立不同統編發票，請分別購買方案。

## 購買星益欣方案
**後台路徑:** 「管理中心」→「發票設定」

1. 點選「帳號驗證」，輸入統一編號。  
    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/電子發票-美麗科技01.png){ .screenshot }
2. 選擇方案並填寫購買資訊（方案、聯絡人、公司發票資訊）。  
    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/電子發票-美麗科技02.png){ .screenshot }
3. 完成付款後，CYBERBIZ 會將聯絡資料傳送給星益欣專員，協助開通帳號。  
    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/電子發票-美麗科技03.png){ .screenshot }  
    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/電子發票-美麗科技04.png){ .screenshot }
4. 帳號開通後，填寫串接資訊以啟用電子發票服務。

## 串接並啟用服務
**後台路徑:** 「管理中心」→「發票設定」

1. 登入星益欣後台 →「營業人資訊」→選擇 POS 機 →「下載」。  
    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/電子發票-美麗科技07.png){ .screenshot }
2. 複製以下資訊：
    - 門市店碼
    - POS機碼
    - POS機序號(PID)
    - 認證碼(RID)
    - 產品名稱(PNAME)
    - AES KEY  
    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/電子發票-美麗科技08.png){ .screenshot }
3. 貼入 CYBERBIZ 後台對應欄位。  
    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/電子發票-美麗科技06.png){ .screenshot }
4. 開啟「結帳頁顯示發票」並選擇「星益欣電子發票」。  
    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/電子發票-美麗科技09.png){ .screenshot }
5. 選擇開立時間：
    - **付款時**：訂單付款狀態為「已收到款項」時，自動開立。
    - **出貨時**：訂單配送狀態為「已出貨」時，自動開立。
    - **取貨時**：訂單配送狀態為「已收貨」時，自動開立。

!!! tip "發票開立時間建議"
    建議選擇「出貨時」以避免客戶取消訂單而作廢已開立發票。

!!! note "取貨時選項限制"
    - 僅適用於 CYBERBIZ 已串接貨態的運送方式（黑貓、宅配通、順豐、綠界/EZShip 超取）。
    - 若使用自訂物流，配送狀態將停留「已出貨」，取貨時無法自動開立發票。建議同時勾選「出貨時」。

6. 前往星益欣後台設定公司發票章圖片，開立發票將自動帶入發票章。  
    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/電子發票-美麗科技11.png){ .screenshot }

## 多 EC 站台共用帳戶

1. 完成指定站台購買方案與串接啟用。
2. 其他站台輸入相同統編，並填入相同串接資訊（門市代碼、POS機碼、POS機序號、RID、PNAME、AES KEY）。
3. 完成結帳頁發票設定後，即可開立發票。  
    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/電子發票-美麗科技12.png){ .screenshot }

## 串接啟用對帳發票
**後台路徑:** 「管理中心」→「對帳中心」→「對帳發票設定」

1. 選擇由 CYBERBIZ 代開消費者發票，輸入統編。
2. 若尚未購買方案，請先完成 [購買方案](#購買星益欣方案)。
3. 完成串接啟用設定。  
    ![](https://www.cyberbiz.io/support/wp-content/uploads/電子發票-美麗科技13.png){ .screenshot }  
    ![](https://www.cyberbiz.io/support/wp-content/uploads/電子發票-美麗科技14.png){ .screenshot }

## 開立不同統編發票

1. 各站台輸入不同統編，分別購買方案並獨立串接啟用。
2. 系統將建立對應星益欣帳戶。

### 情境舉例
- **多個 EC 站台，各自開立不同統編發票**：分別購買方案並完成串接。  
- **同時營運 EC 與 POS 站台**：各自設定統編，分開購買方案並完成串接。  

## 續購方案
**後台路徑:** 「管理中心」→「發票設定」

1. 點選「續購」，填寫結帳資訊延長使用期限。  
    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/續購_電子發票01.png){ .screenshot }
2. 付款方式：虛擬 ATM 或信用卡付款。  
    ![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/電子發票-美麗科技02.png){ .screenshot }

## 常見問題

??? quote "一組星益欣帳號可以開立不同統編的發票嗎？"
    不行。每個星益欣帳號僅綁定一個統編，需分別購買方案。

??? quote "如果星益欣帳號未開通就進行串接，會發生什麼事？"
    發票功能將無法使用。請等待專員聯繫並完成帳號開通後再串接。

??? quote "是否所有 CYBERBIZ 方案都支援星益欣電子發票串接？"
    僅支援電子發票串接方案，請洽業務或客服確認方案適用性。

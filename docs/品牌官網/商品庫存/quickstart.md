# 新手上路

歡迎使用商品與庫存管理新手上路指南！  
本清單將帶您完成商品建立、分類、上架、庫存設定與補貨管理，協助您快速準備商店商品並正式開始銷售。

商品與庫存管理流程分為 **五大階段（I～V）**，每階段提供預估完成時間與必要任務。

---

## 認識商品管理介面

> **預估時間：~30–60 分鐘**

此階段協助您建立商品頁、準備商品資訊並完成初步上架設定。

!!! abstract "新增商品"  
	- 單一商品新增  
	- 商品圖片、影片上傳  
	- 設定商品描述、規格、售價

!!! abstract "商品分類與分組"  
	- [ ] 設定商品分類（大／中／小分類）  
	- [ ] 設定自訂商品群組（如加價購、不適用折扣群組）

!!! abstract "商品上架設定"  
	- [ ] 配送方式設定（宅配 / 超取）  
	- [ ] 溫層綁定（常溫／冷藏／冷凍）  
	- [ ] 設定商品可見性（是否顯示於前台）

---

## 使用商品篩選器

> **預估時間：~30–45 分鐘**

確保商品數量正確，並設定好各倉庫庫存、預購與補貨通知。

??? abstract "庫存數量管理"  
	- [ ] 庫存調整（入庫 / 出庫）  
	- [ ] 多倉庫管理（若適用）

??? abstract "庫存相關設定"  
	- [ ] 安全庫存  
	- [ ] 預購模式  
	- [ ] 補貨通知（售完自動通知會員）

---

## 批次上架與批次更新（大量商品管理）

> **預估時間：依商品量而定（通常 30–120 分鐘）**

大量商品可透過 Excel 一次新增、更新資訊或調整設定。

??? abstract "批次匯入 & 更新"  
	- [ ] Excel 批次匯入商品  
	- [ ] 批次更新描述／價格／配送方式／溫層  
	- [ ] 批次圖片更新（使用 image URL）

---

## 商品強化與行銷設定（提升商品曝光）

> **預估時間：每項 10–20 分鐘**

透過加價購、限時折扣、標籤等方式優化前台呈現。

??? abstract "促銷設定"  
	- [ ] 設定訂單／商品加價購  
	- [ ] 設定限時折扣  
	- [ ] 設定不適用優惠群組

??? abstract "商品呈現最佳化"  
	- [ ] 商品標籤管理  
	- [ ] 上傳商品影片  
	- [ ] 設定 SEO、頁面標題、關鍵字

---

## 配送、溫層與通路審核（商品可售狀態確認）

> **預估時間：20–30 分鐘**

確認商品在所有銷售通路均能正常販售。

??? abstract "配送設定"  
	- [ ] 聯動配送（宅配／貨到付款／超商取貨）  
	- [ ] 超商配送限制與排除設定  
	- [ ] 商品排除上架至第三方平台（如必要）

??? abstract "通路檢查"  
	- [ ] 商品前台可見性確認  
	- [ ] 商品是否在各分類／群組中正常顯示  
	- [ ] 商品庫存狀態是否為「可銷售」

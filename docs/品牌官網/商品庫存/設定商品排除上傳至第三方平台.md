## 說明概述
> 「排除商品上傳至第三方平台」功能是透過套用特定「商品標籤」（例如 `贈品` 或 `排除product feed`）來實現。系統會依據這些標籤自動排除商品，使其不會同步至指定的第三方平台。

=== ":material-information-outline: 使用須知"
	當商品被設定排除標籤後，系統將不會把該商品資料上傳至以下第三方平台：
	
	- [x] Google Merchant Center
	- [x] Facebook 動態產品目錄 DPA
	- [x] Line 購物
	- [x] 美安 (僅限 `贈品` 標籤)

=== ":material-lightbulb-outline: 使用情境"

	- 贈品管理：將贈品設定為不推播至第三方平台，避免贈品被誤認為可單獨購買的商品。
	- 特定通路商品：某些商品僅限於特定銷售管道（如實體門市或特定活動頁面），不希望在第三方平台曝光。
	- 測試商品：將仍在測試階段的商品排除，避免在前台或第三方平台誤顯示。
	- 避免重複曝光：針對已在其他平台手動上架的商品，避免系統自動同步造成重複。

## 操作流程
![排除商品](https://www.cyberbiz.io/support/wp-content/uploads/排除贈品不要上傳至第三方平台01.png){ .screenshot }
	
您可以透過為商品添加特定標籤，來設定商品排除上傳至第三方平台：

=== ":material-numeric-1-circle: 進入操作頁面"
	 - [ ] 登入 CYBERBIZ 電商後台，前往「商品」→「所有商品」。
	 - [ ] 在商品列表中，點擊欲排除商品的「商品名稱」，進入商品編輯頁面。
=== ":material-numeric-2-circle: 建立排除標籤"
	 - [ ] 在商品編輯頁面中，找到「商品標籤」區塊。
	 - [ ] 在商品標籤欄位中，輸入以下任一排除標籤：
     -   `排除product feed`：此標籤適用於 Google Merchant Center、Facebook 動態產品目錄 DPA、Line 購物等平台。
     -   `贈品`：此標籤適用於所有第三方平台，包括美安。

	!!! warning "標籤輸入注意事項"
	    -   `排除product feed` 標籤中，「排除」與「product」之間請勿添加空格。
	    -   美安平台目前僅支援透過 `贈品` 標籤排除商品，不支援 `排除product feed` 標籤。

=== ":material-numeric-3-circle: 儲存變更"
	- [ ] 點擊「儲存」，完成設定。

## 常見問題

??? question "如果我同時設定了 `排除product feed` 和 `贈品` 標籤，會有什麼影響？"
    若同時設定兩個標籤，系統將優先依據 `贈品` 標籤的規則進行排除，確保商品不會上傳至所有支援的第三方平台。

??? question "排除商品後，顧客還能在我的 CYBERBIZ 商店中找到並購買嗎？"
    可以。此排除設定僅影響商品是否上傳至第三方平台，不影響商品在您 CYBERBIZ 商店中的可見性與購買功能。顧客仍可透過商店內搜尋或直接連結找到並購買商品。

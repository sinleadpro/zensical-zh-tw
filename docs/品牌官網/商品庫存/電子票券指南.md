# :lucide-ticket: 電子票券指南

:lucide-lock:{ title="適用方案" } |  PLUS 企業

電子票券功能支援販售各類數位商品與服務，並提供線上購買、線下 QR Code 核銷與門市權限管理。  
本指南依操作流程整理，可協助您快速完成設定、啟用銷售，掌握票券的建立、販售、核銷、退款與權限控管。

!!! quote "票券發行人"
	票券發行人為順立智慧股份有限公司，票券面額為實際售價（含優惠），已存入發行人於永豐銀行開立之「信託專戶」，專款專用，保障履約安全。

## 了解核心規範
確保票券販售與核銷機制符合平台規範
> **預估時間：5 分鐘**

=== "功能限制"  
	- 可加入「限購群組」，限制每位消費者可購買張數  
	- 僅支援「電子票券任選折扣」與「分潤」，不可與其他優惠併用  
	- 付款方式僅支援「信用卡一次付清」

=== "應用情境"  
	- 課程／活動票券銷售  
	- 商品兌換組／服務預購  
	- OMO：線上購買、門市 QR Code 核銷  
	- 多票種管理（早鳥票、團體票、兌換組）

## 顧客購買流程（前台）
了解消費者從購買到取得票券的完整行為
> **預估時間：3–5 分鐘**

1. **票券商品頁面**  顯示[使用限制與注意事項](購買電子票券.md#票券商品頁面)
2. **購物車**  若與一般商品合併，會要求[拆分結帳](購買電子票券.md#購物車分拆)
3. **結帳**  顧客需[勾選「同意票券條款」](購買電子票券.md#勾選同意)
4. **取得 QR Code** 
    - 會員中心 > 電子票券訂單查詢
    - 電子票券確認信（Email）

## 常見問題

??? quote "核銷後可以退款嗎？"  
	不行。已核銷的票券無法進行退款。

??? quote "若不小心核銷錯誤，能復原嗎？"  
	目前無法復原，請務必確認後再核銷。

??? quote "為什麼票券會計算到運費？"  
	您的物流設定可能仍為舊版，請聯繫客服協助升級。

??? quote "購買 5 張會有幾個 QR Code？"  
	**1 個**。QR Code 為一對多設計，可包含多張票券。

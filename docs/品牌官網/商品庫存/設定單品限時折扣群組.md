# 設定單品限時折扣群組 :material-lock-outline:
[:material-tag-text:](){ .badge-icon title="適用版本" }[PLUS | 企業](){ .badge-text }

在 CYBERBIZ 後台，您可以為單一商品設定限時折扣活動，包含固定金額折抵、百分比折扣或直接設定活動價格。此功能有助於您靈活規劃商品促銷策略，吸引顧客購買。

## 說明概述
=== ":material-lightbulb-outline:  使用情境"

	- 短期促銷活動：針對特定商品推出限時優惠，刺激顧客快速下單。
	- 清倉或庫存去化：對滯銷商品設定折扣，加速庫存周轉。
	- 新品上市推廣：為新上架商品提供短期折扣，提升初期銷售量與市場關注度。

## 操作流程
### 步驟一：新增單品折扣群組

- [ ] 登入 CYBERBIZ 電商後台，前往「行銷活動」→「單品限時折扣」。
- [ ] 點擊「新增群組」。

![](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/單品折扣位置.png){ .screenshot }

### 步驟二：設定計價規則
![](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/設定計價規則1.png){ .screenshot }

=== ":material-numeric-1-circle: 選擇折扣方式"
	在新增的群組中，設定折扣的計價方式。您可以選擇：
    
    - [ ] 折固定金額：商品價格減去固定金額。
    - [ ] 打折：商品價格乘以折扣百分比。
    - [ ] 設定活動價格：直接指定商品在活動期間的銷售價格。

=== ":material-numeric-2-circle: 填寫折扣數值"
	- [ ] 根據選擇的計價方式，填寫對應的折扣數值。

!!! warning "優惠計算順序"
    請務必確認您設定的行銷活動是否符合預期的結帳計算順序。不同行銷活動の計算優先級可能影響最終折扣金額。詳細請看 [[結帳優惠計算順序]]。

    [![優惠順序](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/TOP02.png)](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/TOP02.png)

### 步驟三：將活動商品加入群組並設定活動期間
=== ":material-numeric-1-circle: 選擇商品"
	在群組設定頁面，您可以選擇將商品加入此活動：
	
	- [ ] 手動加入：商品會依款式展開，您可以點選「加入」按鈕，將特定商品款式加入群組。
	- [ ] 匯入加入：若需加入的商品數量眾多，可使用匯入功能，透過 Excel 檔案批次將商品加入群組。

=== ":material-numeric-2-circle: 設定活動日期"
	- [ ] 設定活動的「開始日期」及「結束日期」。

    ![](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/單品折扣匯入.png)
    ![](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/單品折扣匯入-2.png)

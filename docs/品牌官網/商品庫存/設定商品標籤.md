# 設定商品標籤
為商品建立與管理自訂標籤，以便分類、篩選、行銷應用及控制第三方平台同步。
{ .page-subtitle } 

## 說明概述
=== ":material-lightbulb-outline: 使用情境"

	- 商品分類管理：為不同類型的商品設定標籤，方便快速篩選與管理。
	- 行銷活動應用：將特定商品批次加入標籤群組，以便進行促銷活動或組合銷售。
	- 提升搜尋效率：透過標籤快速定位商品，優化後台操作效率。

## 操作流程
### 步驟一：新增商品標籤

- [ ] 登入 CYBERBIZ 電商後台，前往「商品」→「商品標籤管理」。
- [ ] 點擊「新增標籤」按鈕，輸入標籤名稱，例如：「衣服」。
- [ ] 點擊「儲存」完成新增。

[![標籤群組位置](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組位置.png)](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組位置.png)

### 步驟二：將商品加入標籤群組

您可以透過兩種方式將商品加入標籤群組：

=== ":material-tag-arrow-right-outline: 方法一：手動加入"
     - [ ] 在「商品標籤管理」頁面，找到您要編輯的標籤群組，點擊「新增商品」。
    - [ ] 搜尋並勾選欲加入的商品，例如：「襯衫」。
    - [ ] 點擊「加入」完成。

    [![標籤群組加入](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組加入.png)](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組加入.png)

=== ":material-file-upload-outline: 方法二：透過 Excel 匯入"
    - [ ] 匯出商品 Excel 表格，並在「商品標籤」欄位中填入欲加入的標籤名稱，例如：「衣服」。
    - [ ] 登入 CYBERBIZ 電商後台，前往「商品」→「Excel 大量匯入商品」。
    - [ ] 上傳已編輯完成的 Excel 檔案。

    !!! warning "重要提醒"
        -   透過 Excel 匯入商品時，無法新增商品到不存在的標籤中。
        -   透過 Excel 匯入已在群組內的商品，系統將會略過，不會提醒。

    [![標籤群組Excel1](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組Excel1.png)](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組Excel1.png)
    [![標籤群組Excel2](https://www.cyberbiz.io/support/wp-content/uploads/商品標籤管理設定01.png)](https://www.cyberbiz.io/support/wp-content/uploads/商品標籤管理設定01.png)
    [![標籤群組Excel3](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組Excel3.png)](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組Excel3.png)
    [![標籤群組Excel4](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組Excel4.png)](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組Excel4.png)

### 步驟三：從標籤群組移出商品

- [ ] 在「商品標籤管理」頁面，找到您要編輯的標籤群組。
- [ ] 在商品列表中，點擊欲移出商品旁的「移出」按鈕或「叉叉」圖示。

[![標籤群組移出](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組移出.png)](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組移出.png)

### 步驟四：刪除商品標籤

- [ ] 在「商品標籤管理」頁面，找到欲刪除的標籤群組。
- [ ] 點擊標籤群組旁的「刪除」按鈕。

!!! warning "刪除標籤後，所有已綁定此標籤的商品將會自動取消綁定。"

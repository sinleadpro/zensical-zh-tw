# 新增電子票券

:lucide-lock:{ title="適用方案" } | PLUS 企業

建立新的電子票券商品。

## 操作步驟

### 進入操作頁面
![[ec-電子票券列表-新增票券.png]]
	
- [ ] 登入 CYBERBIZ 電商後台，前往「商品」→「電子票券列表」→「新增票券」，進入「新增票券」頁面。
### 填寫相關資訊
=== "基本設定"
	![[ec-新增票券-基本設定.png]]
		
=== "票券圖片"
	![[ec-新增票券-票券圖片.png]]
		
=== "款式管理"
	![[ec-新增票券-款式管理.png]]
    ??? example "票種範例"
        - 活動票券：如「早鳥票」、「團體票」等，以區分不同票種。
        - 商品兌換券：如「灌籃高手兌換組」、「七龍珠兌換組」等，以分類不同的商品兌換模組。
### 儲存變更
- [ ] 點擊「儲存」，套用變更。
- [ ] 新增的電子票券會出現在「電子票券列表」頁面。
!!! warning "款式規格限制"
	按下儲存後，電子票券不能像一般商品一樣編輯「款式規格」（顏色、尺寸、甚至是自定義的規格）。系統會預設一組名為「票種名稱」的款式規格。

# 商品快速上手

歡迎使用 CYBERBIZ 商品管理快速上手指南！  
本清單幫助您快速建立、整理、上架與管理商品，從單品到批次操作，一步步完成商品管理流程。

快速上手流程分為 **五個主要階段**，每個階段提供建議操作時間與核心任務。

# 電子票券分票
將一個可核銷多次的 QR Code 拆分為多個獨立的 QR Code。
{ .page-subtitle } 

[:material-tag-text:](){ .badge-icon title="適用方案" }[PLUS | 企業](){ .badge-text }
[:material-compass:](){ .badge-icon title="相關指南" }[[電子票券指南]]{ .badge-text }

---

## 說明概述

=== ":material-information-outline: 注意事項"
    -   分票操作不可回復（一個 QR Code 可以拆分為多個，但多個 QR Code 無法合併成一個）。
    -   分票後，原核銷碼即無法再使用。
    -   電子票券為不記名制，只要持有核銷碼，任何人皆可使用，請自行妥善保管。
    -   若需[[電子票券退款|退票]]，僅能由原購票人申請。請顧客勿隨意分票給他人，避免造成後續紛爭。

---

## 操作流程
### 操作分票（顧客）

- [ ] 登入前台會員後，點選「我的帳戶」→「電子票券訂單查詢」，按下「分票」按鈕。

![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券設定33.png)

- [ ] 確認提示資訊，點擊「確認分票」。

![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券設定34.png)

- [ ] 分票後，QR Code 會被拆分成多個（可左右滑動查看其他 QR Code）。
-   分票前的核銷碼為 10 碼，分票後的核銷碼為 25 碼。

![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券設定35.png)
        
### 查看與核銷
#### 查看分票後的票券
- [ ] 登入 CYBERBIZ 電商後台，前往「商品」→「票券核銷列表」。已分票的票券，核銷碼欄位會顯示「查看」按鈕。

![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券設定36.png)

- [ ] 點擊「查看」後，會跳出彈窗顯示該票券所有核銷碼與其核銷狀態。

#### 核銷分票後的核銷碼
分票後，「票券核銷列表」頁將不會有該票券的「前往核銷」按鈕。

- [ ] 登入 CYBERBIZ 電商後台，前往「商品」→「核銷票券頁」。
- [ ] 輸入核銷碼進行核銷。（票後的核銷碼為 25 碼，核銷數量將預設為 1 且不可編輯）。
- [ ] 點擊「查看票券」後，點擊「確認核銷」。

![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券設定37.png)

# 網域管理

當您在第三方平台購買了專屬網域後，需要透過 DNS 設定將該網域指向您的 CYBERBIZ 商店，並在 CYBERBIZ 後台完成綁定。
{ .subtitle }

![](../../assets/images/EC-後台-管理中心-網域管理-新增網域01.png){ .hero-page }

## 使用須知

依您選擇的網域廠商，設定流程通常包含以下兩個關鍵配置：

1.  **CNAME 設定（必要項目）**：將 `www` 開頭的網址（如 `www.yourshop.com`）指向 CYBERBIZ 的伺服器。
2.  **根網域轉址（視廠商而定）**：確保顧客輸入不含 `www` 的網址（如 `yourshop.com`）時，也能順利跳轉至您的商店。




## 各大廠商 DNS 設定指引

請根據您的網域註冊商，參考下方的特定設定說明：

=== "HiNet (中華電信)"

    1. 登入 [HiNet 後台](https://domain.hinet.net/#/dnlogin)。
        ![](../../assets/images/HiNet-後台-網域資訊-新增CNAME01.png)
    2. **CNAME 設定**：進入 **HiNet DNS 代管 > 更新 DNS 紀錄**。刪除原本的 `A 紀錄` 後，新增紀錄。
        ![](../../assets/images/HiNet-後台-網域資訊-新增CNAME02.png)

        - **Ldata(主機名稱/別名)**：`www`  
        - **紀錄類型**：`CNAME`
        - **Rdata(IP/主機名稱)**：填入系統預設網址，例：`xxx.cyberbiz.co`

            > 主機網址後方 **不可加上`.`**。
        
            ![](../../assets/images/HiNet-後台-網域資訊-新增CNAME03.png)


    3. **轉址設定**：進入 **轉址服務 > 設定轉址**。
        ![](../../assets/images/HiNet-後台-網域資訊-設定轉址01.png)

        - **網站標題**：自訂網站標題
        - **來源網址**：留空
        - **目的網址**：填入CNAME中的 **Ldata(主機名稱/別名)**，例：`www...`
        - **網址列顯示**：`目的網址`

            ![](../../assets/images/HiNet-後台-網域資訊-設定轉址02.png)

    4. 複製 **說明** 的 **主機IP**。
        ![](../../assets/images/HiNet-後台-網域資訊-設定轉址03.png)
    5. 前往 **DNS代管 > 更新DNS紀錄**，點選 **新增一列**。
 
        - **紀錄類型**：`A`
        - **Rdata(IP/主機名稱)**：貼上 **主機IP**

        ![](../../assets/images/HiNet-後台-網域資訊-設定轉址04.png)

=== "PChome"

    1. 登入 [PChome 後台](https://myname.pchomebiz.com.tw/)，點擊 **管理我的網址**。
        ![](../../assets/images/PCHOME-後台-管理我的網址-新增CNAME01.png)
    2. **CNAME 設定**：點選 **DNS設定與修改** 欄位的 **進入**。
        ![](../../assets/images/PCHOME-後台-管理我的網址-新增CNAME02.png)

        - **主機**：`www`  
        - **地址**：填入系統預設網址，例：`xxx.cyberbiz.co.`

            > 主機網址後 **請加上 `.`**。

        - **類型**：`CNAME`
        
        ![](../../assets/images/PCHOME-後台-管理我的網址-新增CNAME03.png)

    3. **轉址設定**：

        - **地址**：填入品牌自訂網址，例：`http://www...`
        - **類型**：`redirect`

        ![](../../assets/images/PCHOME-後台-管理我的網址-設定轉址01.jpg)
        

=== "GoDaddy"

    1. 登入 [GoDaddy 後台](https://www.godaddy.com/zh)，點擊網域區域的 **DNS**。
        ![](../../assets/images/Godaddy-後台-網域-新增CNAME01.png)
    2. **CNAME 設定**：選擇 **DNS 記錄** 頁籤，找到 CNAME 類型的網域資料，點選 **編輯**。
        ![](../../assets/images/Godaddy-後台-網域-新增CNAME02.png)

        - **類型**：`CNAME`
        - **名稱**：`www`
        - **內容值**：填入系統預設網址，例：`xxx.cyberbiz.co.`
        - **TTL**：`1天`

        ![](../../assets/images/Godaddy-後台-網域-新增CNAME03.png)

    3. **轉址設定**：前往 **轉址** 頁籤，選擇 **新增網域轉址**。
        ![](../../assets/images/Godaddy-後台-網域-設定轉址01.png)

        - **輸入目標網址**：填入品牌自訂網址，例：`http://www...`

        ![](../../assets/images/Godaddy-後台-網域-設定轉址02.png)

    
=== "Gandi"

    1. 登入 [Gandi 後台](https://www.gandi.net/zh-Hant)，前往 **域名**，點選購買的網域。
        ![](../../assets/images/GANDI-後台-域名-新增CNAME01.png)
    2. **CNAME 設定**：點選 **DNS 紀錄**，點選 **新增紀錄**。
        ![](../../assets/images/GANDI-後台-域名-新增CNAME02.png)

        - **類型**: `CNAME`
        - **TTL(暫存時間)**: `10800`
        - **單位**：`秒`
        - **名稱**：`www`
        - **主機名稱**：填入系統預設網址，例：`xxx.cyberbiz.co.`

            > 主機網址後 **請加上 `.`**。
        
        !!! info "網域設定錯誤排除"
            若於設定過程中出現錯誤訊息，請先刪除 Gandi 帳戶內預設的 CNAME 紀錄，完成後再重新執行設定流程。

        ![](../../assets/images/GANDI-後台-域名-新增CNAME03.png)

    3. **轉址設定**：前往 **域名 > 網頁轉址**，點選 **新增網頁轉址位址**。
        ![](../../assets/images/GANDI-後台-域名-設定轉址01.png)
        ![](../../assets/images/GANDI-後台-域名-設定轉址02.png)

        - **位置**：選擇 `http:// + https://`，後方欄位留空
        - **轉址至**：選擇 ` http://`，填入品牌自訂網址，例：`http://www...`
        ![](../../assets/images/GANDI-後台-域名-設定轉址03.png)

=== "TWNIC"

    1. 登入 [TWNIC 後台](https://rs.twnic.tw/cgi-bin/setupdns.cgi)，前往 **DNS 設定服務 > DNS 設定/代管**。
    
        - **網域名稱**：填入您欲新增的網域，選擇網址後綴
        - **判別密碼**：輸入密碼

        ![](../../assets/images/TWNIC-後台-DNS設定代管-新增CNAME01.png)

    2. **CNAME 設定**：選擇 **主機模式(TWNIC代管)**。

        - **域名/主機名稱**：填入品牌自訂網址，例：`www...com.tw`
        - **紀錄型態**：`CNAME`
        - **IP 位址/主機名稱**：填入系統預設網址，例：`xxx.cyberbiz.co.`

            > 主機網址後 **請加上 `.`**
        
        ![](../../assets/images/TWNIC-後台-DNS設定代管-新增CNAME02.png)

=== "網路中文"        

    1. 登入[網路中文後台](https://www.net-chinese.com.tw/)，前往 **域名管理 > 域名總覽**，點選購買的網域。
        ![](../../assets/images/網路中文-後台-域名管理-新增CNAME01.png)
    2. **CNAME 設定**：左側選單選擇 **DNS > DNS紀錄設定**，輸入以下資訊：

        - **記錄類型**：`CNAME`
        - **主機名稱/別名**：`www`
        - **IP/域名**：填入系統預設網址，例：`xxx.cyberbiz.co.`

        ![](../../assets/images/網路中文-後台-域名管理-新增CNAME02.png)

=== "台灣大哥大"

    1. 登入[台灣大哥大後台](https://domains.tfn.net.tw/domains/session/showConsoleIndex.do)，前往 **產品服務 > 域名申請服務 > 轉址設定**。
        ![](../../assets/images/台灣大哥大-後台-轉址設定-新增CNAME01.png)
    2. **CNAME 設定**：點選  **DNS代管**。
        ![](../../assets/images/台灣大哥大-後台-轉址設定-新增CNAME02.png)

        - **設定類別**：`CNAME(別名)`
        - **別名**：`www`
        - **目標主機名稱**：填入系統預設網址，例：`xxx.cyberbiz.co`

        ![](../../assets/images/台灣大哥大-後台-轉址設定-新增CNAME03.png)

=== "遠振"

    1. 登入[遠振後台](https://twnoc.net/whmcs/login)，前往 **網域 > Manage DNS**。
        ![](../../assets/images/遠振-後台-ManageDNS-新增CNAME01.png)
    2. **CNAME 設定**：選擇您購買的網域，點選 :lucide-pencil:，點擊 **Add Record**。
        ![](../../assets/images/遠振-後台-ManageDNS-新增CNAME02.png)

        - **TYPE**：`CNAME`
        - **NAME**：`www`
        - **TTL**：`14400`
        - **Cname**：填入系統預設網址，例：`xxx.cyberbiz.co.`
        ![](../../assets/images/遠振-後台-ManageDNS-新增CNAME03.png)

=== "捕夢網"

    1. 登入[捕夢網後台](https://www.pumo.com.tw/)，前往 **網域管理**。
        ![](../../assets/images/捕夢網-後台-網域管理-新增CNAME01.png)
    2. **CNAME 設定**：選擇您購買的網域，並點選下拉選單，選擇 **設定DNS紀錄**。
        ![](../../assets/images/捕夢網-後台-網域管理-新增CNAME02.png)

        - **新增類型**：`CNAME`
        - **指定主機名稱**：`www`
        - **CNAME別名**：填入系統預設網址，例：`xxx.cyberbiz.co`

        ![](../../assets/images/捕夢網-後台-網域管理-新增CNAME03.png)




=== "遠傳"

    1. 登入[遠傳後台](https://rs.seed.net.tw/)，前往 **我的域名管理**。
        ![](../../assets/images/遠傳-後台-我的域名管理-新增CNAME01.png)
    2. 填入帳號資訊。

        - **統編/身分證字號**：依身分填入
        - **密碼**：依身分填入
        - **網域名稱**：填入您欲新增的網域，選擇網址後綴
        ![](../../assets/images/遠傳-後台-我的域名管理-新增CNAME02.png)
    3. 選擇 **DNS ( Host / IP 指定)**。
        ![](../../assets/images/遠傳-後台-我的域名管理-新增CNAME04.png)
    4. 選擇 **遠傳 DNS 代管模式**。
        ![](../../assets/images/遠傳-後台-我的域名管理-新增CNAME05.png)
    5. **CNAME 設定**：
        - **Host**：`www`
        - **紀錄型態**：`CNAME`
        - **IP/ Domain/內容**：填入系統預設網址，例：`xxx.cyberbiz.co`
        ![](../../assets/images/遠傳-後台-我的域名管理-新增CNAME06.png)
    6. 完成後點選 **代管設定**。
        ![](../../assets/images/遠傳-後台-我的域名管理-新增CNAME07.png)


## CYBERBIZ 後台新增網域

完成廠商端的 DNS 設定後，請回到 CYBERBIZ 進行最後綁定：

1.  登入CYBERBIZ 後台，前往 **管理中心 > 網域管理**。
2.  點選 **新增一個您已經擁有的網域**，輸入完整網址（如：`www.yourshop.com`）。
3.  勾選 **總是將顧客重新導向到這裡**。


!!! info "生效時間"
    - DNS 設定生效通常需要 **24 至 48 小時**。
    - 生效約 1 小時後可於 CYBERBIZ 後台填入網址。
    - 可使用 [DNS Checker](https://dnschecker.org/#CNAME/) 檢查您的網域是否已正確指向。

![](../../assets/images/EC-後台-管理中心-網域管理-新增網域01.png)

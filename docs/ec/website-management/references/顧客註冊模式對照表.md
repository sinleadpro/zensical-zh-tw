# 顧客帳號註冊模式對照表 { #ref-customer-registration-modes }

「顧客註冊設定」頁面最上方的「**顧客註冊**」設定，決定顧客是否必須先註冊才能下單，有兩種模式可選。

| 模式 | 加入購物車 | 註冊時機 | 適用情境 |
|:--|:--|:--|:--|
| **快速結帳並註冊** | 不需登入，可匿名加入 | 結帳時(可選擇登入，或結帳後成為會員) | 一般電商，降低結帳門檻 |
| **必須註冊** | 需先登入或註冊 | 加入購物車前必須完成註冊 | 會員制商店、批發網站、需要嚴格管控銷售對象的店家 |

!!! note "註釋"
    * 「**快速結帳並註冊**」是大多數商店的預設模式，允許顧客匿名加入購物車，結帳時再決定是否成為會員。
    * 「**必須註冊**」搭配 [Email + 手機雙重驗證](../設定顧客 Email 與手機雙重驗證.md){ data-preview } 時，顧客必須先完成兩項驗證後才能開始購物。
    * 切換註冊模式不會影響既有會員，既有會員仍可正常登入購物。

# 匯出分潤報表
透過分潤報表，您可以精確統計各類推廣對象的業績成效與應分潤金額。系統提供「結案後」的正式報表，以及「結案前」的即時訂單覽表。
{ .subtitle }

[:lucide-layers:{ title="適用產品" }](../../resources/conventions#適用產品) | EC / POS
[:lucide-tag:{ title="適用方案" }](../../resources/conventions#適用方案) | 進階 / 高手 / 所有PLUS / 企業
{ .doc-badge }


!!! tip "應用情境"
	- **每月對帳結算**：每月初匯出上月 **已結案** 的分潤總表，進行獎金發放。
	- **第三方成效追蹤**：提供專屬連結給網紅或團媽，讓其即時查看推廣訂單明細。
	- **即時業績監控**：在訂單結案前，透過訂單報表預先掌握潛在的分潤支出。


## 使用須知

- **權限差異**：
    - **網站擁有者**：可查詢全站所有人員的分潤數據。
    - **協同管理者 (員工)**：僅能查詢自己的分潤成效。


## 操作流程

### 任務一：匯出正式分潤報表

適用於「推薦、註冊、結帳、取件」等所有分潤方案的結算。

1. 登入 CYBERBIZ 管理後台，前往 **分潤 > 分潤報表**。
2. 在 **報表類型** 下拉選單中選擇欲查詢的類型。
3. 選擇具體的報表名稱：
    - **推薦分潤**：可選擇 **推薦分潤總表** 或 **個人別推薦分潤報表**。
    - **註冊分潤**：可選擇 **註冊分潤總表** 或 **員工別註冊分潤報表**。

    !!! note "報表內容解析"
        - **分潤總表**：呈現特定月份內，**所有合作夥伴** 各自的推薦分潤總額。適合用於每月財務結算與大數核對。
        - **個人別推薦分潤報表/員工別註冊分潤報表**：呈現特定月份內， **單一合作夥伴** 的所有推薦訂單明細及分潤金額列表。適合提供給夥伴核對每一筆訂單的獲酬詳情。

    - **結帳人/取件分潤**：直接選擇對應的總表或個人報表。
4. 選擇欲匯出的 **月份**。
5. 設定 **收件者 Email**，並點選 **匯出報表**。
    > 系統將於背景處理並發送下載連結至您的電子信箱。

!!! note "結算與產出規範"
    - **結算準則**：正式分潤報表僅採計 **已結案** 的訂單。若訂單於 12 月底成立，但在 1 月才結案，該筆數據將出現在 1 月的報表中。
    - **資料範圍**：報表匯出以 **單一月份** 為單位，不可跨月匯出。

![](../../../assets/images/EC-後台-分潤-分潤報表-畫面總覽01.png){ .screenshot }

### 任務二：匯出即時分潤資訊

若需在訂單結案前預覽分潤狀況，可使用訂單報表功能。

1. 前往 **訂單 > 訂單報表匯出**。
2. 在 **報表可用欄位** 中，勾選以下分潤相關欄位：
    - **使用推薦分潤**
    - **分潤人名稱**
    - **分潤比例**
    - **分潤金額**
    - **推薦代碼**
3. 篩選欲查詢的訂單狀態。
4. 設定時間區間與 Email 後，點選 **匯出**。

![](../../../assets/images/EC-後台-訂單-訂單報表匯出-匯出分潤訂單01.png){ .screenshot }

## 常見問題

??? quote "推薦人反應查不到某筆訂單，該如何處理？"
    1. 確認顧客在下單時是否確實輸入推薦碼或點擊推薦連結。
    2. 檢查該訂單是否在方案的 **生效期間** 內成立。
    3. 若為第三方查詢連結，請確認該訂單是否已出現在後台的 **分潤查詢** 中。

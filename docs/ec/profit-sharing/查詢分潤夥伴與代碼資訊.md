# 查詢分潤夥伴與代碼資訊
您可以隨時在後台查詢所有分潤合作夥伴（包括外部網紅、站內會員與內部員工）所擁有的分潤方案與專屬代碼，以便隨時提供推廣資訊給合作夥伴。
{ .subtitle }

[:lucide-layers:{ title="適用產品" }](../../resources/conventions#適用產品) | EC / POS
[:lucide-tag:{ title="適用方案" }](../../resources/conventions#適用方案) | 進階 / 高手 / 所有PLUS / 企業
{ .doc-badge }

!!! tip "應用情境"
	- **協助夥伴領取代碼**：當合作網紅忘記其專屬推薦碼時，管理員可快速查詢並告知。
	- **核對方案綁定狀況**：確認特定員工是否已被正確加入預期的分潤方案中。
	- **驗證推薦關係**：當顧客反應推薦無效時，可先核對推薦人的代碼是否正確。


## 使用須知

- **查詢權限**：
    - **網站擁有者**：具備最高權限，可查詢 **本人** 及 **所有其他使用者** 的分潤資料。
    - **協同管理者**：僅能點選 **查看我的分潤方案**，查閱與自己相關的資料。


## 操作流程

### 任務一：查詢合作夥伴的分潤資訊

適用於查詢已加入分潤方案的所有對象。

1. 登入 CYBERBIZ 管理後台，前往 **分潤 > 分潤查詢**。
2. 若您是 **網站擁有者**，您將看到搜尋欄位：
    - 在搜尋框中輸入夥伴的 **姓名、信箱** 或 **手機號碼**。
    - 系統會自動帶出符合條件的清單，請點選目標對象。
3. 系統將列出該對象參與的所有分潤方案資訊，包括：
    - **分潤類型**：推薦分潤、註冊分潤等。
    - **方案名稱**：目前綁定的具體方案。
    - **分潤代碼**：該對象在該方案中的專屬推薦碼或註冊代碼。
    - **分潤比例**：對應的線上/線下抽成趴數。

![](../../../assets/images/EC-後台-分潤-分潤查詢-畫面總覽01.png){ .screenshot }

### 任務二：員工自我查詢分潤資訊

適用於門市人員或官網小編查詢自己的推廣代碼。

1. 員工登入後台後，前往 **分潤 > 分潤查詢**。
2. 點選頁面中的 **查看我的分潤方案**。
3. 頁面將直接顯示該員工目前所屬的所有分潤方案及對應的代碼連結。

    > 員工可直接點選連結圖示進行複製，或產出 QR Code 用於推廣。


### 任務三：提供第三方推薦人外部查詢連結

針對外部合作夥伴（如團媽、網紅），您可以提供專屬連結，讓其在不進入後台的情況下即時查看成效。

1. 前往 **分潤 > 推薦人分潤**，點擊 **第三方總表** 頁籤。
2. 找到指定合作夥伴，複製頁面中的 **報表連結** 與 **密碼**，提供給推薦人。
   
    > 此報表包含 **未結案** 訂單，讓推薦人能即時掌握推廣狀況。

![](../../../assets/images/EC-後台-分潤-推薦人分潤-查看第三方分潤夥伴名單01.png){ .screenshot }

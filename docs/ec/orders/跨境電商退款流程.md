# 訂單退款流程

當訂單完成退貨流程後，最終步驟即是將款項退還給消費者。系統會依據您的站台方案與消費者的支付方式，自動或手動觸發退款流程。
{ .subtitle }

[:lucide-layers:{ title="適用產品" }](../../resources/conventions#適用產品) | 跨境電商 (北美站 / 日本站 / 東南亞站)
[:lucide-tag:{ title="適用方案" }](../../resources/conventions#適用方案) | Pro / Business
{ .doc-badge }


!!! info "退款核心規則"
    - **次數限制**：每筆訂單僅接受 **一次** 退款申請。
    - **退款生效**：當日付款之訂單無法立即退款，需 **隔日** 系統結算後方可操作。

## 步驟 2：操作退款步驟

退款流程會隨著 **退貨狀態** 的變動而自動觸發。

=== "系統自動退款"

    1. 商家將訂單 `退貨狀態` 改為 **已退貨** 或 **部分退貨**。
    2. 系統自動將 `付款狀態` 更新為 **待退款** 或 **待部分退貨**。
        ![](../../assets/images/EC-後台-訂單-所有訂單-訂單顯示待退款待部分退款01.png)
    3. 系統即刻觸發金流退刷請求。
    4. 退刷成功後，`付款狀態`自動更新為 **已退款** 或 **已部分退款**。
        ![](../../assets/images/EC-後台-訂單-所有訂單-訂單顯示已退款01.png)

=== "自行匯款 / Fiuu 錢包退款"

    1. 商家將訂單 `退貨狀態` 改為 **已退貨** 或 **部分退貨**。
    2. 系統自動將 `付款狀態` 更新為 **待退款** 或 **待部分退貨**。
    3. 
        - **自行匯款**：商家與消費者聯繫並完成線下匯款。
        - **Fiuu 錢包退款**：前往 Fiuu 後台，完成退款。
    4. 返回後台，勾選該筆訂單。
    5. 點擊右上方 **更多操作** > 將付款狀態改為 **已退款** 或 **部分退款**。

    ![](../../assets/images/EC-後台-訂單-所有訂單-操作已退款01.png)

本對照表列出新竹物流託運單各規格的 **Cyber 幣** 扣抵金額,以及低溫不受理品項與單號時效。實際扣抵以後台「金物流」>「新竹物流託運單」>「加印託運單」的「選擇貨品大小」下拉選單顯示為準。

## 費率對照表 { #hct-rate-card }

金額均為 **Cyber 幣**。三邊總和以最長邊+次長邊+最短邊計算。

=== "常溫"

    | 三邊總和 | 費用(Cyber 幣) |
    | :-- | --: |
    | 60cm | 80 |
    | 90cm | 90 |
    | 120cm | 100 |
    | 150cm | 125 |
    | 160cm | 150 |
    | 170cm | 175 |
    | 180cm | 215 |

=== "低溫(冷藏 / 冷凍)"

    冷藏與冷凍同價。

    | 三邊總和 | 費用(Cyber 幣) |
    | :-- | --: |
    | 60cm | 145 |
    | 90cm | 180 |
    | 120cm | 220 |

!!! note "註釋"
    - 若您的店家與 CYBERBIZ 已協議客製費率,系統會自動套用,實際扣抵以後台下拉選單顯示為準。
    - **促銷期間** 部分尺寸可能會有優惠價,下拉選單會額外標示促銷訊息;系統會以優惠價扣抵。
    - **PLUS 版 / 企業版** 不需預先儲值 Cyber 幣,費用列入每期對帳單,費率同上。
    - 若遇特殊材積、貨品內容、特殊寄送地點(例如離島、偏遠地區)等,應依物流商之特殊規定計價收費辦理;詳細資訊以新竹物流公告為準。

## 尺寸與重量上限 { #hct-rate-card-limits }

新竹物流的尺寸與重量上限依溫層而異,超出範圍將無法寄送:

| 溫層 | 三邊總和上限 | 單邊上限 | 重量上限 |
| :-- | :-- | :-- | :-- |
| 常溫 | 180cm | — | 20kg |
| 低溫(冷藏 / 冷凍) | 120cm | 60cm | 20kg |

## 低溫不受理品項 { #hct-rate-card-restricted }

下列品項 **無法** 透過新竹物流低溫配送,寄出前請務必確認商品類別:

- **生醫類**:疫苗、血液 / 尿液檢體、冷藏藥品
- **易變質食品**:
    - 鮮奶油蛋糕
    - 母奶
    - 冰淇淋
    - 未密封包裝之榴槤、鹹魚
    - 保存期短於 3 天之商品

!!! warning "若包裹被退回"
    若新竹物流發現包裹含不受理品項,可能 **退回寄件人** 或拒絕配送,且不退回 Cyber 幣。出貨前請確認商品符合規範。

## 計費規則與單號時效 { #hct-rate-card-billing }

### 扣費時點 { #hct-rate-card-billing-timing }

- **一般版**(基本版、達人級、進階版、高手版、專業版、POS 獨賣):列印託運單時 **立即扣除 Cyber 幣**,餘額不足將擋下列印。請先至「儲值中心」儲值。
- **PLUS / 企業版**(專業 plus 版、進階 plus 版、高手 plus 版、企業版、尊爵版):不需儲值,費用 **列入每期對帳單**。

### 託運單 14 日失效 { #hct-rate-card-billing-expiry }

託運單產出後 **14 日內未實際寄件**,單號會自動失效:

- 單號狀態改為 **「取消寄件」**。
- 一般版已扣除的 Cyber 幣會 **全額回補**(可於「金物流」>「新竹物流託運單」>「單號使用紀錄」確認對帳備註)。
- 失效後若仍要寄件,需重新從訂單列表下載或加印託運單,系統會產生新單號並重新扣費。

### 逆物流預扣 { #hct-rate-card-billing-reverse }

由商家建立的新竹逆物流(於「金物流」>「新竹物流託運單」頁面建立)會 **預扣常溫 60cm = 80 Cyber 幣**,實際運費以包裹實際尺寸對帳調整。**若未順利收回包裹,不收費**。

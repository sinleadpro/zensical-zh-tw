

# 訂單列表操作功能

=== "狀態變更操作" 

	|**操作項目**|**適用情境與說明**|
	|---|---|
	|**結案訂單**|將訂單狀態標記為「已完成」或「結案」，通常用於已取貨且過鑑賞期之訂單。|
	|**開啟訂單**|將「已結案」的訂單重新恢復為「進行中」狀態。|
	|**更改退貨狀態**|手動變更售後進度，選項包含：**退貨中**、**退貨審查**、**已退貨**、**拒絕退貨**。|

=== "配送與物流操作" 

根據不同的物流類型（串接或自訂），請選擇對應的出貨操作：

	- **串接物流 / 超商物流：**
	    
	    - **下載託運單並改為已出貨：** 系統將自動產生物流單據，並同步將配送狀態更新為「已出貨」。
	        
	- **自訂物流 (Custom Logistics)：**
	    
	    - **更改配送狀態：** 手動切換為「未出貨」、「準備出貨」或「已出貨」。
	        
	    - **大量匯入自訂物流託運單號：** 於畫面左上角下載範本，填寫單號後上傳，即可批次更新多筆訂單貨態。
	        

=== "文件列印"

支援批次或單筆列印以下出貨文件：

- **列印訂單明細：** 包含訂單商品內容、金額及收件資訊，供內部查核或隨貨附帶。
    
- **列印揀貨單：** 依據訂單內容彙整商品清單，便於倉庫人員進行揀貨作業。
    

### 4. 訂單標籤管理 (Order Tagging)

用於自定義分類，便於後續篩選與管理：

- **新增標籤：** 為選定訂單貼上自定義標記（如：VIP 客戶、急件）。
    
- **移除標籤：** 批次撤銷不再適用的標記。

# 超商大宗寄倉 B2C 方案與計費對照

  各方案可開通哪些 B2C 通路，以及運費的扣款方式。供「設定超商大宗寄倉 B2C」（適用版本）與「使用超商大宗寄倉 B2C 出貨」（運費扣款提示）共用。
  { .subtitle }

  ## 各方案可開通的超商通路 { #reference-cvs-b2c-plans-eligibility }

  CYBERBIZ 平台依商家「金物流串接方式」分為兩條開通路徑：

  === "走 CYBERBIZ 帳號（自帶金物流）"

      | 方案 | 可開通通路 |
      | :-- | :-- |
      | 企業版 | 7-ELEVEN、全家、萊爾富 |
      | PLUS 版（高手 PLUS、進階 PLUS、老闆 PLUS） | 7-ELEVEN、全家、萊爾富 |
      | 高手版、進階版 | 7-ELEVEN、全家、萊爾富 |
      | 其他方案 | 不適用，請洽客服升級 |

  === "串接綠界金物流"

      | 方案 | 可開通通路 |
      | :-- | :-- |
      | 高手版 | 透過綠界 B2C 大宗寄倉服務 |
      | 其他方案 | 不適用，請洽綠界科技辦理 |

  !!! note "註釋"
      * 通路是否真正可用，仍需通過該超商的「測標審核」流程。
      * 即使方案支援，個別通路仍可能因合約或業務需求未開放，請以後台「B2C 超商大宗」設定頁實際顯示為準。
      * 若不確定您的方案是否包含此功能，請聯絡 CYBERBIZ 客服確認。

  ---

  ## 計費方式 { #reference-cvs-b2c-plans-billing }

  下載託運單時的運費扣款方式依方案分為兩種：

  | 方案類型 | 扣款方式 | 餘額 / 額度不足時 |
  | :-- | :-- | :-- |
  | 一般版 / 高手版 / 進階版 | 即時扣除 **CYBER 幣** | 無法下載託運單，系統提示「CYBER 幣餘額不足」 |
  | PLUS 版、企業版 | 列入「**對帳單**」，月結 | 不影響當下下載，月底統一結算 |

  !!! tip "技巧"
      * CYBER 幣可於後台「商店設定 > CYBER 幣」儲值，建議在大量出貨日前先確認餘額。
      * 對帳單可於「報表 > 對帳單」查詢當月累計運費。
      * 補印託運單（已下載過的訂單再次下載）**不會重複扣款**。

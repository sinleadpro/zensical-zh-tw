## 訂單各項折扣拆分


| 商品折扣欄位 | 說明 | 備註 |
| :--- | :--- | :--- |
| 商品任選折扣總金額 | 針對「任選 X 件 Y 折」或「任選滿額折」活動之折抵總額 | |
| 商品全館折扣金額 | 套用全站促銷活動（如全館 9 折）所分攤的折扣金額 | |
| 商品 VIP 折扣總金額 | 根據會員等級（VIP/VVIP）設定之優惠所產生的折扣 | |
| 商品優惠券折扣總金額 | 顧客於結帳使用優惠券（Coupon）後分攤之金額 | |
| 商品紅利折扣總金額 | 顧客使用紅利點數（Points）進行現金折抵之金額 | |
| 商品店長改價折扣總金額 | 由門市人員於結帳時手動修改價格產生的折扣差額 | **POS 限定** |
| 商品單品折扣總金額 | 針對單一商品直接設定特價與原價之間的差額 | |
| 商品組合優惠折扣總金額 | 針對「組合商品（A+B）」活動所產生的優惠折扣 | |
| 商品分類優惠折扣總金額 | 針對特定分類設定之滿額/滿件折扣（含多層級） | |
| 商品第三方折扣總金額 | 來自外部串接平台（如 LINE 購物）之專屬折抵 | |
| 商品遊戲折扣總金額 | 透過行銷互動遊戲（如轉盤、刮刮卡）獲得之折抵 | |
| 商品會員專屬價格折扣總金額 | 針對特定會員身份直接設定的「固定專屬價」與原價價差 | |

## UTM 參數


| UTM 參數名稱 | 系統對應欄位 | 說明 |
| :--- | :--- | :--- |
| **「UTM 來源」** | `utm_source` | 流量的來源平台，例如：google, facebook, newsletter |
| **「UTM 活動名稱」** | `utm_campaign` | 特定的促銷活動或產品推廣名稱，例如：spring_sale |
| **「UTM 媒介」** | `utm_medium` | 行銷的媒介類型，例如：cpc, email, social |
| **「UTM 活動字詞」** | `utm_term` | 通常用於關鍵字廣告，標記搜尋的關鍵字 |
| **「UTM 點擊時間」** | `utm_click_time` | 顧客點擊該追蹤連結並進入網站的具體時間紀錄 |
| **「UTM 活動內容」** | `utm_content` | 用於區分相同連結的不同版本（A/B 測試），例如：logolink, textlink |

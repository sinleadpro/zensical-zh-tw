黑貓快速到店（常溫 / 冷藏 / 冷凍）為與超商合作的特殊物流，其逾期判定天數與退貨計費規則與一般 7-11、全家、萊爾富超商不同。本文說明逾期判定的時間點與規費計算機制。

## 功能介紹 { #intro-tcat-quick-store-overdue }

黑貓快速到店物流由黑貓宅急便將商品配送至顧客指定超商門市，若顧客未在期限內取貨，商品會由黑貓退回。退貨運費由黑貓依其官方計費標準向商家收取。

## 逾期判定天數 { #specs-tcat-quick-store-overdue-days }

| 溫層 | 物流代號 | 逾期判定 |
| :-- | :-- | :-- |
| 常溫 | 黑貓快速到店-常溫 | 出貨後 4 日（D+3）顧客未取，系統判定為逾期 |
| 冷藏 | 黑貓快速到店-冷藏 | 同上 |
| 冷凍 | 黑貓快速到店-冷凍 | 同上 |

!!! note "註釋"
    逾期判定天數由 CYBERBIZ 依黑貓官方規則設定，商家不可調整。

## 規費計算機制 { #reference-tcat-quick-store-overdue }

### 計費邏輯 { #reference-tcat-quick-store-overdue-billing }

黑貓快速到店訂單若顧客逾期未取，會產生 **退貨運費**。實際向商家收取的總運費為：

> **出貨運費 + 退貨運費**

* **出貨運費：** 顧客下單時即依材積扣抵 Cyber 幣的運費
* **退貨運費：** 顧客逾期未取後，由黑貓再扣抵一筆運費

---

### 費率依據 { #reference-tcat-quick-store-overdue-rate }

| 溫層 | 計費依據 |
| :-- | :-- |
| 黑貓快速到店-常溫 | 依黑貓官方常溫材積費率（依包裹尺寸 / 重量） |
| 黑貓快速到店-冷藏 | 依黑貓官方冷藏材積費率 |
| 黑貓快速到店-冷凍 | 依黑貓官方冷凍材積費率 |

!!! warning "注意"
    退貨運費的實際金額依黑貓官方公告的最新費率為準，並依包裹材積（如 60cm、90cm、120cm 等）有不同單價。請以您與 CYBERBIZ / 黑貓簽訂的最新「黑貓快速到店合約規範」為準。

---

### Cyber 幣扣抵 { #reference-tcat-quick-store-overdue-coin }

* 出貨與退貨運費皆以 **Cyber 幣** 扣抵商家帳戶
* 每筆扣抵都會記錄在後台的 Cyber 幣使用紀錄中，可供日後對帳

## 注意事項 { #specs-tcat-quick-store-overdue-notes }

* **「快速到店-常溫」訂單只能搭配常溫運費規格**，冷藏 / 冷凍同理。混用會被系統擋下並提示對應錯誤。
* 黑貓快速到店訂單逾期未取的退款條件，與一般超商物流相同，請見 [自動退款支援對照](退貨與退款狀態對照（超商訂單逾期未取）.md#reference-overdue-cvs-refund-flow-auto-refund){ title="退貨與退款狀態對照（超商訂單逾期未取）" }。

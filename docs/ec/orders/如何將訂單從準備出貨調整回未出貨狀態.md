## 訂單調整回未出貨說明

當訂單處於「準備出貨」狀態時，商家可以手動將其調整回「未出貨」狀態，主要的目的是為了 **讓訂單恢復成可被取消的狀態，以便消費者在前台執行取消訂單的操作**。

以下為調整貨態的詳細說明與教學：

### **一、 設定前注意事項**
在執行狀態調整前，請務必確認訂單的「退貨狀態」，否則將無法變更：
*   **支援的情境**：退貨狀態必須為 **「不需退貨」** 或 **「拒絕退貨」**。
*   **不支援的情境**：若訂單已進入「退貨中」、「退貨審查」或「已退貨」等狀態，則無法將配送狀態改回未出貨。

### **二、 操作步驟教學**
**【後台路徑】：** 「訂單」 → **「所有訂單」**

1.  **勾選訂單**：在訂單列表中，找到配送狀態為 **「準備出貨」** 的訂單並勾選。
2.  **選擇操作**：點擊頁面右上方（新版介面）的 **「更多操作」** 或 **「選擇操作」** 下拉選單。
3.  **執行變更**：在選單中點選 **「未出貨」** 選項。
4.  **完成調整**：點選後，該筆訂單的配送狀態即會變更為「未出貨」，此時消費者即可在官網前台看到取消訂單的按鈕（若商家有開啟顧客取消訂單功能）。

### **三、 相關功能提示**
*   **消費者端影響**：當配送狀態為「準備出貨」時，消費者通常不被允許取消訂單；一旦調回「未出貨」，權限即會恢復。
*   **商家備註**：若您只是要修改收貨人資訊或配送日期，在「未出貨」狀態下亦可直接於訂單明細頁面進行修改。

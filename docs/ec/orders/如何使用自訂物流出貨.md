## 自訂物流出貨說明

「**自訂物流**」功能允許商家使用自己合作的物流廠商進行出貨，適用於非系統串接的物流需求（如大榮物流、嘉里大榮或商家自送等）。


## 功能使用前提與限制

在執行操作前，請務必確認訂單符合以下條件：

*   [x] **配送狀態**：僅支援配送狀態為「**未出貨**」或「**準備出貨**」的訂單。
*   [x] **付款與物流限制**：目前僅支援「**已收到款項**」且使用「**宅配**」配送的訂單；**不支援**超商取貨（不論付款與否）及宅配貨到付款訂單。
*   [x] **不可逆性**：訂單貨態一旦變更為「已出貨」，將**無法退回**原始狀態。
*   [x] **無文件下載**：此操作僅修改系統貨態，**不提供**託運單檔案下載，商家需自行準備物流單據。


## 操作出貨三種方法

### 勾選多筆訂單快速出貨

此方式適用於大批量處理，但**訂單內不會有託運單號**供會員查詢。

1.  **進入路徑**：管理後台 →「訂單」→「所有訂單」。
2.  **勾選訂單**：選取欲出貨的訂單（需為相同配送方式）。
3.  **執行操作**：點擊右上方「選擇操作」或「更多操作」→ 點選「**已出貨**」。
4.  **確認資訊**：選擇使用的自訂貨運公司，勾選是否發送 Email 通知顧客，點選「確認出貨」完成。

### 單筆訂單出貨（可回填單號）

此方式可將託運單號提供給會員，方便其自行查詢貨態。

1.  **進入訂單**：在訂單列表中點擊「訂單編號」進入明細頁。
2.  **選擇商品**：勾選要出貨的商品品項。
3.  **填寫資訊**：
    *   出貨方式選擇「**自訂出貨方式**」。
    *   輸入「**快遞單號**」並選擇「**託運商名稱**」。
    *   勾選是否發送郵件通知顧客。
4.  **確認出貨**：點選確認後，貨態即轉為「已出貨」。

### 大量匯入自訂物流託運單號

適合需要一次為多筆訂單回填不同單號的情境。

1.  **下載範本**：在訂單列表頁點擊「**大量匯入自訂物流託運單號**」，並下載 Excel 範本。
2.  **填寫 Excel**：
    *   **訂單編號（必填）**：須與後台顯示內容完全一致（包含符號）。
    *   **託運單號（選填）**：填入物流商提供的單號。
    *   **物流商名稱（選填）**：須填寫系統「物流商列表」分頁中存在的名稱。
3.  **上傳檔案**：系統會自動檢查格式，並發送通知信告知匯入成功與失敗的筆數及原因。
4.  **完成更新**：匯入成功的訂單，配送狀態將自動更新為「已出貨」並發送通知信。

## 特殊情境處理

*   **一單多包裹**：若一筆訂單需拆成多個包裹，在 Excel 匯入時，可在託運單號欄位使用「**；**」（英文鍵盤分號）隔開多組單號。
*   **顧客自取**：請使用「自訂物流」出貨流程，「快遞單號」欄位可隨意填入文字，並手動點選確認出貨。
*   **手寫宅配單**：若為貨到不付款訂單，可使用自訂物流搭配手寫單（注意請填寫商家自有的客代，勿填 CYBERBIZ 客代）。

## 常見問題與注意事項

*   **貨態追蹤**：自訂物流未與 CYBERBIZ 系統串接，系統**無法自動追蹤**配送進度，商家需自行追蹤，系統最終貨態即為「已出貨」。
*   **行銷活動觸發**：紅利點數、滿額贈優惠券及分潤計算通常在「已結案」狀態觸發。因自訂物流訂單不會自動轉為已收貨，商家需**手動結案**或開啟「**出貨後 N 天自動結案**」功能。
*   **帳務認列**：對於 PLUS 或企業版商家，自訂物流訂單在「**已出貨**」時即會認列至當期對帳單中。

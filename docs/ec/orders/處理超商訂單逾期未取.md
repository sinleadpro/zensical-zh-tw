超商訂單逾期未取的處理流程，包含自動退款條件、大宗寄倉與店到店退貨方式、人工退款操作步驟。
{ .subtitle }

## 超商訂單逾期處理說明

當顧客未在期限內至超商取貨，訂單會被超商系統判定為「逾期未取」。本文說明系統自動發生的變動、不同物流類型的退貨流程，以及商家需要操作的人工退款步驟。

## 功能介紹 { #intro-overdue-cvs-orders }

CYBERBIZ 串接的超商物流（7-11、全家、萊爾富、黑貓快速到店）會在顧客逾期未取後，由超商端主動回報，CYBERBIZ
系統自動將訂單推進到「逾期未取」狀態，並視條件觸發退款與發票折讓。商品則依出貨方式（大宗寄倉或店到店）退回至物流總倉或原寄件門市。

**系統自動執行**：

* 配送狀態由「已出貨」變更為「逾期未取」
* 退貨狀態設為「轉運中心」
* 若符合自動退款條件，付款狀態改為「待退款」並執行退款
* 若已開立發票且未關閉自動作廢，執行發票作廢或折讓

**商家需要做**：

* 依物流類型在物流總倉或原寄件門市領回商品
* 對於系統無法自動退款的訂單，至訂單詳情頁填寫顧客退款資訊執行人工退款
* 通知顧客若仍想購買必須重新下單，逾期訂單 **不能** 再次出貨

!!! info "提示"
    逾期未取訂單無法重新出貨。若顧客仍想購買，請引導顧客重新下單，原訂單依本文流程結案。

## 使用前提與限制 { #prerequisites-overdue-cvs-orders }

### 適用範圍 { #prerequisites-overdue-cvs-orders-scope }

「逾期未取」狀態的自動判定僅適用於 **CYBERBIZ 串接的超商物流**：

* 7-11 一般取貨 / 取貨付款（大宗寄倉 B2C）
* 7-11 交貨便（店到店 C2C）
* 全家一般取貨 / 取貨付款（大宗寄倉 B2C）
* 全家店到店（C2C）
* 全家冷鏈
* 萊爾富一般取貨 / 取貨付款（大宗寄倉 B2C）
* 萊爾富店到店（C2C）
* 黑貓快速到店（常溫 / 冷藏 / 冷凍）

使用「自訂出貨方式」或一般宅配（黑貓宅急便、宅配通、新竹物流、順豐速運）的訂單，系統不會自動將其轉為「逾期未取」。

---

### 顧客取貨期限 { #prerequisites-overdue-cvs-orders-expired-days }

各物流的逾期判定天數由 CYBERBIZ 依照超商端規則設定，商家不可調整。

| 物流類型 | 逾期判定天數 |
| :-- | :-- |
| 7-11 交貨便（C2C） | 7 天 |
| 全家店到店（C2C） | 6 天 |
| 黑貓快速到店（常溫 / 冷藏 / 冷凍） | 4 天 |
| 其他超商物流 | 依各超商系統判定 |

---

### 自動退款的開通條件 { #prerequisites-overdue-cvs-orders-auto-refund }

並非所有逾期訂單都會自動退款，需同時滿足下列條件：

- [x] **付款狀態為「已收到款項」**：等待付款、貨到付款等狀態不適用自動退款。
- [x] **付款方式為支援自動退刷的金流**：詳細金流清單與退刷期限請見 [自動退刷時效對照表](references/自動退刷時效對照表.md){ data-preview }。
- [x] **未安裝「超商逾期未取不自動退款」加值功能**：若商家有開通此加值功能，所有逾期訂單一律走人工退款。
- [x] **整筆訂單完整出貨**：若為部分出貨（部分超商、部分宅配），超商品項逾期不會即時觸發退款，需待商家確認後處理。

詳細條件對照請見 [退貨與退款狀態對照](references/退貨與退款狀態對照（超商訂單逾期未取）.md#reference-overdue-cvs-refund-flow-auto-refund)。

!!! plan "方案 / 加值功能"
    自動退款的支援方案、各付款方式的退款支援、是否安裝「超商逾期未取不自動退款」，會影響系統行為。若不確定您的店家設定，請聯繫您的 CYBERBIZ 業務窗口。

---

## 操作步驟 { #operate-overdue-cvs-orders }

### 大宗寄倉（B2C）：商品退回物流總倉 { #operate-overdue-cvs-orders-b2c-return }

7-11 一般取貨 / 取貨付款、全家一般取貨 / 取貨付款、全家冷鏈、萊爾富一般取貨 / 取貨付款的逾期商品，會由超商統一退回對應的物流總倉。

**退回位置**：

* 7-11 商品退回統一數網（大智通）
* 全家商品退回全家日翊文化
* 萊爾富商品退回 HiLife 物流中心

**商家領取方式**（依您與 CYBERBIZ **開通超商物流時** 設定的「退貨週期」與「退貨方式」）：

1. **退貨週期：** 系統依您事先約定的週期將商品集中退回，可選擇「每週退」[^fn-every-week]、「隔天收退」[^fn-next-day] 或「當日退」[^fn-same-day]。
2. **退貨方式：** 由指定物流（如黑貓宅急便）至總倉取件並送達您指定的退貨地址；或由您自行前往物流中心提貨。
3. **未指定取件物流時：** 若您未指定取件物流，總倉會以 **宅配到付** 的方式將商品寄回您的退貨地址，相關運費由商家負擔。

!!! note "註釋"
    退貨週期、退貨方式與退貨地址在您與 CYBERBIZ 業務開通超商物流時設定，後台無日常編輯入口。如需修改，請聯繫 CYBERBIZ 業務窗口。

---

### 店到店（C2C）：商品退回原寄件門市 { #operate-overdue-cvs-orders-c2c-return }

7-11 交貨便、全家店到店、萊爾富店到店的逾期商品，會先退回 **原寄件門市**，等待商家領回。

**領回流程**：

1. **接收通知：** 系統會在商品退回原寄件門市時，發送 **簡訊** 與 **Email** 通知商家。
2. **準備證件：** 攜帶與「寄件人姓名」相符的 **身分證件正本** 前往原寄件門市領貨。
3. **限期領取：** 商品退回門市後請於約定期限內領取，逾期未領的處理規則依各超商規範。
4. **逾期未領的後果：** 若超過門市保留期限，商品會被退回超商物流總倉，再由超商以 **黑貓貨到付款** 寄回商家所留的退貨地址，相關運費由商家負擔。

!!! warning "注意"
    若商家 **拒收** 由超商物流總倉退回的包裹，導致商品再次退回物流中心，部分超商規範下貨品會執行 **拋棄處理**，無法再次取回。

!!! note "註釋"
    商品從寄件門市退回原寄件門市的階段 **不收取運費**。後續若由總倉以宅配到付寄回商家，才會產生運費。

---

### 系統自動退款（無需操作） { #operate-overdue-cvs-orders-auto-refund }

當訂單符合 [自動退款開通條件](#prerequisites-overdue-cvs-orders-auto-refund){ data-preview } 時，系統在收到超商逾期通知後會自動：

1. 將「配送狀態」改為「逾期未取」
2. 將「退貨狀態」改為「轉運中心」
3. 將「付款狀態」改為「待退款」並透過原金流通道執行退款，完成後改為「已退款」
4. 若已開立發票，依設定執行發票作廢或折讓

商家在訂單列表頁可直接看到配送狀態變為「逾期未取」、付款狀態變為「已退款」，**無需後台操作**。

---

### 人工退款：填寫退款資訊 { #operate-overdue-cvs-orders-manual-refund }

若訂單不符合自動退款條件（例如付款方式不在自動退刷清單中、或商家有安裝「超商逾期未取不自動退款」加值功能），系統會將付款狀態設為 **「待退款」**，並由商家手動執行退款。

1. **進入訂單詳情頁：** 至後台「訂單」>「所有訂單」，點選配送狀態為「逾期未取」、付款狀態為「待退款」的訂單編號。
2. **填寫退款資訊：** 在訂單詳情頁找到 **「請填寫退款資訊」** 區塊（標題顯示 **「人工退款」**，並提示「已超過自動退款期限，將執行人工退款」）。
3. **輸入顧客銀行資料：** 依序選擇 / 填寫：
    * **銀行：** 從下拉選單選擇顧客的銀行
    * **分行：** 從下拉選單選擇對應分行
    * **戶名：** 顧客帳戶戶名
    * **帳號：** 顧客銀行帳號
    * **退款原因：** 選填，最多 250 字
4. **送出：** 點擊確認後，系統將訂單退貨狀態標記為「已退貨」，並依設定處理發票作廢或折讓。實際撥款由商家依退款資訊以匯款方式執行。

!!! tip "技巧"
    若顧客的銀行資訊由顧客自行提供，您也可以請顧客登入會員帳戶，點擊訂單填寫「退款資訊」（顧客端入口）。您再依顧客提供的資料於後台執行人工退款。

## 重要規範與限制 { #specs-overdue-cvs-orders }

### 逾期訂單不能再次出貨 { #specs-overdue-cvs-orders-no-reship }

逾期未取訂單 **不能** 重新出貨。若顧客仍想購買，請引導顧客重新下單。原訂單依本文流程完成退貨與退款後即可結案。

---

### 部分出貨的特殊處理 { #specs-overdue-cvs-orders-partial }

當訂單為「部分出貨」，且其中 **第一筆為超商物流**、 **第二筆為其他物流** 時：

* **第一筆（超商）：** 配送狀態會變為「逾期未取」，但 **不會自動觸發退款**。商家必須評估後手動執行人工退款（即使原本符合自動退款條件）。
* **第二筆（其他物流）：** 配送狀態維持「已出貨」，不會被連帶更新為「逾期未取」。

詳細狀態流轉請見 [退貨與退款狀態對照](references/退貨與退款狀態對照（超商訂單逾期未取）.md#reference-overdue-cvs-refund-flow-partial){ data-preview }。

---

### 「超商逾期未取不自動退款」加值功能 { #specs-overdue-cvs-orders-no-auto-refund-plugin }

當您安裝此加值功能後， **所有**
超商逾期未取訂單一律走人工退款流程，系統不會自動觸發退款也不會自動作廢發票。退貨狀態也會從預設的「轉運中心」改為「原寄件門市」，待商家確認商品狀況後再決定退款處理。

!!! example "適用情境"
    希望先確認商品退回再退款給顧客，例如有商品損壞風險、或希望從退款金額扣除手續費的場景。

---

### 貨物遺失賠償 { #specs-overdue-cvs-orders-lost }

若包裹未於指定時間退回物流中心或原寄件門市，且超商確定遺失，系統會啟動判賠程序，由超商端進行賠償。請保留出貨單據與系統紀錄以利舉證。

---

### 發票處理 { #specs-overdue-cvs-orders-invoice }

* **CYBERBIZ 代開發票：** 退款完成後系統自動執行折讓。
* **非代開發票（自家系統開立）：** 退款完成後執行作廢或折讓。
* **安裝「超商逾期未取不自動取消發票」加值功能：** 即使逾期，系統不會自動處理發票，需商家手動處理。

## 常見問題 { #faq-overdue-cvs-orders }

??? quote "顧客還想買，可以把逾期訂單重新出貨嗎？"
    [](){ #faq-overdue-cvs-orders-reship }
    不行。逾期未取訂單一旦進入「逾期未取」狀態，系統不再允許出貨。請引導顧客重新下單，原訂單依退貨與退款流程結案即可。

??? quote "為什麼我的訂單逾期了卻沒有自動退款？"
    [](){ #faq-overdue-cvs-orders-no-auto-refund }
    可能原因：

    - 付款方式不在自動退刷支援清單中（詳見 [自動退刷時效對照表](references/自動退刷時效對照表.md){ data-preview }）
    - 商家有安裝「超商逾期未取不自動退款」加值功能
    - 訂單為部分出貨，僅第一筆超商品項逾期
    - 訂單付款狀態尚未到達「已收到款項」

    詳細條件請見 [自動退款的開通條件](#prerequisites-overdue-cvs-orders-auto-refund){ data-preview }。

??? quote "商品退回原寄件門市後，多久內必須去領？"
    [](){ #faq-overdue-cvs-orders-pickup-deadline }
    商品退回原寄件門市後請盡快領取，逾期未領的處理規則依各超商規範。逾期未領後，包裹會被退回超商物流總倉，再由超商以 **黑貓貨到付款** 寄回商家所留的退貨地址，相關運費由商家負擔。如不確定領取期限，建議收到系統 Email / 簡訊通知後 **盡早前往**。

??? quote "顧客的退款資訊（銀行 / 帳號）要從哪裡取得？"
    [](){ #faq-overdue-cvs-orders-bank-info }
    兩種方式：

    - **商家主動聯繫：** 請顧客提供銀行 / 分行 / 戶名 / 帳號，商家在訂單詳情頁的「請填寫退款資訊」區塊輸入
    - **請顧客自行填寫：** 引導顧客登入會員帳戶，至「退款資訊」頁面自行填寫

??? quote "黑貓快速到店逾期會被收幾天的運費？"
    [](){ #faq-overdue-cvs-orders-tcat-fee }
    黑貓快速到店若顧客於 **4 日內未取**（系統判定為逾期），會產生一筆 **退貨運費**（依材積與溫層計費）。實際金額請見 [黑貓快速到店逾期說明](references/黑貓快速到店逾期說明.md){ data-preview } 與您的合約規範。

??? quote "拋棄處理是什麼？什麼情況會被拋棄？"
    [](){ #faq-overdue-cvs-orders-discard }
    當商品從原寄件門市退回物流總倉後，由總倉以宅配到付寄回商家時，若商家 **拒收**，部分超商規範下貨品會被執行「拋棄處理」，無法再次取回。請務必收到退貨通知後依約定方式領取或受收。

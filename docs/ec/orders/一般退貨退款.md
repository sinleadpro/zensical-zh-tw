# 一般退貨退款

訂單商品退貨、退款的操作流程，包含商家與消費者雙方的步驟與注意事項。
{ .subtitle }

![](../../assets/images/ec-orders-returning.zh-tw.png){ .hero-page }

## 使用須知

- **申請次數**：每筆訂單僅可申請一次退貨或退款。
- **人工退款**：若需由 CYBERBIZ 進行人工退款，將酌收每筆 30 元帳務處理費，並於 7-10 個工作天內完成。
- **倉儲用戶**: 使用 CYBERBIZ 電商倉儲的商家，若需自行驗收退貨，請參閱[商家自行驗收退貨](https://www.cyberbiz.io/support/?p=4414)文件。

### 紅利/優惠券處理方式

- 該訂單中使用的紅利/優惠券，系統 *不會* 自動歸還至顧客帳戶。
- 該訂單消費所獲得的紅利/優惠券回饋，系統 *不會* 從顧客帳戶扣除。
- 如需刪除個人紅利/優惠券，請至 **會員 > 所有會員 > 至顧客個人頁面** 進行刪除。

### 分潤計算

- 已 *結案* 的訂單會進行分潤計算。
- 即使發生退貨或退款，已 *結案* 訂單的分潤仍不受影響。

### 退款發票處理

-   **CYBERBIZ 代開**: 發票將進行折讓。
-   **非 CYBERBIZ 代開 (如星益欣、綠界)**: 依開立時間作廢或折讓（通常為當月 5 號前作廢，之後折讓）。

### 訂單退款方式說明

依訂單的支付方式判斷訂單需進行自動退款或人工退款。不同方案支援的退款方式有所差異。請根據您的方案點擊下方頁籤，查看對應的退款形式。

## 操作流程

退貨退款流程可分為兩個主要階段：

- [顧客申請退貨退款](#顧客申請退貨流程)
- [商家處理退貨退款訂單](#處理退貨退款訂單)

### 顧客申請退貨退款

顧客必須在設定的 *退貨期限*  內透過訂單查詢頁面提出退貨申請，逾期訂單需由商家在[後台手動處理](#處理退貨退款訂單)。系統預設退貨期限如下：

- **CYBERBIZ 串接出貨方式**（宅配、超商取貨）：可於 _10 天內_ 申請退貨。
    
- **自訂物流出貨方式**（其他方式）：可於 _14 天內_ 申請退貨。

!!! info "進一步了解 [如何調整退貨期限](#調整-退貨期限)。"

1. 前往 **訂單查詢 > 我的帳戶 > 近期訂單**。
2. 選擇想要退貨的訂單，點選 **可退貨** 。
> 列表中訂單會呈現 **可退貨** 的狀態，點選欲退貨的訂單編號，進入訂單明細。
3. 點擊 **申請退貨**。
4. 填寫退貨原因及退款資訊並點選 **確認退貨**。
	- 若訂單的支付方式可自動退刷將自動退款。(如：信用卡、LINE Pay)
	- 若訂單的支付方式若不支援自動退刷，須輸入顧客退款資料，由 CYBERBIZ 進行人工退款。  
	    (如：貨到付款、虛擬ATM)
5. 顧客於前台申請完成後，商家需進行[退款操作](#商家處理退貨退款流程)。
> 顧客申請完成後，訂單退貨狀態將變更至 *退貨申請*。

#### 調整 *退貨期限*

1. 在 CYBERBIZ 管理後台，前往 **金物流 > 結帳頁 & 物流設定 > 前台退貨設定**。
2. 在 **退貨時間調整設定** 區塊下方的相關欄位輸入數值。

![](../../assets/images/ec-return-policy-eligible-days.zh-tw.png)

## 常見問題

??? quote "退貨運費應由誰負擔？"
	依據退貨原因及商家政策而定。建議商家於退貨政策中明確說明。
??? quote "退款通常需要多久時間入帳？" 
	視退款方式（例如：信用卡刷退、銀行匯款）及銀行處理時程而異。一般而言，信用卡刷退可能需數個工作天，銀行匯款則依銀行間作業速度而定。
??? quote "已使用的紅利點數或優惠券，退貨後會返還嗎？" 
	需依商家設定的紅利/優惠券規則處理。部分情況下可能不予返還或以其他形式補償。
??? quote "退貨後，訂單會自動取消嗎？"
	退貨流程完成後，訂單狀態會更新為「已退貨」或「已退款」，但訂單記錄通常會保留。

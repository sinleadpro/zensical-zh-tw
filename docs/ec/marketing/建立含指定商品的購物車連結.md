## 指定商品購物車連結說明

這項功能允許商家生成一個特殊的網址，當顧客點擊該連結時，系統會 **自動將預先設定好的商品與數量加入購物車**。這對於社群導購、EDM 行銷或針對特定受眾的促銷活動非常有效。

## 如何取得商品款式 ID

要製作連結，首先需要取得後台系統記錄的「款式 ID」。

1.  **進入後台路徑**：前往「**商品**」→「**所有商品**」。
2.  **勾選商品**：在商品列表中勾選您想要導入購物車的商品品項。
3.  **匯出資料**：點擊右上角的「**匯出商品**」按鈕。
4.  **查找 ID**：下載並開啟 Excel 檔案，展開隱藏的 **B 欄： 商品款式 id**。
5.  **紀錄數值**：該欄位顯示的數字即為「款式 ID」，請將其記錄下來以利後續填入網址。

## 購物車連結的網址格式

請依照以下格式手動組合您的導購連結：

*   **基本網址格式**：
    `[您的網址]/cart/replace?products=[款式ID]:[數量],[款式ID]:[數量]`
*   **參數說明**：
    *   **您的網址**：請輸入您官網的完整網址。
    *   **products=**：接續商品資訊參數。
    *   **[款式ID]**：填入剛剛從 Excel 取得的款式 ID。
    *   **[數量]**：設定該商品預計加入購物車的數量。
    *   **多樣商品設定**：若要一次加入多項商品，請使用**逗號 `,`** 或**分號 `;`** 將不同商品的 ID 與數量組隔開。

## 實際範例

若您的官網網址為 `https://www.goodthing.com`，想要讓顧客點擊後自動加入款式 ID 為 `001234` 與 `001235` 的商品各 1 件，連結如下：

`https://www.goodthing.com/cart/replace?products=001234:1,001235:1`

打開此連結後，購物車將自動載入這兩項商品。

## 應用情境範例

您可以將製作好的連結應用在以下管道：

*   **社群導購連結**：放在 LINE 或 Instagram 貼文中，縮短顧客尋找商品的流程，提升轉換率。
*   **EDM 快速下單**：在電子報中針對熱門組合加入連結，讓顧客一鍵完成加購。
*   **分眾導流活動**：針對不同的會員受眾設定專屬的購物車內容，搭配行銷策略推廣。

**💡 溫馨提醒：**
在正式對外發布前，**請務必先自行測試網址**，確認點擊後是否能正確引導至購物車並載入指定的商品與數量。

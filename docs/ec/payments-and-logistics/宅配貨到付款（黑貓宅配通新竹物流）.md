# 宅配貨到付款物流（黑貓/宅配通/新竹物流）

透過 CYBERBIZ 串接黑貓宅急便、宅配通與新竹物流的貨到付款服務，您可以直接在後台產生託運單，簡化對帳與出貨流程。
{ .subtitle }

[:lucide-toggle-right:{ title="適用功能" }](../../resources/conventions#適用功能) | CYBERBIZ PAYMENTS
{ .doc-badge }

!!! info "串接物流 vs. 自訂物流"
    - **串接物流**：指與 CYBERBIZ 系統深度整合的物流服務，系統可自動產生對應物流商的託運單，並自動更新配送狀態。
    - **自訂物流**：若您使用非上述三家（或非透過串接模式）的宅配服務，請參考[自訂物流設定](./啟用自訂物流.md)。

---

## 步驟 1：開通申請

### 1. 黑貓宅急便與宅配通

這兩家服務通常於系統開通後即可在後台進行設定，不需額外的人工審核流程。

### 2. 新竹物流貨到付款

新竹物流需要進行實地廠勘與合約簽署，流程如下：

1. 前往 **金物流 > 宅配物流 > 串接物流 > 新竹物流貨到付款**。
2. 勾選 **同意合約規範**。
3. 填寫申請表格並提交。
4. CYBERBIZ 客服將通知新竹物流進行廠勘。**廠勘完成後**，約 **7-10 個工作天** 內會完成系統開通。
5. 開通成功後，您將收到系統發出的通知信。

!!! note "資料變更"
    若新竹物流開通後需要更改負責人或聯繫資訊，請務必聯繫 CYBERBIZ 客服人員協助處理，不可自行修改已驗證之欄位。



## 步驟 2：啟用物流

前往 **金物流 > 宅配物流 > 串接物流**，選擇對應的物流商進行編輯。

### 1. 基本啟用設定

- **啟用服務**：將開關切換至 **ON**。
- **離島配送**：選擇是否開放配送至台灣離島地區（黑貓/宅配通/新竹物流皆支援）。

### 2. 運費與免運設定

- **免運設定**：開啟後可設定單筆訂單的免運門檻（金額）。
- **結帳頁提示免運門檻**：開啟後，消費者在購物車與結帳頁可看到「再買 $XXX 即可享免運」的提醒。
- **運費設定**：依據您的營運規範，分別設定 **常溫、冷藏、冷凍** 的運費金額。
    - **選項顯示門檻**：訂單金額須達設定之起始門檻；若未達標，系統將自動於結帳頁面隱藏該物流選項。
    - **不限金額設定**：如需開放不限金額皆可使用，請將任一項條件區間之 **起始金額** 配置為 0 元。

### 3. 金流門檻設定 (PLUS 以上方案)

- 您可以設定該物流方式適用的 **最高** 或 **最低** 結帳金額門檻，避免代收金額過高或不符效益。
  >   :lucide-lock: 此功能僅適用於 PLUS 與企業版方案。



## 常見問題

??? quote "免運門檻判斷邏輯為何？"
    系統會綜合判斷 **免運設定**（物流層級）與 **運費設定**（溫層層級）。只要訂單金額達到其中之一的免運條件，結帳頁面即會顯示免運，並以對消費者最優（從優提示）的方式呈現。

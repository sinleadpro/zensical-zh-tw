# 建立門市
在提供實體門市相關服務前，您需要先在管理後台建立門市資訊。根據您的營運需求與是否使用 POS 系統，可選擇建立「一般門市」或「POS 門市」。
{ .subtitle }

[:lucide-layers:{ title="適用產品" }](../../resources/conventions#適用產品) | EC / POS
[:lucide-tag:{ title="適用方案" }](../../resources/conventions#適用方案) | 進階 PLUS / 高手 PLUS / 企業
{ .doc-badge }

!!! tip "應用情境"
    - **提供官網門市取貨**：建立門市清單，讓顧客在結帳時可選擇鄰近的門市自取商品。
    - **整合線下實體店點**：將實體店資訊同步至官網，作為 **門市取貨** 與 **電子票券** 的核銷據點。


## 門市類型比較

在建立門市前，請根據下表選擇正確的門市類別：

| 門市類別 | 適用對象 | 門市取貨 | CYBERBIZ NOW 快速到貨 | 電子票券核銷 |
| :--- | :--- | :--- | :--- | :--- |
| **一般門市** | 純官網商家<br>**使用快速到貨之 POS 商家** | ✓ | ✓ | ✓ |
| **POS 門市** | 官網搭配 POS 系統之商家 | ✓ | ✕ | ✓ |


## 操作流程

### 單筆建立門市

1. 登入 CYBERBIZ 管理後台，前往 **金物流 > 所有門市**。
2. 點擊右上角 **新增門市**。
3. 根據您的需求選擇 **門市類別**：
    - 若為純官網或需使用快速到貨，請選擇 **一般門市**。
    - 若需整合 POS 進行門市取貨與核銷，請選擇 **POS 門市**，並從下拉選單選擇對應的 POS 店名。
4. 填入 **門市基本資訊**（名稱、電話、地址、營業時間等）。
5. 點擊 **儲存**，完成門市建立。

!!! tip "設定預設門市"
    若您有多間門市且運費/付款規則相同，可將其中一間設為 **預設門市**。後續建立新門市時，點選 **帶入預設門市資料** 即可快速套用設定。

### 批次建立門市

若您需同時建立多間門市並開啟 **門市取貨** 功能，可使用 **批次匯入** 一併建立。

1. 前往 **金物流 > 所有門市**，點擊 **下載 Excel 範例**。
2. 開啟檔案，根據您的門市類型填寫對應的分頁（頁籤）：
    - **一般門市**：請填寫 **門市取貨店家** 頁籤。
    - **POS 門市**：請填寫 **POS 商店** 頁籤，並確保門市名稱與 POS 商店名稱一致。
3. 填寫門市基本資訊。
4. 填入付款方式。

    - **支援選項**：門市取貨訂單僅支援 **非貨到付款** 選項，**恕不支援貨到付款**。填寫貨到付款後，將造成 POS 門市到店條碼單無法印出。
    - **填寫方式**：以英文半形逗號 `,` 隔開

5. 完成運費設定。

    - **填寫範圍**：請於 **[4]物流名稱**、**[5]消費金額**、**[6]運費** 欄位填寫對應資訊。
    - **填寫格式**：限用中文、半形數字、波浪號 `~`。
    - **級距設定**：單一門市若需建立多層運費區間，請於該門市資料列下方 **新增空白列** 進行擴充填寫。

      > 新增級距時，**[4]物流名稱** 欄位須保持相同。
    

6. 回到後台點擊 **Excel 匯入門市** 並選擇檔案。
7. 匯入成功後，門市預設為關閉狀態，請逐一進入編輯頁面確認資訊並開啟。

!!! warning "注意事項"
    - 批次匯入時，若只需新增其中一種類型，請清空另一個頁籤的內容，避免重複建立或匯入錯誤。


### 建立預設門市

透過設定 **預設門市**，您可以建立一套標準的付款與運費規範範本。當您有多間門市需要設定相同的邏輯時，只需設定一次預設門市，後續新增門市時即可一鍵導入，無需重複手動填寫。

!!! info "功能定義"
    **預設門市** 是指後台設定的 **邏輯範本**，並非在官網結帳頁面為顧客 **預先勾選** 某間取貨門市。

1. 前往 **金物流 > 所有門市**，選擇一間已設定好規則的門市進入編輯頁面。
2. 在 **基本設定** 區塊中，勾選 **設為預設門市** 並點擊 **儲存**。
3. 後續建立其他門市時，於編輯頁面勾選 **帶入預設門市資料**。
4. 系統將自動填入預設門市的 **付款方式設定** 與 **訂單金額運費設定** 資訊。

## 常見問題


??? quote "啟用 POS 系統後，可否選擇建立 **一般門市** 來處理訂單？"
    可以，但需注意 **一般門市** 的訂單不具備 POS 前端操作權限。這類訂單不會出現在 POS 前台介面中，管理員必須登入管理後台進行狀態變更（如：變更為已到店、已收貨）。

# 宅配貨到不付款物流（黑貓/宅配通/新竹物流）
消費者在下單時已透過信用卡、ATM 等方式完成付款，商家僅需負責配送。在 CYBERBIZ 中，此類服務透過「自訂物流」進行設定。
{ .subtitle }


## 使用須知

即使是自訂物流，您仍可於出貨時仍可直接從系統下載對應的託運單，無需手動抄寫單據。



## 步驟 1 ：建立自訂物流

### 1. 新增物流項目

1. 前往 **金物流 > 宅配物流 > 自訂物流**。
2. 點擊 **新增自訂物流**。
3. **物流名稱**：此處名稱會顯示於結帳頁，請選擇具辨識能力的名稱。

     > 範例：「一般宅配（黑貓）」或「黑貓宅急便」

4. **運送地區**：地區設定層級至 **縣市** 級別，恕無法針對特定 **鄉鎮區** 進行獨立配置。
5. **運送溫層**：設定此物流適用的溫層（常溫/冷藏/冷凍）。
    - 若您的商品包含多溫層，請為每個溫層各建立一個物流選項。

### 2. 綁定付款方式

**付款方式**：勾選欲支援的金流選項。


### 3. 設定運費與免運門檻

**運費設定**：可依 **訂單金額** 與 **訂單重量** 設置運費。
  - **計費優先順序**：若訂單同時符合兩類型的運費條件，系統將自動套用 **最優惠（即金額最低）** 之條件進行計費。
  - **選項顯示門檻**：訂單金額須達設定之起始門檻；若未達標，系統將自動於結帳頁面隱藏該物流選項。
  - **不限金額設定**：如需開放不限金額皆可使用，請將任一項條件區間之 **起始金額** 配置為 0 元。

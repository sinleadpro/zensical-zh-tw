# 順豐託運單(設定、加印、單號使用紀錄)

## 功能介紹 { #intro-sf-waybill }

後台「金物流」→「順豐託運單」(路徑:`/admin/shipping_notes/sf`)是 **順豐相關功能的單一管理頁面**,集中三大區塊:

1. **順豐設定** —— 第一次使用前必須建立寄件人資訊;若有開通海外宅配,還可指定報關方式。
2. **加印託運單** —— 針對 **單一訂單** 額外產生託運單號(適用一單多包、原本單號遺失補印等情境)。
3. **單號使用紀錄** —— 列出所有曾經產生過的順豐單號,可查詢扣費、狀態、可否轉單。

頁面頂端會顯示 **目前 Cyber 幣** 餘額(一般 / 高手 / 進階 / 專業 / 基本版)或 **無需儲值** 的提示(PLUS / 企業版)。

---

## 使用前提與限制 { #prerequisites-sf-waybill }

* 須先開通 **順豐託運單**(一般版以上方案皆可)。
* 寄送海外件另需開通 **順豐海外宅配**(僅企業版 / 全球商務 / 全球專業 / 全球試用方案可用),畫面才會出現「託運單預設品名(海外)」與「報關方式」欄位。
* 加印託運單採 **預扣 Cyber 幣** 機制,**一般版商家若 Cyber 幣餘額不足將無法列印**,需先至「儲值中心」儲值。
* 加印託運單 **只會印出「純配送」託運單**;貨到付款的託運單請至訂單列表操作,本頁無法產生 sf_cod 託運單。

---

## 操作步驟 { #operate-sf-waybill }

### 順豐設定 { #operate-sf-waybill-setting }

第一次使用前先完成這個區塊,寄件人資訊會被印在每一張順豐託運單上。

1. 進入後台 **「金物流」→「順豐託運單」**。
2. 找到頁面上方的 **「順豐設定」** 區塊。**尚未建立** 設定的商家,標題後方會加註「(您尚未建立順豐設定,可於此建立)」提示。
3. 依序填入:
    * **託運單預設品名** —— 最多 **8 個中文字**,例如「服飾」、「食品」。**空白或超過 8 字** 系統會擋下並提示「品名請勿空白」或「品名請勿超過 8 個中文字」。
    * **寄件人名稱** —— 30 字內。
    * **寄件人電話** —— 30 字內。
    * **寄件人地址** —— 30 字內,單行文字。
    * **報關方式**(僅開通海外宅配時顯示)—— 三選一,預設為 **簡易報關(樣品04)**。詳見 [海外報關方式說明][specs-sf-waybill-customs]{ data-preview }。
4. 點擊 **「確認」** 按鈕,系統顯示「已成功變更設定」即完成。

!!! info "提示"
    寄件人名稱 / 電話 / 地址三欄 **任一空白都會被擋下**,提示「寄件人名稱和寄件人電話,寄件人地址請勿空白」。請確保三欄都填寫完整。

### 加印託運單 { #operate-sf-waybill-reprint }

當您需要為 **單一訂單** 額外產出託運單時(例如一筆訂單拆成多包寄送、託運單檔案不慎遺失需要補產一張),使用此功能。

1. 至 **「金物流」→「順豐託運單」**,捲動至 **「加印託運單」** 區塊。
2. 依序填入:
    * **訂單編號** —— 直接輸入訂單編號(不含商店設定的訂單編號前綴)。若空白會提示「請輸入訂單編號」。
    * **選擇貨品大小** —— 從下拉中挑選對應的順豐尺寸,下拉會顯示「(規格) Cyber 幣 XX 點」格式,海外宅配開通者會多出「海外 80cm」選項。
    * **加印張數** —— 預設 **1 張**,最多 **8 張**。
3. 勾選 **「我同意 [CYBERBIZ 物流串接服務條款] 與 [順豐合約規範]」**(預設已勾選),否則 **「確認下載」** 按鈕無法點擊。
4. 點擊 **「確認下載」**,系統會彈出二次確認視窗 **「確認要消費 Cyber 幣列印順豐託運單?」**,按 **「確認」** 即會:
    * 立即扣除「Cyber 幣單價 × 加印張數」(一般版),或列入該期對帳單(PLUS / 企業版)。
    * 呼叫順豐 OpenAPI 取得新單號。
    * 自動下載託運單 PDF / 圖檔。
    * 將該訂單與單號寫入下方的「單號使用紀錄」表格。

!!! note "註釋"
    加印託運單 **只會印出「純配送」託運單**,即使原訂單是貨到付款,本功能也只能產生不含代收款的單號。需要列印貨到付款託運單,請改回「訂單」→「所有訂單」勾選訂單後執行 [下載順豐託運單][use-sf-shipping]{ data-preview }。

### 單號使用紀錄 { #operate-sf-waybill-history }

頁面最下方的「單號使用記錄」表格,列出所有曾為該店家產生過的順豐託運單。資料以 **建立時間倒序** 排列(最新在最上)。

各欄位定義:

| 欄位 | 內容 |
| :-- | :-- |
| **單號類型** | 「出貨」(由訂單產生)、「退單」(逆物流)、「訂單刪除」(原訂單已刪除,僅保留單號紀錄) |
| **時間** | 託運單建立的精確時間(年/月/日 時:分:秒) |
| **訂單編號** | 對應的 CYBERBIZ 訂單編號,**點擊可跳轉至訂單詳情頁**(刪除訂單者顯示「訂單刪除」字樣) |
| **順豐託運單號** | 順豐回傳的單號 |
| **配送方案** | 託運單對應的配送尺寸,例如「常溫 110cm」 |
| **客戶選定 Cyber 幣額度** | 建立時設定的 Cyber 幣扣抵金額(訂價)|
| **確認扣除 Cyber 幣額度** | 實際扣除的 Cyber 幣金額;**已退單或失效退款者顯示 0**(此欄會出現「不扣 Cyber 幣」字樣)|
| **對帳備註** | 系統內部備註,如失效退款註記 |
| **單號狀態** | 順豐回傳的狀態,例如「已列印」、「已寄出」、「取消寄件」、「客戶已取消寄件」 |
| **轉單狀態** | 顯示「可轉單」或「不可轉單」,代表此單號於失效時是否能轉用至其他訂單 |

!!! tip "技巧"
    要尋找特定訂單的所有順豐託運單,可直接用瀏覽器 ++ctrl+f++(Mac 為 ++cmd+f++)在頁面上搜尋訂單編號。表格本身支援排序與分頁。

## 重要規範與限制 { #specs-sf-waybill }

### 海外報關方式說明 { #specs-sf-waybill-customs }

僅在開通 **順豐海外宅配** 後,「順豐設定」會出現報關方式下拉,共三種選項:

| 報關方式 | 適用情境 | 額外費用與文件 |
| :-- | :-- | :-- |
| **簡易報關(樣品04)** | 預設選項,適用樣品 / 小額 / 非銷售用途 | 無補收 |
| **簡易報關(銷售02)** | 銷售用途且申報金額 **未超過 5 萬元** | **每筆補收 100 元**(對帳時),需備妥 Invoice、個案委任書、出口簡易報單申請書 |
| **正式報關** | 申報金額 **超過 5 萬元** 強制適用 | **每筆補收 500 元**(對帳時),需備妥 Invoice、個案委任書、出口正式報單申請書 |

* 上述報關文件可至 **順豐官網下載中心** 取得。
* **寄往美國** 的貨件,目前 **稅金一律由寄方支付**,無法選擇收方支付。

### 加印託運單的扣費規則 { #specs-sf-waybill-charge }

* **下載即扣款**:點擊「確認下載」後立即扣除 Cyber 幣或列入對帳單,無法事後撤銷。
* **21 天失效規則**:託運單下載後,若 **超過 21 天(約 3 週)未實際寄件**,系統會依順豐回傳貨態判定為失效,單號狀態改為「取消寄件」並自動退回 Cyber 幣(對帳單方案則不會列入該期帳單)。
* **超過 170cm 拆箱建議**:單一包裹建議不超過 170cm。順豐司機可能不收取超大件包裹,屆時需要重新加印託運單。

### 春節繁盛期加收 { #specs-sf-waybill-busy-fee }

春節假期 **每張託運單加收 60 元**,以順豐官方公告為準。後續若有其他加收項目,以順豐公告為準。

## 後續操作 { #next-steps-sf-waybill }

完成順豐設定後,即可前往 **「訂單」→「所有訂單」** 批次或單筆下載順豐託運單。完整出貨流程請參考 [使用順豐宅配出貨][use-sf-shipping]{ data-preview }。

各尺寸對應運費請參考 [順豐配送商品尺寸與運費對照表][sf-shipment-sizes]{ data-preview }。

## 常見問題 { #faq-sf-waybill }

??? quote "更改順豐設定的寄件人資訊後,已產生的順豐單號會跟著更新嗎?"
    #### 更改順豐設定的寄件人資訊後,已產生的順豐單號會跟著更新嗎? [](){ #faq-sf-waybill-sender-change } { .hidden-header }

    **不會**。寄件人資訊在託運單建立時即固定,後續修改僅會影響 **新產生的單號**。已下載且已寄出的單號維持原本的寄件人資訊。

??? quote "我想加印託運單,但下拉裡找不到海外尺寸選項?"
    #### 我想加印託運單,但下拉裡找不到海外尺寸選項? [](){ #faq-sf-waybill-oversea-missing } { .hidden-header }

    請確認:

    * 店家已開通 **順豐海外宅配** 加值功能(僅企業版及全球系列方案可用)。
    * 訂單的 **收件地址** 為支援的海外國家。若收件地址在台灣,系統不會顯示海外尺寸。

??? quote "Cyber 幣不足時可以先列印,之後再補嗎?"
    #### Cyber 幣不足時可以先列印,之後再補嗎? [](){ #faq-sf-waybill-points-insufficient } { .hidden-header }

    **不行**。加印託運單採 **預扣機制**,點擊「確認下載」時即立即驗證餘額,不足將直接擋下並提示「Cyber 幣不足,請至儲值中心進行儲值」。請先至 **「管理中心」→「儲值中心」** 完成儲值,再回頭加印。PLUS / 企業版商家不受此限制(列入對帳單)。

??? quote "為什麼貨到付款訂單在「加印託運單」只會印出純配送單?"
    #### 為什麼貨到付款訂單在「加印託運單」只會印出純配送單? [](){ #faq-sf-waybill-cod-pure-shipping } { .hidden-header }

    本功能設計上 **只負責產生純配送單**,因為貨到付款託運單會綁定代收款金額,若隨意加印會造成代收款分散在多張單據、結帳對帳錯亂。需要列印貨到付款託運單,請至「訂單」→「所有訂單」勾選訂單後操作。

??? quote "「單號使用記錄」中的「轉單狀態」是什麼意思?"
    #### 「單號使用記錄」中的「轉單狀態」是什麼意思? [](){ #faq-sf-waybill-transfer-status } { .hidden-header }

    「可轉單」代表此單號 **失效後可以轉用至其他訂單**,系統會在後續加印時優先使用未失效的可轉單號;「不可轉單」則代表該單號只能用於原訂單。一般情況商家不需手動操作此設定,系統會自動處理。

??? quote "託運單檔案不小心關掉了,可以重新下載嗎?"
    #### 託運單檔案不小心關掉了,可以重新下載嗎? [](){ #faq-sf-waybill-redownload } { .hidden-header }

    可以,但要分情境:

    * **同一個單號重印**:目前頁面 **沒有單號層級的重印按鈕**,需從訂單詳情頁的出貨紀錄處重新觸發下載。
    * **產生新單號**:在本頁的「加印託運單」輸入該訂單編號,**會額外扣一次 Cyber 幣** 產生新單號,原單號保留在紀錄中。

## 跑馬燈轉場時間說明

您可以透過修改程式碼來調整 **首頁跑馬燈（輪播圖）的轉場停留時間**，這有助於提供客戶更佳的閱讀體驗。

以下為跑馬燈轉場時間設定的詳細教學：

## 設定操作步驟

!!! warning "適用範圍與版型限制" 
	- **適用範圍：** 此設定 **僅適用於「首頁跑馬燈」**。若您是使用自行貼上 liquid 語法的「進階跑馬燈」，則不包含在內。
	- **版型限制：** 此教學僅適用於「預設版型（舊版）」。 

1. **進入路徑：** 登入 CYBERBIZ 管理後台，前往 **網站外觀 > 套版主題管理 > CSS/HTML 編輯器**。
2. **開啟檔案：** 在搜尋欄輸入並點開檔案 **`_main_slider.liquid`**。
3. **搜尋關鍵字：** 在檔案內搜尋「**delay:**」。
4. **修改停留時間：** 更改 `delay:` 後方的數字即可調整更換圖片的時間。該數值以 **毫秒 (ms)** 為單位。
    - 範例：`3000` 代表 3 秒。
    - 範例：`20000` 代表 20 秒。
5. **儲存設定：** 修改完畢後按下儲存。

``` liquid hl_lines="9" title="_main_slider.liquid"
// Enable lazy loading
lazy: {
enabled: true,
loadPrevNext: true,
loadPrevNextAmount: 1
	},
loop: true,
autoplay: {
  delay: 3000,
},
pagination: {
  el: '.swiper-pagination',
  type: 'bullets',
  clickable: true
},
```

## 重要注意事項

- **責任歸屬：** CYBERBIZ 提供程式碼修改權限，但 **不提供語法教學或代碼撰寫服務**，建議由貴公司技術人員執行。
- **恢復機制**：若修改後導致頁面異常，可點擊編輯器中的 **查看之前版本**，可[回溯至先前版本](使用樣板編輯器恢復網頁代碼.md#操作步驟){ data-preview }。


## 常見問題

??? quote "修改後跑馬燈停止自動輪播了，該怎麼辦？" 
	這通常是因為程式碼語法錯誤（例如誤刪了逗號 `,` 或冒號 `:`）。
	
	- **檢查語法：** 請確認 `delay: 3000` 後方的逗號是否還在。 
	- **立即恢復：** 若無法自行修正，請參考下方「重要注意事項」中的 **恢復機制**，將檔案還原至前一個正常的版本。

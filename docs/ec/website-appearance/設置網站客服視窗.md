透過 GetButton、Facebook 即時客服或 Omnichat 在官網設置客服視窗，提升品牌與顧客溝通效率。
{ .subtitle }

![](../../../assets/images/ec-網站外觀-客服視窗.png){ .hero-page }


## 客服視窗說明

設定客服視窗旨在 **建立品牌與客戶間的溝通管道**，有助於增加網站成交機會並減少客服成本。目前主要透過外掛插件（如 GetButton）或 Facebook 即時客服功能來達成。以下為詳細的操作說明與教學：

## GetButton 外掛客服插件設定

!!! warning "GetButton 並非完全免費的外掛，商家需自行評估其相關費用。"

GetButton 是一款能同時整合 Messenger、LINE 等多種通訊軟體的第三方客服外掛。

1. **建立 GetButton 程式碼：**
    
    - 搜尋並進入 GetButton 網頁，點擊 **「Get Your Button」**。
    - **選擇通訊管道：** 選擇 Messenger 或 LINE。輸入 FB 粉絲專頁網址（取得 Page ID）或將從 [LINE OA 管理後台複製的好友連結](../integrations/line/LINE 官方帳號增加好友指南.md#前置準備取得加入好友連結){ data-preview }  貼入對應欄位。
    - **設定 Chatbot 外觀：** 可自訂視窗顏色、位置（置左或置右）以及 **Call to action**（引導文字）。
    - **取得程式碼：** 完成設定後，複製系統產生的整段 **GetButton Code**。
    - 完整設定流程，請參閱 [GetButton 官方教學影片 :lucide-external-link:](https://youtu.be/dBY17EJkgWM)。

	!!! tip "若需更直觀的操作示範，請觀看 [GetButton 官方教學影片 :lucide-external-link:](https://youtu.be/dBY17EJkgWM)。"
	
2. **串接至 CYBERBIZ 後台：**
    
    - **後台路徑：** 前往 **網站外觀 > 套版主題管理**。
    - **開啟編輯器：** 在目前使用的主題上點選「選擇操作」>「CSS/HTML 編輯器」。
    - **埋設語法：** 搜尋並點開 **`theme.liquid`** 檔案，將從 GetButton 複製的程式碼貼入檔案中（通常建議放在 `</body>` 標籤前）並儲存。
    - **完成展示：** 儲存後即可在官網前台右下角看到客服視窗。

	``` liquid title="theme.liquid" hl_lines="2-11"
	...
	        <script type="text/javascript">
	            (function () {
	                var options = {
	                    line: "//line.me/ti/p/@您的ID", 
	                    facebook: "您的專頁ID",
	                    // ... 其他設定屬性 ...
	                };
	                // ... 系統初始化語法 ...
	            })();
	        </script>
	    </body>
	</html>
	```
	
## 臉書即時客服 (FBE 2.0)

針對使用 **進階版以上** 的商家，可以直接透過 Facebook 商業擴充套件開啟即時客服功能。

- **功能開關：** 商家可在後台執行 FBE 2.0 串接流程，並在設定中找到「臉書即時客服」開關進行啟閉。
- **優點：** 系統會自動將 Facebook 帳號與 CYBERBIZ 顧客列表關聯，方便篩選透過 FB 登入的會員。

## 進階第三方客服整合 (Omnichat)

若商家有更高階的多通路整合需求，CYBERBIZ 也支援安裝 **Omnichat 客服應用**。

詳細設定說明，請參閱 [Omnichat 官方教學說明 :lucide-external-link:](https://docs.omnichat.ai/features/tong-xun-qu-dao/wang-zhan-dui-hua-cha-jian/install/cyberbiz-an-zhuang-omnichat#bu-zhou-2)。 

## 重要注意事項

- **二擇一建議：** CYBERBIZ 建議 **Facebook 即時客服或 GetButton 插件擇一使用**，以避免版面混亂或語法衝突。
- **程式碼風險：** 商家自行修改程式碼若導致版面跑版，需自行承擔風險，建議操作前先記錄時間點，發生異常時利用編輯器中的 **「查看之前版本」** 功能[回溯至先前版本](使用樣板編輯器恢復網頁代碼.md#操作步驟){ data-preview }。
- **版型限制：** 拖拉版型僅支援部分程式碼編輯，實際操作需視後台開放的功能而定。


## 常見問題

??? quote "為什麼我貼上 GetButton 程式碼後，官網前台沒有出現客服圖示？" 
	這通常有以下幾種可能原因： 
	
	- **快取延遲：** 請嘗試使用「無痕視窗」開啟官網，或清除瀏覽器快取後再重新整理。 
	- **位置貼錯：** 請檢查程式碼是否確實貼在 `theme.liquid` 檔案的 `</body>` 標籤 **之前**。 
	- **語法衝突：** 若您同時開啟了 FBE 2.0 或其他第三方客服插件，可能會產生衝突，建議先暫停其他外掛進行測試。

??? quote "我可以自訂客服按鈕的顯示時間或特定頁面嗎？" 
	- **GetButton 免費版：** 通常為全站顯示，無法設定特定頁面或時間。 
	- **Omnichat：** 支援進階的顯示規則設定（例如：僅在商品頁顯示、或顧客停留 30 秒後才彈出），建議有此需求的商家優先考慮 [Omnichat 方案](../integrations/omnichat/Omnichat 串接指南.md)。

??? quote "如果我發現客服按鈕遮擋了官網原有的「回到頂部」按鈕怎麼辦？" 
	您可以透過 GetButton 的設定頁面調整 **Position**（位置）： * 嘗試將位置從 `Right` 改為 `Left`。 * 或是在設定中的 `Pro` 選項（需付費）調整垂直/水平間距（Offsets），以避開官網原有的排版元件。

??? quote "使用 GetButton 會影響網站載入速度嗎？" 
	由於 GetButton 屬於外部載入的輕量級 Script，對網站載入速度影響極微。但為確保最佳效能，建議 **不要同時安裝多個性質相同的客服外掛**。

設定網站標題、簡述與關鍵字，以優化搜尋引擎 (SEO) 排名並提升社群分享時的品牌辨識度。

## 網站標題說明

### 欄位定義與建議

- **網站標題 (Title)**：網頁的核心名稱。應包含「品牌名」與「核心關鍵字」。
- **網站簡述 (Description)**：網頁內容的摘要。建議以吸引點擊的文案為主，包含主要服務項目。

---

### 顯示位置說明

這兩項設定主要會出現在以下三個地方：

#### 瀏覽器頁籤 (Browser Tabs)
    
- **顯示內容：** 網站標題 (Title)。
	
- **作用：** 幫助使用者在開啟多個視窗時，快速辨識您的品牌。

![](../../../assets/images/ec-網站外觀-網站標題-瀏覽器分頁顯示.png)

---

#### 搜尋引擎結果頁 (SERP)
    
- **顯示內容：** 標題（藍色大字）與 簡述（下方灰色小字）。
	
- **作用：** 這是消費者對品牌的第一印象，直接影響點擊率 (CTR)。

![](../../../assets/images/ec-網站外觀-網站標題與簡述-google搜尋顯示.png)

---

#### 社群平台分享 (Social Media Sharing)
    
- **顯示內容：** 當連結貼至 LINE 或 Facebook 時，會抓取標題與簡述。
	
- **注意：** 若有設定 **OG Image (分享縮圖)**，效果會更佳。

![](../../../assets/images/ec-網站外觀-網站標題-社交媒體分享縮圖.png)

**網站標題**（TITLE）設定與修改方式，會根據您使用 **版型類型**（一般版型或拖拉版型）而有所不同。

以下為網站設定中「網站標題」與相關 SEO 資訊的詳細教學：

## 設定基礎「網站名」

在修改標題文字前，應先設定好品牌的基本名稱，此名稱會與網站標題連動。

- **後台路徑**：登入 CYBERBIZ 管理後台，前往 **管理中心 > 一般設定 > 網站名**。
- **設定規範**：中文字請勿超過 15 字，英文字請勿超過 30 字。此名稱預設為您網址中的英文字串，建議修改為品牌名稱或公司名稱以利消費者辨識。

!!! note "詳細設定流程，請參考 [如何設定網站名稱](../website-management/設定網站基本資訊.md#關於您的網站){ data-preview }。"

## 修改「網站標題」：依版型設定

### 預設版型（一般版型）

預設版型僅能設定「網站標題」，且需透過程式碼編輯器修改。前台標題預設會顯示「網站名 | 歡迎光臨」。

- **設定步驟**：
    1. 登入 CYBERBIZ 管理後台，前往 **網站外觀 > 套版主題管理**。
    2. 在欲編輯的主題點選 **選擇操作 > CSS/HTML 編輯器**。
    3. 搜尋並點開 **`theme.liquid`** 檔案。
    4. 於檔案中找到此段程式碼：`{{ shop.name }} {{ page_title }}`。
- **修改邏輯**：
    - **顯示自訂文字**：刪除 `{{ page_title }}` 後，在 `{{ shop.name }}` 後方直接輸入您想顯示的文字。

``` liquid title="theme.liquid" hl_lines="3"
<title>
  {% if template == 'index' %}
	{{ shop.name }} 自訂文字
  {% elsif template == '404' %}
	404 頁面不存在 {{ shop.name }}
  {% else %}
	{{ page_title }} {{ shop.name }}
  {% endif %}
</title>
```


!!! tip "更新時間與復原機制" 
	- **Google 搜尋結果不會立即更新**：新設定的標題需等待 Google 完成下一次排程抓取與更新後才會顯示。 
	- **程式碼復原機制**：若修改後導致版面異常，請善用編輯器內的 **查看之前版本** 功能，[回溯至先前版本](使用樣板編輯器恢復網頁代碼.md#操作步驟){ data-preview }。

### 拖拉版型

拖拉版型不需修改程式碼，可直接在視覺化介面填入資訊。

- **後台路徑**：登入 CYBERBIZ 管理後台，前往 **網站外觀 >  套版主題管理 >  網站設定 > 全站設定**。
- **設定欄位**：
    - **網站標題**：輸入自訂標題。若不需要額外標題，保持留空即可。
    - **網站簡述**：請依照商家需求撰寫品牌簡介。
    - **網站關鍵字**：此欄位不會顯示在頁面上，但有助於搜尋引擎搜尋到您的商品。

## 常見問題

??? quote "為什麼我修改了標題，但在 Google 搜尋結果上還是顯示舊的名稱？" 
	這屬於正常現象。Google 的搜尋結果是基於其資料庫中的「索引快取」： 
	
	- **更新週期**：Google 爬蟲（Spider）需要時間重新抓取（Crawl）您的網站資料，通常需 **1 至 2 週** 才會反應在搜尋結果中。 
	- **手動加速**：若您有建立 [Google Search Console](https://search.google.com/search-console/about)，可以提交「網址審查」並請求重新編制索引，以縮短等待時間。

??? quote "「網站標題」和「一般設定中的網站名」有什麼不同？" 
	兩者為連動關係，但層次不同： 
	
	- **網站名 (Shop Name)**：是您品牌的基礎名稱（如：CYBERBIZ 官方商城）。 
	- **網站標題 (Title)**：是顯示在瀏覽器頁籤上的完整字串。 
	- **關聯性**：在預設版型中，標題通常由 `{{ shop.name }}` + `{{ page_title }}` 組成。如果您修改了一般設定中的網站名，前端標題中的品牌部分也會同步更新。

??? quote "在編輯 `theme.liquid` 時，如果不小心刪錯程式碼導致網站變空白怎麼辦？" 
	請保持冷靜，這可以透過系統機制快速修復： 
	
	1. **切勿關閉視窗**：先嘗試使用快捷鍵 `Ctrl + Z` (Windows) 或 `Cmd + Z` (Mac) 撤銷動作。 
	2. **版本回溯**：若已儲存，請點擊編輯器右上方的 **「查看之前版本」**。 
	3. **選擇時間點**：系統會列出近期的儲存紀錄，選擇修改前（正常狀態）的時間點進行 **「恢復」**，即可立即將網站還原。

??? quote "「網站關鍵字」欄位填寫越多越好嗎？" 
	不是。現代搜尋引擎（如 Google）已不再將 HTML 中的 `keywords` 標籤作為主要的排名依據： 
	
	- **建議做法**：填寫 **3-5 個** 與品牌最相關的核心詞彙即可。 
	- **避免懲罰**：過度堆疊無意義的關鍵字（Keyword Stuffing）可能會被搜尋引擎視為垃圾訊息，反而對 SEO 有負面影響。

# 使用樣板編輯器恢復網頁代碼

透過程式碼編輯器的版本管理功能，快速回溯 HTML/CSS 至先前時間點或初始狀態，確保網頁修改後的穩定性。
{ .subtitle }

![](../../../assets/images/ec-網站外觀-樣板編輯器-查看之前版本.png){ .hero-page }

## 樣板編輯器恢復樣板版本說明

樣版編輯器的 **查看之前版本** 功能讓商家在自行修改 HTML、CSS 或 JavaScript 程式碼後，若發生頁面異常或跑版等狀況，可以隨時將檔案回溯至先前的版本。

以下為恢復樣版編輯器的操作說明與注意事項：

## 操作步驟

1. **進入路徑**：登入  CYBERBIZ 管理後台，前往 **網站外觀 > 套版主題管理**。
2. **開啟編輯器**：點選「**選擇操作**」（或直接點擊）進入「**CSS/HTML 編輯器**」。
3. **查看版本**：在編輯器介面點選「**查看之前版本**」按鈕。
4. **選擇紀錄**：系統會列出先前儲存過的紀錄，這些紀錄僅提供「**更新的時間點**」供參考。
5. **執行恢復**：選擇欲回溯的時間點。

![](../../../assets/images/ec-網站外觀-樣板編輯器-查看之前版本操作.gif)

## 重要注意事項

- **責任歸屬**：CYBERBIZ 公開版型程式碼供商家自行客製化調整，但商家需委託自家工程師或外部設計單位處理，**官方不提供現有文件外的修改、語法教學或代碼撰寫服務**。
- **自行修復**：若因自行設定導致頁面異常，商家需自行參考恢復功能將檔案還原。
- **版型限制**：部分功能（如於[訂單成立頁新增 LINE 好友連結](../integrations/line/設定訂單成立頁與付款完成頁顯示 LINE 加入好友連結.md){ data-preview }  ）僅套用於「目前已發布」的版型，若更換版型後程式碼遺失，可能需重新申請或自行保留代碼。
- **拖拉版型限制**：拖拉版型雖支援部分程式碼編輯（如鎖右鍵、SEO 設定等），但不支援所有的程式碼修改，建議依後台開放的功能為主。

## 常見應用場景

當您在執行以下進階客製化卻遇到異常時，可利用此恢復機制：

- **首頁設定**：如修改首頁跑馬燈秒數或轉場時間。
- **商品頁優化**：如鎖定圖片放大預覽、修改商品標語文字呈現、或嵌入影片自動播放。
- **結帳頁修改**：如調整海外訂單優先填寫收件人、或修改店長改價的顯示名稱。
- **外掛嵌入**：如在自訂頁面嵌入 Google 表單或 Instagram 動態牆語法時發生衝突。

## 常見問題

??? quote "恢復版本後，會影響到已經上傳的商品圖片或素材嗎"
	**不會。** 樣板編輯器的恢復功能僅針對 `.html`, `.css`, `.js` 等程式碼檔案。您儲存在媒體庫中的圖片或後台設定的商品資料屬於資料庫層級，不會因為程式碼回溯而被刪除。

??? quote "為什麼我在「查看之前版本」中找不到剛才儲存的紀錄"
	系統的版本紀錄通常在點擊「儲存」時產生。若頻繁連續儲存，系統可能會合併極短時間內的異動。建議每次重大修改測試成功後再執行儲存，以便建立清晰的版本節點。

# 編輯商品描述與設定
設定商品內容、通路與物流屬性，確保前台呈現正確並支援搜尋與行銷需求。
{ .subtitle }

![](../../assets/images/ec-product-description-and-settings.zh-tw.png){ .hero-page }

## 商品編輯頁介紹

在商品編輯頁面中，你可以編輯與設定商品的各項資訊與屬性。頁面主要分為以下三個頁籤：

- [商品資訊]((新增單一商品#操作流程) )：商品名稱、規格、價格與庫存等核心資料。
- [商品描述](#商品描述)：商品說明、規格與運送資訊，呈現給前台顧客。
- [設定](#設定)：類型、通路、廠商、群組、標籤、物流與 SEO，管理後台屬性與行銷設定。

## 操作流程

### 商品描述

1. 登入 CYBERBIZ 管理後台，前往 **商品 > 所有商品**。
2. 點擊欲編輯的商品名稱進入編輯商品頁面。
3. 依序編輯以下欄位：
	- **商品介紹**：提供詳細商品資訊
	- **規格說明**：說明商品規格細節 
	- **運送方式**：說明商品運送相關資訊

### 設定

![](../../assets/images/ec-product-settings.zh-tw.png)

#### 進階設定

- **自訂群組**：將商品加入已建立的商品群組，請參考[設定商品分類群組](設定商品分類群組)。
- **任選折扣群組**：商品加入已建立的任選折扣群組，請參考[任選折扣](設定任選折扣群組)。
- **標籤**：為商品添加標籤，通常用於行銷活動，例如指定商品優惠券，請參考[設定商品標籤](設定商品標籤)。 
- **商品類型***：為商品添加類型標籤，以利商品管理篩選（資訊會顯示於前台），可自行設定。
- **商品通路***：設定商品可銷售的通路，不同通路[^1]商品結帳時將拆分成[多購物車](設定多購物車#使用須知)。
- **商品廠商***：為商品添加廠商標籤，以利商品管理篩選（資訊會顯示於前台），可自行設定。 
- **Google 產品類別**：選擇設定類別以覆蓋 Google 自動判斷結果，讓廣告投放更加準確。了解如何 [設定 Google 購物廣告](設定 Google 購物廣告)。

!!! warning "注意事項"
	商品類型、商品通路、商品廠商等設定，新增後無法刪除。

#### 溫層和物流配送設定

![](../../assets/images/ec-shipping-temperature-logistics-setting.zh-tw.png)

> 建議先行[設定物流方式](設定商品配送屬性（一般宅配）)後再做設定，可直接選到適配物流方式 。

設定商品的運送溫層與可用的物流配送方式。

- **運送溫層**：設定商品可接受的運送溫層。了解如何綁定[一般宅配](設定商品配送屬性（一般宅配）)跟[宅配貨到付款](設定商品配送屬性（宅配貨到付款）)商品的配送物流、溫層及通路。
- **物流綁定狀態**：設定商品可使用的物流配送方式。了解如何[設定商品超商物流限制與排除選項](設定商品超商物流限制與排除選項)

#### 設定相關商品

![](../../assets/images/ec-related-collection.zh-tw.png)

透過設定 *商品關聯群組*，商家可以自訂商品之間的關聯性，以提升商品頁下方 *相關商品* 區的展示效果，進而增加連帶銷售。

#### SEO 設定
![](../../assets/images/ec-product-seo-setting.zh-tw.png)

商品設定 SEO 可增加網頁在搜尋引擎上的排名，請依照商品特性設定。

- **網頁標題**：顯示於網頁瀏覽器，影響消費者搜尋時找到此商品頁的標題。
- **網頁描述**：顯示於網頁瀏覽器，影響消費者搜尋時找到此商品頁的描述。
- **網頁關鍵字**：通常不顯示於網頁瀏覽器，影響消費者搜尋時找到此商品頁的關鍵字。

## 常見問題

??? quote "商品描述可以使用 HTML 標籤嗎？"
    商品描述區塊（商品介紹、規格說明、運送方式）支援 CKEDITOR 編輯器，可進行文字排版、圖片與連結編輯。若需更進階的排版功能，建議使用拖拉版型。

??? quote "設定商品溫層與物流配送時，有什麼需要注意的？"
    建議您在設定商品溫層與物流方式前，先行設定好物流方式，以便直接選取適配的物流選項。

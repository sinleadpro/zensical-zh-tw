## 說明概述
> 票券發行人為順立智慧股份有限公司，票券面額為實際售價（含優惠），已存入發行人於永豐銀行開立之「信託專戶」，專款專用，保障履約安全。

=== ":material-information-outline: 使用須知"
	- [x] 可加入「限購群組」，以限制消費者購買張數。
	- [ ] 僅可使用「電子票券任選折扣」或「分潤功能」，無法與其他優惠、折扣或行銷活動併用。  
	- [ ] 僅支援「信用卡一次付清」，不支援分期付款或其他付款方式。

=== ":material-lightbulb-outline: 使用情境"
	-  銷售數位商品/服務：品牌可透過網站銷售各類型的電子票券，如課程、活動門票或服務兌換券。
	- 線上線下整合：實現消費者線上購買票券，線下透過 QR Code 掃碼核銷，促進會員導流與消費體驗。
	- 彈性票種管理：針對不同活動或商品，設定多種票種規格（如早鳥票、團體票、商品兌換組），滿足多元銷售需求。
	- 門市核銷管理：提供多種核銷方式，並可設定專屬門市店員權限，簡化核銷流程。

## 顧客購買流程
1.  **商品頁面**：電子票券商品頁面會顯示[[購買電子票券#步驟一：票券商品頁面|票券注意事項]]，提醒顧客使用辦法。
2.  **購物車**：若購物車內同時包含票券商品與一般商品，系統會要求[[購買電子票券#步驟二：購物車分拆|拆分結帳]]。
3.  **結帳**：消費者需勾選「[[購買電子票券#步驟三：勾選同意|同意電子票券條款]]」才能完成結帳。
4.  **取得票券**：付款完成後，消費者可透過「會員專區 > [[購買電子票券##電子票券訂單查詢-前台|電子票券訂單查詢]]」或「[[購買電子票券##電子票券確認信-信箱|電子票券確認信]]」取得核銷用的 QR Code。

## 常見問題

??? question "核銷後可以再進行退款嗎？"
    核銷後則無法退款。若消費者仍須退款，須由商家與消費者自行溝通處理。

??? question "若不小心核銷錯誤，怎麼辦？"
    核銷錯誤目前無法復原。

??? question "為什麼我的票券購物車會計算到運費？"
    您的物流設定頁有可能還是舊版。請通知 CYBERBIZ 客服升級新版物流，即可修正此問題。

??? question "若同一票券購買 5 張，會拿到幾個 QR Code 呢？"
    1 個。票券的 QR Code 為一對多的形式，一個 QR Code 可能包含一個或多個數量的商品。全數核銷完畢後，該 QR Code 將無法再進行核銷。

# 設定超商配送限制與物流排除

設定商品的材積與重量，讓系統自動判斷訂單是否符合超商物流（B2C／C2C）的配送限制。
{ .subtitle }

![](../../assets/images/ec-product-variants-volumn-weight.zh-tw.png){ .hero-page }

---

## 超商物流限制規格

以下為超商物流的尺寸與重量限制，請務必參考最新規定：

| 超商類型           | 尺寸限制                               | 重量限制             |
|------------------|----------------------------------------|----------------------|
| **B2C / C2C**    | 三邊總和 ≤ 105cm，單邊最長 ≤ 45cm<br>(全家B2C：三邊總和 ≤ 120cm，最長邊 ≤ 50cm)      | C2C：重量 ≤ 10 kg<br>B2C：重量 ≤ 10 kg     |
| **全家冷凍 B2C / C2C** | 最長邊 ≤ 45cm | S60：重量 ≤ 5kg<br>S120：重量 ≤ 10kg |

!!! tip "以規格條件觸發配送排除"
	當商品的材積（三邊總和）或重量超過超商物流限制時，系統會自動判定該商品不符合超商配送條件，結帳頁面將不顯示超商物流選項。  
	瞭解 [如何設定指定商品無法使用超商物流](#設定指定商品無法使用超商物流)。

!!! warning "物流限制易變"
	超商物流規格可能隨時異動，請依各超商物流業者最新公告為準，以確保商品材積與重量設定符合實際配送要求。

## 設定商品材積與重量

您可以為每個商品設定其材積與重量資訊，以利系統自動判斷是否符合超商配送條件。

1.  登入 CYBERBIZ 管理後台，前往 **商品 > 所有商品**。
2.  點擊欲設定的商品，進入編輯頁面。
3.  在 **款式管理** 區域中，找到商品的材積與重量欄位：
    -   於「材積」欄位輸入商品三邊總和（單位：公分）。
    -   於「重量」欄位輸入商品重量（單位：公斤）。
4.  點擊右上角 **儲存**，完成設定。

!!! info "系統判斷邏輯"
    - 顧客進入訂單結帳頁時，系統將自動計算該訂單 *材積與重量總和* 是否符合超商物流配送條件。若超過超商限制，則訂單結帳頁不會顯示超商選項。
    - 若材積或重量欄位未填寫，系統預設為 `0`，即視為符合所有配送條件。

![](../../assets/images/ec-product-volume-weight-setting.zh-tw.png)

## 設定指定商品無法使用超商物流

若您希望特定商品無法使用超商物流，可透過以下方式手動設定：

1.  登入 CYBERBIZ 管理後台，前往 **商品 > 所有商品**。
2.  點擊欲設定的商品，進入編輯頁面。
3.  在 **款式管理** 區域中，設定以下任一條件：
    -   **材積欄位**：輸入 `106` 公分（超過超商材積上限 105 公分）。
    -   **重量欄位**：
        -   常溫商品輸入 `6` 公斤（超過超商重量上限 5 公斤）。
        -   冷藏冷凍商品輸入 `11` 公斤（超過全家冷凍 S120 重量上限 10 公斤）。
4.  點擊 **儲存**，完成設定。

!!! info "效果說明"
    當訂單中包含 *材積或重量超出超商物流限制的商品* 時，系統將判定整筆訂單不符合超商配送條件，結帳頁面將不顯示超商取貨選項。

## 情境範例

### 情境一：商品總材積超過限制

- 大倉鼠商品材積：30 cm/件
- 小豬豬商品材積：50 cm/件

??? quote "購買 1 個大倉鼠和 2 個小豬豬，這樣可以使用超取嗎？"
	商品總材積為 30 + (50 * 2) = 130 cm。由於總材積超過超商材積限制 105 cm，因此無法使用超商取貨付款/不付款。

![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/排除超取選項設定02.png){ .screenshot }

### 情境二：商品總材積符合限制（含未填寫欄位）

- 紫水晶商品材積：55 cm/件
- 粉水晶商品材積：50 cm/件
- 綠水晶商品材積：留空（系統預設為 0 cm/件）

??? quote "購買 1 個紫水晶、1 個粉水晶以及 1 個綠水晶，是否可以使用超取呢？"
	商品總材積為 55 + 50 + 0 = 105 cm。由於總材積等於超商材積限制 105 cm，因此可以使用超商取貨付款/不付款。

![](https://www.cyberbiz.io/helpcenter/wp-content/uploads/排除超取選項設定03.png){ .screenshot }

## 常見問題

??? quote "如果商品材積或重量未填寫，系統會如何判斷？"
    若商品材積或重量欄位未填寫，系統將預設為 `0`，並視為符合所有配送條件。

??? quote "超商物流限制會隨時間變動嗎？"
    是的，超商物流規格可能隨時異動。請以各超商物流業者最新規定為主。

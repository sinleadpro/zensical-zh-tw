# 設定前台商品群組排序
調整「自訂群組」、「智慧群組」及「任選折扣群組」在首頁「商品列表」的顯示順序，並設定顯示的群組數量。
{ .subtitle }

!!! tip "應用情境"

	- **優化商品曝光**：根據行銷策略或商品重要性，調整群組顯示順序，確保重點商品群組優先曝光。
	- **精簡頁面呈現**：控制首頁商品列表顯示的群組數量，避免資訊過載，提升顧客瀏覽體驗。
	- **活動商品管理**：在特定促銷活動期間，將相關群組置頂，提高活動商品的能見度。

## 操作流程

### 調整商品列表群組排序

您可以透過拖曳方式，調整首頁「商品列表」中各商品群組的顯示順序。

1.  登入 CYBERBIZ 管理後台，前往 **網站外觀 > 全站商品列表群組排序**。
2.  在群組列表中，點選 **十字箭頭符號** :material-drag-variant:，上下拖曳以調整群組順序。
    *   **僅顯示** 設定為 `公開` 狀態的群組。

    ![](https://www.cyberbiz.co/support/wp-content/uploads/2019/03/群組排序1.png){ .screenshot }
    ![](https://www.cyberbiz.co/support/wp-content/uploads/2019/03/群組排序2.png){ .screenshot }

### 設定首頁商品列表顯示群組數量

您可以設定首頁「商品列表」中顯示的群組個數。

1.  登入 CYBERBIZ 管理後台，前往 **網站外觀 > 套版主題管理 > 導覽列Navbar > 導覽列選單**。
2.  直接修改 **群組數量**。
    *   **範例**：若將群組數量設定為 `4`，即使有 `8` 個群組設為公開，首頁的「商品列表」仍只會顯示 `4` 個群組。

    ![](https://www.cyberbiz.co/support/wp-content/uploads/2019/03/群組排序3.png){ .screenshot }

## 常見問題

??? quote "為什麼我調整了群組排序，但前台沒有生效？"
    請確認您調整的群組是否已設定為 `公開` 狀態。此外，也請檢查「導覽列選單」中的「群組數量」設定，確保顯示數量足夠包含您想看到的群組。

??? quote "「自訂群組」、「智慧群組」和「任選折扣群組」有什麼不同？"
    -   **自訂群組**：手動選擇商品加入的群組。
    -   **智慧群組**：根據您設定的條件自動篩選商品加入的群組。
    -   **任選折扣群組**：用於設定任選折扣活動的商品集合。

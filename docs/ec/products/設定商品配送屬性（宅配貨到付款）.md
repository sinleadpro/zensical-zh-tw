# 設定商品配送屬性（宅配貨到付款）
綁定宅配貨到付款商品的配送物流、溫層及出貨通路
{ .subtitle } 

!!! tip "應用情境"

	- 配送物流綁定：當商品需指定特定物流（如黑貓、宅配通、郵局等）配送時，可彈性設定。
	- 配送溫層綁定：針對需特殊溫層（如常溫、冷藏、冷凍）配送的商品，可設定多溫層選項。
	- 出貨通路綁定：當商品分屬不同出貨地點（如不同倉庫、門市或廠商）時，可設定商品通路。

## 使用須知

- 當購物車內的商品屬於不同 *配送屬性* 時，系統將自動拆分成[多購物車結帳](設定結帳拆分多購物車.md#多購物車結帳說明){ data-preview }，並個別計算優惠與運費。
- 商品綁定後新增物流時，請檢查僅限特定物流的商品設定，避免漏綁。

## 操作流程

### 設定物流不同溫層的運費

首先，您需要在後台設定各物流方式及其對應溫層的運費規則。

1. 登入 CYBERBIZ 管理後台，前往 **金物流 > 宅配物流**。
2. 找到欲設定的物流選項，點擊 :material-text-box-edit-outline: 進入編輯頁面。

	![](../../assets/images/ec-conntect-logistics-edit.zh-tw.png)
	
3. 開啟貨到付款及免運

	> 以設定黑貓貨到付款為例。
	
	- 開啟黑貓貨到付款功能。
	- 開啟免運並設定免運金額。
	
	!!! warning "若設定的代收金額已超過上方設定的免運門檻，運費將被忽略，視同免運費。"

	![](../../assets/images/ec-conntect-logistics-settings.zh-tw.png)
	
4. 設定溫層運費
	設定不同溫層運費（只能依代收的訂單金額作為運費標準）。

	![](../../assets/images/ec-conntect-logistics-temp-fees.zh-tw.png)

### 設定商品配送溫層

完成物流運費設定後，您可以為個別商品綁定其適用的運送溫層。

1. 登入 CYBERBIZ 管理後台，前往 **商品 > 所有商品。
2. 點擊欲設定的商品，進入編輯頁面。
3. 點擊 **設定** 頁籤。
4. 在 **溫層和物流配送設定** 區塊中，勾選該商品綁定單個或多個可[設定溫層選項](#設定物流不同溫層的運費)（預設為常溫）。

![溫層配送物流設定](https://www.cyberbiz.co/support/wp-content/uploads/2020/07/溫層配送物流設定-一般宅配11.png){ .screenshot }

### 設定商品配送方式

您可以為個別商品綁定其可用的配送物流方式。

1. 登入 CYBERBIZ 管理後台，前往 **商品 > 所有商品**。
2. 點擊欲設定的商品，進入編輯頁面。
3. 在商品編輯頁面中，找到「溫層和物流配送設定」區塊。
4. 若有開啟黑貓/宅配通/順豐相關貨到付款功能，即會顯示對應選項。

	- 物流綁定狀態： `ON` 代表可使用該快遞/物流配送；`OFF` 則代表此商品不適用於該快遞/物流配送。
	- 商品預設為全部物流選項皆綁定（即所有物流皆可配送）。
	
	![溫層配送物流設定](https://www.cyberbiz.co/support/wp-content/uploads/2020/07/溫層配送物流設定-一般宅配7.png){ .screenshot }

5. 確認修改提醒：首次修改綁定狀態時，會跳出設定提醒。請閱讀完注意事項後，勾選並點擊「確認」。若未勾選確認，每次修改物流綁定都會跳出提醒。

	![](../../assets/images/ec-logistics-binding-dialog.zh-.tw.png)

## 情境範例
### 商品綁定後新增貨到付款物流
 如果您先設定商品配送綁定，再開啟黑貓／宅配通貨到付款物流功能，商品綁定狀態會依下列規則更新：

- :material-check: **原本「適用所有物流」的商品**  ➜ 系統會自動綁定新物流（仍適用所有物流）。
- :material-alert-outline: **原本「僅綁定特定物流」的商品** ➜ 不會自動綁定新物流，需[[#步驟三：設定商品配送方式（物流）|手動設定]]。

| 商品 | 原物流設定 | 新增物流後結果 |
|---|---|---|
| A | 適用黑貓＋宅配通（全部物流） | :material-check: 自動新增黑貓／宅配通貨到付款 |
| B | 僅適用黑貓 | :material-close: 不自動新增物流（仍僅黑貓） |
	
!!! warning "新增物流後，請檢查特定物流綁定商品，避免漏綁。"

### 不同配送物流、溫層、出貨通路

**後台設定**

- 物流運費設定：設定黑貓、宅配通兩種物流配送方式，商品設定頁面就會有此兩種物流可選擇綁定。
- 溫層配送：已內建常溫、冷藏、冷凍三種，商品設定頁面會有此三種物流可選擇綁定。
- 商品頁面設定：設定台北門市、台中門市兩個商品通路（出貨地點）。

??? tip "簡化商品 *配送屬性* 設定以降低棄單風險"
	過多的商品配送屬性（商品通路、溫層、配送物流）會導致系統在結帳時自動將商品拆分至多個購物車。每個購物車需獨立結帳，並生成各自的訂單，且優惠與運費將分別計算。此情況會增加結帳步驟與運費成本，可能導致顧客棄單。建議僅在必要時設定多種配送屬性，以簡化購物流程並提升結帳體驗。

**前台呈現**
當顧客購買了不同配送物流、溫層、出貨通路的商品時，前台畫面呈現如下：

-   最上方會列出不同的出貨通路，顧客先選擇一種出貨通路，再選擇該通路中其中一個配送溫層，溫層中再選擇配送物流，進行結帳。
-   剩餘未結帳的商品會保留在購物車中等待結帳。
-   顧客須完成多個購物車的結帳，建立多筆訂單（若未達各購物車免運門檻，則須支付多筆運費）。

## 常見問題

??? quote "如果我修改了物流名稱，商品綁定會失效嗎？"
    是的，若您變更物流方式的名稱，原本商品綁定的物流都會被取消綁定。您需要到商品頁面重新綁定物流配送方式。

??? quote "我可以設定多個溫層給一個商品嗎？"
    可以。在商品編輯頁面的「設定」頁籤中，您可以勾選多個溫層來綁定商品。

??? quote "為什麼顧客結帳時會出現多個購物車？"
    當顧客購買的商品包含不同溫層、配送物流或出貨通路時，系統會自動將這些商品分屬在獨立的購物車中，每個購物車需獨立結帳。瞭解[[設定結帳拆分多購物車#多購物車結帳說明|更多]]。

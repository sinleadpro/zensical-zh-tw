# 設定商品分類群組
自訂分類群組手動整理商品，優化商店頁面、支援行銷活動並提升 SEO 可見性。

!!! tip "應用情境"
	- 商品歸類與頁面優化：手動勾選商品，將其歸類至自訂分類，使商店頁面更整齊，提升消費者查找商品的便利性。
	- 行銷活動應用：可針對特定分類商品進行行銷活動，例如[單品限時折扣](設定單品限時折扣群組)，有效提升銷售效果。
	- SEO 優化：各自訂分類可獨立設定分類描述與關鍵字，優化 SEO 表現，讓商品更容易被搜尋引擎找到。

## 操作流程

1. 登入 CYBERBIZ 管理後台，前往 **商品 > 商品自訂分類**。

![商品自訂分類介面](https://www.cyberbiz.io/support/wp-content/uploads/商品自訂分類01.png){ .screenshot }

- 欄位顯示：依需求勾選欲展示於下方商品列表的詳細商品資訊。
- 搜尋商品：使用關鍵字（商品名稱或 SKU）快速搜尋商品。
- 篩選選項：依類型、廠商、商品標籤、商家等條件篩選商品。了解更多[[篩選商品建立分類群組|篩選細節]]。
- 商品列表：顯示現有商品資訊。
- 群組列表：顯示現有分類群組，可編輯群組名稱、網址及商品排序方式。

### 將商品加入分類群組

1. 在群組列表下方欄位中輸入欲新增的群組名稱。
2. 點擊 **新增群組** 按鈕，建立新的商品分類群組。

=== ":material-numeric-2-circle: 選擇分類群組"
	![[ec-商品自訂分類-展開分類群組.png]]
	
	- [ ] 在群組列表中，點擊欲新增商品的分類群組名稱。
	- [ ] 被選取的分類群組會展開顯示相關資訊。
	
=== ":material-numeric-3-circle: 加入商品"
	![[ec-商品自訂分類-加入商品.png]]
	
	- [ ] 使用關鍵字(名稱、SKU) 「搜尋商品」:material-magnify: 或「篩選項目」:material-filter-outline: 定位目標商品。更多細節，請看如何[[篩選商品建立分類群組]]。
	- [ ] 單筆加入：勾選欲加入選定群組的單一商品，點擊「加入」。
	- [ ] 批次加入：勾選多個商品後，點擊「加入群組」。
	- [ ] 成功加入的商品會顯示於右方展開的商品群組清單中，顯示所屬商店、名稱及溫層等資訊。

=== ":material-numeric-4-circle: 移出商品"
	![[ec-商品自訂分類-移出商品.png]]
	
	- [ ] 單筆移出：勾選欲加入選定群組的單一商品，點擊「移出」。或點擊右方商品群組清單中的 :material-close:。
	- [ ] 批次移出：勾選多個商品後，點擊「移出群組」。

=== ":material-numeric-5-circle: 快捷按鈕功能"
	![[ec-商品自訂分類-群組列表-快捷按鈕.png]]

	=== ":material-earth: 瀏覽群組頁"

		- [ ] 點擊 :material-earth: 前往官網前台，查看該群組的展示畫面。

	=== ":material-square-edit-outline: 編輯群組"
	
		- [ ] 點擊 :material-square-edit-outline: 進入群組編輯頁面，設定群組描述與 Meta Tag。

	=== ":material-eye: 切換公開狀態"

		- [ ] 點擊 :material-eye: 切換群組在前台的顯示狀態：公開 :material-eye:；不公開 :material-eye-off:。
	
	=== ":material-close: 刪除群組"
		
		- [ ] 點擊 :material-close: 移除選定的商品群組。

=== ":material-numeric-6-circle: 排序商品"
	![[ec-商品自訂分類-排序商品.png]]
	
	- [ ] 點擊「調整排序」下拉選單，依據需求選擇預設選項自動排序或手動排序。
	- [ ] 選擇「手動排序」，然後點擊 :material-arrow-all: 拖曳商品以調整順序。

	!!! warning "設定商品排序時，若「商品名稱」包含系統不支援的標點符號，可能導致排序功能無法正常運作。點擊名稱可進入編輯頁面修改。" 

### 步驟三：編輯分類描述與 SEO 設定
=== ":material-numeric-1-circle: 進入編輯頁面"
	![[ec-商品自訂分類-編輯群組.png]]
	
	- [ ]  前往「商品」→「商品自訂分類」，選擇欲編輯的群組，然後點擊「編輯群組」 :material-square-edit-outline:。
	- [ ] 點擊「群組名稱」可直接對其編輯。

=== ":material-numeric-2-circle: 編輯群組描述"
	![[ec-商品自訂分類-編輯群組-群組描述分頁.png]]

	- [ ] 在「群組描述」分頁的編輯器中，輸入或修改商品群組的文字描述。
	- [ ] 完成編輯後，點擊「儲存」套用變更。

=== ":material-numeric-3-circle: 設定群組頁橫幅"
	<div class="grid cards two-columns" markdown>
	
	- ![](https://www.cyberbiz.io/support/wp-content/uploads/EC-前台-商品群組頁-置頂橫幅版位01.png)
	- ![](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-商品-商品自訂分類-群組描述01.png)

	</div>
	
	在前台商品群組頁設計置頂橫幅，作為行銷活動或重點資訊的展示區塊。
	
	- [ ] 點擊「群組描述」頁籤。
	- [ ] 在編輯器內新增圖片、影片及相關文字敘述等。

=== ":material-numeric-4-circle: 設定 Meta Tag"
	![[ec-商品自訂分類-編輯群組-Meta Tag設定分頁.png]]
	
    - [ ] 在「Meta Tag 設定」分頁中，設定商品群組的關鍵字與 SEO 相關資訊，以提升搜尋引擎排名。

### POS 系統商品管理  

[:material-point-of-sale:](){ .badge-icon title="適用系統" }[POS](){ .badge-text }

使用 POS 系統商家，若想管理商品並建立 POS 選單，請設定 POS 商品多層級分類，並建立商品自訂分類或商品類型。條件分類恕不支援 POS 系統。詳細操作可洽 [POS前台選單設定](https://www.cyberbiz.io/support/?p=11224)。

## 常見問題

??? quote "前台的分類一次最多可以顯示幾個？"
	最多 20 個。

??? quote "商品群組可以設定多層級分類嗎？"
    商品自訂分類目前不支援多層級設定。若您需要多層級分類，建議參考 POS 商品多層級分類功能。

??? quote "如果我刪除了商品群組，群組內的商品會被刪除嗎？"
    不會。刪除商品群組僅會移除該分類群組，群組內的商品仍會保留在您的商品列表中。

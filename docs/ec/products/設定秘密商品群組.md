# 設定秘密商品群組
建立隱藏商品群組，透過專屬連結提供給特定顧客購買。
{ .subtitle }

!!! tip "應用情境"
	- 特定客群專屬優惠：提供經銷商、團購主或員工專屬的商品購買連結，享受特殊價格或獨家商品。
	-   網紅分潤導購： 結合第三方推薦碼，為網紅或合作夥伴建立專屬導購連結，追蹤分潤成效。
	-   限量商品預售： 在商品正式上架前，提供給 VIP 顧客或特定會員進行預購。

## 使用須知

 - 建立秘密商店後，請勿將群組連結設於官網導覽列，以免一般使用者點擊進入。
 - 秘密商店群組頁仍可能被 Google 搜尋引擎索引，進一步避免曝光請至 [Google Search Console 設定排除頁面](https://www.cyberbiz.io/support/?p=25441#b) 將頁面排除於搜尋結果。  

## 操作流程
### 設定秘密商店
1. 登入 CYBERBIZ 管理後台，前往 **商品 > 商品自訂分類**。
2. 點選右下角 **新增群組**，輸入自訂的群組名稱。
3. 從左側商品列表中，將欲加入秘密群組的商品拖曳至右側群組內。
	
	![](https://www.cyberbiz.io/support/wp-content/uploads/秘密群組設定01.png)

4. 排除商品站內搜尋
	不讓秘密群組內的商品被其他顧客在站內搜尋到。
	
	- 登入 CYBERBIZ 管理後台，前往 **商品 > 所有商品 > 商品名稱 > 商品資訊** 分頁。
	- 將 **商品搜尋功能** :material-toggle-switch: 開關切換為 `關閉 (OFF)`。詳細設定請看[設定商品排除搜尋](設定商品排除搜尋)。

    ![](https://www.cyberbiz.io/support/wp-content/uploads/秘密群組設定02.png){ .screenshot }

### 運用秘密群組

您可以將秘密群組連結提供給特定用戶，或結合第三方推薦碼作為網紅分潤導購工具。

#### 提供專屬連結給特定用戶

- [ ] 複製秘密群組的網址（例如：`www.yourstore.com/collections/秘密群組`）。
- [ ] 將此網址提供給經銷商、團購主或員工，他們即可透過此連結直接訪問並購買商品。
      
#### 搭配第三方推薦碼進行分潤導購

- [ ] 首先，請參考 [推薦人分潤-推薦連結說明](https://www.cyberbiz.io/support/?p=1874) 設定好第三方推薦碼。
- [ ] 將秘密群組網址與推薦人代碼結合，例如：
- 原網址：`www.rayray.cyberbiz.co/collections/秘密群組`
- 搭配推薦人代碼：`www.rayray.cyberbiz.co/collections/秘密群組?rcode=RAYTEST`
- [ ] 將帶有推薦碼的專屬網址提供給該推薦人，秘密商店即可開始營運，並追蹤分潤成效。

![](https://www.cyberbiz.io/support/wp-content/uploads/秘密群組設定05.png)

## 排除搜尋效果

當商品設定「關閉站內搜尋」時，該商品將不會被 CYBERBIZ 站內搜尋器與 Google 搜尋引擎搜尋到。

=== ":material-magnify: 站內搜尋"
    ![](https://www.cyberbiz.io/support/wp-content/uploads/秘密群組設定03.png)

=== ":material-monitor: 前台頁面"
    ![](https://www.cyberbiz.io/support/wp-content/uploads/秘密群組設定04.png)

!!! warning "秘密商店群組頁索引風險"
	雖然個別商品已關閉商品搜尋功能，但秘密商店群組頁的名稱仍可能被 Google 搜尋引擎索引，若希望避免曝光，請至 [Google Search Console 設定排除頁面](https://www.cyberbiz.io/support/?p=25441#b) 將頁面排除於搜尋結果。 
     

## 常見問題

??? quote "建立秘密群組後，商品會自動排除搜尋嗎？"
    不會。秘密群組僅是將商品歸類，您仍需手動進入商品編輯頁面，將群組內商品的「商品搜尋功能」設定為 `關閉 (OFF)`，才能排除站內搜尋。

??? quote "秘密群組的商品可以被一般顧客購買嗎？"
    如果一般顧客獲得了秘密群組的直接連結，並且該群組內的商品沒有設定「關閉站內搜尋」，那麼他們仍然可以訪問並購買。為確保專屬性，建議將群組內商品設定為排除搜尋。

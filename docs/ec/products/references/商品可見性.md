# 商品可見性
使用不同功能跟設定改變商品能見度。
{ .page-subtitle }

## 說明概述
=== ":material-lightbulb-outline: 使用情境"
    - 避免優惠價曝光：將促銷商品設定為排除搜尋，僅提供「商品連結」給特定顧客。
    - 管理未上架商品：將尚未準備好的商品設為不公開，避免提前曝光。
    - 控制促銷時間：利用上架狀態功能，精準控制商品上架與下架的時間點。

## 可見性功能總覽

<div class="grid cards" markdown>

- #### :material-eye:{ .lg .middle } [[公開狀態]]   
控制商品是否在前台顯示，用於測試、草稿或暫時隱藏。
- #### :material-calendar-range:{ .lg .middle } [[上架狀態]]  
設定商品自動上架或下架，掌控促銷檔期。    
[[archive/新增單一商品#基本設定|商品基本設定]]
- #### :material-magnify-close:{ .lg .middle } [[設定商品搜尋可見性]]   
將商品從網站與 Google 搜尋結果中排除，避免非目標顧客看到優惠價。


</div>

## 典型使用情境
- 在 **活動行銷** 前，先設為不公開，等到正式開跑再一次上架。
- 配合 **員購** 或 VIP 方案，將商品排除搜尋，僅提供專屬連結。
- 使用 **上架排程** 自動化控管，避免人工操作疏漏。

## 延伸應用
- 與 **促銷工具** 搭配：排程上架結合折扣碼，提升活動轉換率。
- 搭配 **SEO** 管理：僅公開需要曝光的商品，避免重複內容影響搜尋排名。
- 與 **API** 串接：讓外部系統同步控制商品可見性，強化自動化營運流程。

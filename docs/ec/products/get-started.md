# :lucide-boxes: 認識商品管理介面

本指南將帶您快速熟悉商品管理介面，並掌握庫存查詢、更新、警示設定與批次操作等核心流程，協助您有效維護商品庫存。

## 認識商品管理介面
商品管理介面是查看庫存、管理商品狀態與執行批次操作的主要入口  
> **預估時間：3–5 分鐘**

=== "介面功能"  
	- 搜尋與篩選商品（分類、通路、庫存狀態）  
	- 查看庫存資訊（可用庫存、已售出、預購庫存）  
	- 進入商品編輯頁面調整屬性  
	- 執行批次庫存更新與批次上下架  

=== "使用技巧"  
	- 搜尋欄搭配篩選器可更快找到目標商品  
	- 建議建立「分類」或「群組」以利大批量管理  

## 查詢庫存
掌握商品現有庫存狀態
> **預估時間：1–2 分鐘**

1. 前往 **商品 > 所有商品**  
2. 使用搜尋或篩選器定位商品  
3. 查看列表內的庫存欄位（可用 / 已售出 / 預購）

## 更新庫存
調整商品可用庫存
> **預估時間：2–3 分鐘**

1. 點擊商品名稱進入商品編輯頁  
2. 調整「可用庫存」數量  
3. 點擊 **儲存** 或 **更新商品** 完成更新  

!!! note
	若商品啟用 **預購**，請確認同步調整預購數量，避免超賣。

## 設定庫存警示
低庫存時自動通知補貨
> **預估時間：2–3 分鐘**

1. 進入 **商品管理介面 > 設定**  
2. 找到「庫存警示」欄位並輸入提醒門檻  
3. 點擊 **儲存設定**  
4. 系統將在庫存低於門檻時自動提醒  

## 批次操作
同時更新多個商品的庫存與屬性  
> **預估時間：3–8 分鐘（依項目而定）**

??? abstract "可執行的批次操作"  
	- **批次修改庫存**（加減庫存或直接覆寫）  
	- **批次上架 / 下架商品**  
	- **批次修改分類、標籤或通路**  
	- **批次啟用 / 關閉預購**

??? abstract "操作步驟"  
	1. 前往 **商品 > 所有商品**  
	2. 使用篩選器選出需調整的商品  
	3. 勾選商品後點擊 **批次操作**  
	4. 選擇操作類型，如「批次修改庫存」  
	5. 輸入調整數量或覆寫值並確認  

## 小技巧

- 使用 **批次修改** 可快速維護大量商品庫存  
- 可利用 **篩選器** 快速找出「低於警示值」的商品  
- 建議將庫存管理與 **訂單狀態、預購設定** 一併檢查以避免超賣

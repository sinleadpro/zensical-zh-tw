# 設定不適用折扣群組
設定特定商品無法參與「全館活動」、「優惠券」、「商品多層級分類滿額折扣」及「VIP 優惠」等行銷活動。
{ .subtitle }

[:lucide-lock:{ title="適用方案" }](../../resources/conventions#適用方案) | PLUS 企業    
[:lucide-layers:{ title="適用產品" }](../../resources/conventions#適用產品) | EC POS


!!! tip "應用情境"

	-   **避免重複優惠**：當商品已有專屬促銷或折扣時，避免其疊加其他全館性優惠，確保利潤空間。
	-   **特殊商品管理**：針對高單價、限量或特殊商品，限制其參與一般行銷活動，維持品牌價值或獨特性。
	-   **精準行銷策略**：根據不同的商品屬性或銷售目標，靈活調整商品參與行銷活動的範圍，優化行銷資源配置。
	-   **POS 獨賣商品**：支援 POS 獨賣商家，確保實體門市的獨家商品不會受到線上優惠的影響。

## 使用須知

### 可排除的行銷活動類型

您可以將商品加入以下四種不適用折扣群組，以排除對應的行銷活動：

**不適用全館活動的商品**   
將商品加入此群組後，該商品將不適用於後台「行銷活動」>「全館折扣 - 紅利&優惠券」>「舉辦全館活動」中設定的全館折扣。

後台路徑： `行銷活動` → `全館折扣 - 紅利&優惠券` → `舉辦全館活動`

![商品排除指定優惠](https://www.cyberbiz.io/support/wp-content/uploads/商品排除指定優惠05.png){ .screenshot }

**不適用優惠券的商品**  
將商品加入此群組後，該商品將不適用於「優惠碼」與「優惠券」活動。

!!! warning "系統自動排除透過特定活動發送的優惠券"
	系統將同時排除透過「首購禮」、「互動遊戲」及「指定商品送優惠券」等活動發送的優惠券。

後台路徑： `行銷活動` → `優惠碼`

![商品排除指定優惠](https://www.cyberbiz.io/support/wp-content/uploads/商品排除指定優惠11.png){ .screenshot }

後台路徑： `行銷活動` → `全館折扣 - 紅利 & 優惠券`

後台路徑： `行銷活動` → `期間限定首購禮`

![商品排除指定優惠](https://www.cyberbiz.io/support/wp-content/uploads/商品排除指定優惠12.png){ .screenshot }

後台路徑： `行銷活動` → `互動遊戲`

![商品排除指定優惠](https://www.cyberbiz.io/support/wp-content/uploads/商品排除指定優惠14.png){ .screenshot }

後台路徑： `行銷活動` → `指定商品送優惠券`

![商品排除指定優惠](https://www.cyberbiz.io/support/wp-content/uploads/商品排除指定優惠15.png){ .screenshot }

**不適用商品多層級分類滿額折扣活動的商品**    
將商品加入此群組後，該商品將不適用於後台「行銷活動」>「商品多層級分類滿額折扣」中設定的滿額折扣。

後台路徑： `行銷活動` → `商品多層級分類滿額折扣`

![商品排除指定優惠](https://www.cyberbiz.io/support/wp-content/uploads/商品排除指定優惠09.png){ .screenshot }

**不適用 VIP 優惠的商品**   
將商品加入此群組後，該商品將不適用於後台「會員」>「VIP 設定」中設定的會員專屬折扣優惠。

後台路徑： `會員` → `VIP 設定`

![商品排除指定優惠](https://www.cyberbiz.io/support/wp-content/uploads/商品排除指定優惠10.png){ .screenshot }

### 排除效果

當商品被設定排除指定優惠後，在前台結帳頁面，系統將會根據設定給予提醒顯示，確保該商品不參與被排除的行銷活動。

![商品群組-不適用折扣群組03](https://www.cyberbiz.io/support/wp-content/uploads/行銷活動-商品排除指定優惠04.png){ .screenshot }

## 操作流程

您可以透過以下步驟，設定商品排除指定優惠：

1.  登入 CYBERBIZ 管理後台，前往 **行銷活動 > 商品排除指定優惠**。

    ![](https://www.cyberbiz.io/support/wp-content/uploads/行銷活動-商品排除指定優惠01.png){ .screenshot }

2.  在頁面左側的商品列表中，點選欲排除的商品，然後點擊 **加入**。
    *   同一個商品可加入多個「不適用折扣群組」。
    *   已加入群組的商品將自動排除，不會重複加入。

    ![](https://www.cyberbiz.io/support/wp-content/uploads/行銷活動-商品排除指定優惠02.png){ .screenshot }

3.  新增商品後，您可以為各商品設定在群組中的排程時間。

    ![](https://www.cyberbiz.io/support/wp-content/uploads/行銷活動-商品排除指定優惠02.png){ .screenshot }


## 常見問題

??? quote "一個商品可以加入多個不適用折扣群組嗎？"
    可以。一個商品可以同時加入多個不適用折扣群組，例如同時排除「全館活動」和「優惠券」。

??? quote "排除指定優惠後，商品還能被顧客購買嗎？"
    可以。此功能僅排除商品參與特定行銷活動，不影響商品在商店中的正常顯示與購買功能。

# 設定加價購群組的多國語系名稱

當您的商店啟用多國語系功能後，您可以為訂單加價購群組設定不同語系的名稱，以提供不同語系顧客更在地化的購物體驗。

[:material-puzzle:](){ .badge-icon title="功能限制" }[多國語系](){ .badge-text }

## 使用情境

當您的商店支援多國語系時，為確保不同語系的顧客在前台能看到正確的加價購群組名稱，您需要針對各語系進行名稱設定。

## 操作流程

### 步驟一：進入訂單加價購群組頁面

1.  登入 CYBERBIZ 電商後台。
2.  前往 **行銷活動** → **訂單加價購**。

    ![](https://www.cyberbiz.co/support/wp-content/uploads/2020/05/多國-訂單加價購01.png)

### 步驟二：編輯多國語系群組名稱

1.  在訂單加價購群組列表頁面，點擊欲編輯的群組。
2.  在群組編輯頁面，點擊語系切換按鈕，選擇您要編輯的語系（例如：英文）。

    ![](https://www.cyberbiz.co/support/wp-content/uploads/2020/05/多國-訂單加價購02.png)

3.  在選定的語系欄位中，填寫該語系的群組名稱。

    ![](https://www.cyberbiz.co/support/wp-content/uploads/2020/05/多國-訂單加價購03.png)

    !!! info "語系標籤欄位"
        只有帶有語系標籤的欄位，前台才會根據語系切換顯示對應的文字。訂單加價購群組中，可編輯多國語系的欄位為「群組名稱」。

    !!! warning "未填寫其他語系內容"
        若您在繁體中文欄位中填寫了內容，但在其他語系欄位中未填寫，則前台切換到該語系時，未填寫的欄位資訊將直接顯示繁體中文的內容。

4.  完成編輯後，點擊 **儲存**，以更新內容至前台。

### 步驟三：前台頁面顯示

=== ":material-flag-variant: 繁體中文商品頁"
    當顧客瀏覽繁體中文商品頁時，訂單加價購將顯示您設定的繁體中文群組名稱。

    ![](https://www.cyberbiz.co/support/wp-content/uploads/2020/05/多國-訂單加價購04.png)

=== ":material-flag-variant: English 商品頁"
    當顧客切換至 English 商品頁時，訂單加價購將顯示您設定的英文群組名稱。

    ![](https://www.cyberbiz.co/support/wp-content/uploads/2020/05/多國-訂單加價購05.png)

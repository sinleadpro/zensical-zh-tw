# 設定電子票券門市權限
設定電子票券的門市權限，與合作品牌共同管理電子票券商品。
{ .subtitle }

[:lucide-lock:{ title="適用方案" }](../../resources/conventions#適用方案) | PLUS 企業

!!! tip "應用情境"
	- **合作品牌銷售**：商家與其他公司（合作品牌）合作銷售電子票券商品。
	- **實體門市核銷**：消費者在商家官網購買電子票券後，可至合作品牌的實體門市進行核銷。
	- **權限精細管理**：在 CYBERBIZ 後台，可為合作品牌的使用者設定專屬權限，確保其僅能操作與其品牌綁定的票券商品。

## 使用須知

- 一個電子票券商品可綁定多個門市，一個門市也可綁定多個電子票券商品。
- 商家可將特定門市設為[管理門市](#設定管理門市)，賦予其查看所有合作門市核銷紀錄的權限。
- 可為合作門市建立專屬的「電子票券門市店員」帳號，權限僅限於票券核銷相關功能。

#### 門市權限
門市權限分為「一般門市」與「管理門市」，主要差異在於查看核銷紀錄的範圍。

![電子票券門市權限功能](https://www.cyberbiz.io/support/wp-content/uploads/%E9%9B%BB%E5%AD%90%E7%A5%A8%E5%88%B8-%E9%96%80%E5%B8%82%E6%AC%8A%E9%99%90%E5%8A%9F%E8%83%BD01.png)

!!! example "範例 1：SPA 電子票券合作"
    商家 A（SPA 公司）與品牌 X、Y、Z 合作推出 SPA 電子票券。

    消費者在商家 A 官網購買票券，至品牌 X、Y、Z 實體門市核銷。
    
    - 品牌 X 設有 3 間實體門市（X-1, X-2, X-3）及 1 個總公司（管理門市）。
    - 商家 A 將「全身芳香按摩 60 分鐘」票券與品牌 X 的 4 個門市綁定。
    - 這 4 間門市皆可查看、核銷此票券。
    - 權限差異：管理門市可查看所有核銷紀錄，而 X-1、X-2、X-3 僅能查看各自門市的核銷紀錄。若票券在 X-1 核銷，X-2 店員無法查看，但管理門市店員可查看。

#### 綁定異動
- **新增綁定**：若電子票券商品綁定新門市，則已售出的電子票券也可在這些新門市核銷。
- **取消綁定**：若電子票券商品與門市取消綁定，針對已售出的電子票券，商家可以選擇是否讓消費者仍可在原門市核銷。

    
!!! example "範例 2：門市暫停營業"
    「全身芳香按摩 60 分鐘」票券與門市 X-1、X-2、X-3 綁定，已售出 100 張。若 X-1 門市暫停營業，商家可取消綁定，並設定讓已售出的票券仍可在 X-1 核銷，避免客訴。
    
    | 票券狀態 | 核銷規則 | 核銷狀態 |
    |---|---|---|
    | 已售出票券 | 可在 X-1 核銷 | ✔ |
    | 之後售出的票券 | 不可在 X-1 核銷 | ✖ |

## 操作流程
### 綁定票券與門市
您可以在以下兩個後台位置進行票券與門市的綁定。

**從票券商品編輯頁**

1. 登入 CYBERBIZ 管理後台，前往 **商品 > 電子票券列表**。
2. 點選欲編輯的票券商品。
3. 切換至 **綁定門市** 分頁，選擇欲綁定的門市。

![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券-門市權限功能02.png){ .screenshot }

**從門市編輯頁**

1. 登入 CYBERBIZ 管理後台，前往 **金物流 > 所有門市**。
2. 點選欲編輯的門市，按下「編輯」按鈕。
3. 切換至「綁定票券」分頁，選擇欲綁定的票券商品。

![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券-門市權限功能03.png){ .screenshot }

### 設定管理門市
您可以將特定門市標記為「管理門市」，賦予其查看所有核銷紀錄的權限。

1. 登入 CYBERBIZ 管理後台，前往 **金物流 > 所有門市**。
2. 點選欲編輯的門市，按下「編輯」按鈕。
3. 切換至「綁定票券」分頁，勾選「電子票券管理門市」選項。

![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券-門市權限功能04.png){ .screenshot }

### 設定門市店員權限
您可以新增專屬的「電子票券門市店員」帳號，使其僅擁有「票券核銷列表」與「核銷票券頁」的權限。

1. 前往後台：**使用者 > 使用者管理**。
2. 點擊 **新增使用者**。
3. 在 **使用者權限** 欄位，選擇 **電子票券門市店員**。
4. 填寫完畢後，點擊 **儲存** 套用變更。

![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定15.png){ .screenshot }

### 取消票券綁定
取消綁定時，系統會詢問是否讓已售出的票券仍可在原門市核銷。

- **選擇「是」**：已售出的票券，仍可在原門市核銷。  
- **選擇「否」**：已售出的票券，無法在原門市核銷。

![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券-門市權限功能05.png){ .screenshot }

### 前台商品畫面
消費者可在前台查看票券商品的核銷門市資訊。

**票券商品頁**：會顯示可核銷的門市列表。

![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券-門市權限功能06.png){ .screenshot }

**會員電子票券訂單查詢**：會員登入後，可在 **我的帳戶 > 電子票券訂單查詢** 中查看可核銷門市。

![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券-門市權限功能07.png){ .screenshot }

## 常見問題

??? quote "門市店員的權限可以修改嗎？"
    「電子票券門市店員」的權限為固定，無法修改。若嘗試修改權限，儲存後會跳出警示訊息「門市用戶的權限無法修改」。

??? quote "門市店員與電子票券門市店員的帳號可以共用嗎？"
    不行，「門市店員」與「電子票券門市店員」的使用者帳號必須分開設定（兩組不同帳號密碼）。

# 設定商品條件分類群組

依商品屬性與條件，自動將符合規則的商品分類至指定群組。
{ .subtitle }

![](../../assets/images/ec-product-conditionalized-collections.zh-tw.png){ title="商品自動分類群組：商品 > 商品條件分類" .hero-page }

## 商品條件分類說明

商品條件分類可自動將符合預設條件的商品加入分類群組，節省手動整理時間並提升管理效率。

應用情境 ：

- **高效管理**：依商品價格、庫存、關鍵字等條件，自動分類商品，提升管理效率。  
- **精準行銷**：針對特定條件分類商品執行行銷活動，例如[單品限時折扣](設定單品限時折扣群組)。  
- **SEO 優化**：設定分類描述與關鍵字，提高搜尋引擎可見度。  
- **多層條件篩選**：支援多層級條件與搜尋，方便管理複雜商品結構。


## 建立商品條件分類

1. 登入 CYBERBIZ 管理後台，前往 **商品 > 商品條件分類**。
2. 點擊 **新增商品分類** 進入編輯頁面。
3. 輸入 **分類名稱**、 **分類網址**，並填寫 **分類描述**。
4. 點選 **儲存** 以建立新的商品條件分類。

![](../../assets/images/ec-conditionalized-collection-settings.zh-tw.png)

## 設定商品篩選規則

設定條件分類規則，自動將符合條件的商品加入群組。

1. 在分類列表中，點選欲設定的 **條件分類**，進入編輯頁面。
2. 切換至 **加入商品** 頁籤，設定商品篩選規則：    
	- **篩選結果**：選擇商品需「符合所有條件」或「符合任一條件」
    - 在篩選規則列表中，點擊 **新增商品規則**，設定商品篩選條件  
3. 點擊 **儲存**，系統將自動將符合條件的商品加入分類，並顯示於下方商品列表中。

![](../../assets/images/ec-conditionalized-collection-set-conditions.zh-tw.png)

### 前台顯示範例

完成設定後，商品條件分類將自動顯示於前台商店頁面。

![](../../assets/images/ec-conditionalized-collections-frontend.zh-tw.png)

### 調整商品排序

控制商品在前台群組頁的顯示順序：

1. 點擊 **商品排序** 下拉選單，選擇自動排序或手動排序。
2. 若選擇 **手動排序**，使用 :lucide-grip-vertical: 拖曳商品調整順序。

![](../../assets/images/ec-conditionalized-collection-order-product.zh-tw.png)

## 編輯 SEO 設定

設定條件分類頁的 SEO 資訊，提高搜尋引擎可見度。

1. 在分類列表中，點選欲設定的 **條件分類**，進入編輯頁面。
2. 切換至 **分類設定** 頁籤，並向下捲動至 **SEO 設定** 區塊。
3. 編輯分類頁的網頁資訊（如標題、描述與關鍵字），完成後儲存設定，以優化搜尋引擎排名。

![](../../assets/images/ec-conditionalized-collection-seo-setting.zh-tw.png)

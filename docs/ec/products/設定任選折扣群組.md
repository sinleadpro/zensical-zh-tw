# 設定任選折扣群組
設定任選折扣群組，為商品組合提供多種優惠方案，提升顧客購物體驗與客單價。
{ .subtitle }

任選折扣群組允許您將特定商品設定為可參與任選優惠活動。顧客選購指定數量的任選商品後，即可享有預設折扣，例如：任選 2 件 8 折、任選 3 件 500 元

## 使用須知
### 功能限制

單一商品僅能參與一個任選折扣活動，即每個商品只能加入一個群組。

!!! example "範例"
    商品 A 已加入「任選 2 件 8 折」群組，無法同時加入其他群組。  
    若要參與「任選 2 件 500 元」活動，請先將商品 A 從原群組移除，再加入新群組。

## 操作流程

1. 登入 CYBERBIZ 管理後台，前往 **行銷活動 > 任選折扣**。
2. 設定任選折扣規則
	在任選折扣設定頁面，您可以根據需求選擇不同的折扣類型並進行詳細設定。請點擊以下連結以查看各折扣類型的詳細教學：
	
	*   [任選折數](https://www.cyberbiz.co/helpcenter/?p=1060)（例如：任選2件8折、任選5件以上6折等）
	*   [任選固定金額](https://www.cyberbiz.co/helpcenter/?p=1065)（例如：任選2件500元、任選3件600元等）
	*   [任選折固定金額 (高手級方案支援)](https://www.cyberbiz.co/helpcenter/?p=1069)（例如：任選2件折50元、任選3件折100元等）
	*   [任選每件折固定金額 (高手級方案支援)](https://www.cyberbiz.co/helpcenter/?p=1072)（例如：任選3件每件折50元、任選5件以上每件折80元等）

3. 完成折扣規則設定後，請務必儲存以套用變更。

## 常見問題

??? quote "任選折扣群組已設定完成，但前台仍顯示 404 錯誤頁面？"

    === ":material-alert-circle-outline: 原因"
        前台顯示 404 錯誤通常表示任選折扣活動或相關商品未正確發布或設定。

    === ":material-lightbulb-on-outline: 解決方法"
        請檢查以下條件是否皆已設定正確：

        1.  **活動是否公開/公佈**：
            確認任選折扣活動已設定為「公開」或「公佈」狀態。

            ![任選折扣群組](https://www.cyberbiz.co/helpcenter/wp-content/uploads/2019/09/任選折扣群組1.png)
        2.  **是否在有效期限內**：
            確認任選折扣活動的起始日期與結束日期設定正確，且目前處於活動期間內。

            ![任選折扣群組](https://www.cyberbiz.co/helpcenter/wp-content/uploads/2019/09/任選折扣群組2.png)
        3.  **是否已設定條件規則**：
            確認任選折扣群組已設定至少一項條件規則（例如：任選件數、折扣方式等）。

            ![任選折扣群組](https://www.cyberbiz.co/helpcenter/wp-content/uploads/2019/09/任選折扣群組3.png)
        4.  **群組內的商品是否皆為公開狀態**：

            *   **商品是否已上架**：確認群組內的商品皆已設定上架時間並已上架。
            *   **商品是否已公開**：確認群組內的商品皆已設定為「公開」狀態。
            *   請前往「商品」→「所有商品」頁面，檢查相關商品的上架時間及公開狀態。

            ![任選折扣群組](https://www.cyberbiz.co/helpcenter/wp-content/uploads/2019/09/任選折扣群組4.png)

        *若您已檢查上述所有設定，但前台仍顯示 404 錯誤，請聯繫客服人員協助查詢。*

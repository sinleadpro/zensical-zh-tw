# 設定商品多層級分類
建立多層級商品分類（最多三層），整理群組、支援行銷活動與導覽列設定。
{ .subtitle }

[:lucide-lock:{ title="適用方案" }](../../resources/conventions#適用方案) | PLUS 企業  [:lucide-toggle-right:{ title="適用功能" }](../../resources/conventions#適用功能) |  拖拉版型

## 什麼是商品多層級分類？

商品多層級分類能協助商家建立清晰、可調整的商品分類架構。透過整理既有群組，網站將呈現 **樹狀結構**，方便會員瀏覽，也利於商家設定導覽列與行銷活動。

> :lucide-flame: 若需快速建立簡單導覽列，可直接在[導覽列綁定商品群組](設定網站選單與導覽列)，省略多層級分類設定。

## 使用須知

- 商品多層級分類功能支援兩種分類層級架構，商家可依需求建立[雙層結構](#雙層結構)或 [三層結構](#三層結構)。
- 設定多層級分類不會改變群組內的商品名單。
	
!!! tip "應用情境"

	- 群組資料夾：將分散的群組集中管理，形成更直觀的分類邏輯。  
	- 清晰導覽：提供會員更易理解的商店導覽階層。  
	- 行銷規劃：在不同分類層級上綁定行銷活動，提高操作彈性。

## 操作流程
### 雙層結構
適用於商家產品種類明確，只需建立一層資料夾就可整理全館商品。

1. 登入 CYBERBIZ 管理後台，前往 **商品 > 商品多層級分類**。
2. 點選 **新增類別**，於新增分類頁面輸入 **類別標題**、**類別連結**。
	
	![](https://www.cyberbiz.io/support/wp-content/uploads/商品多層級分類01.png){ .screenshot }

3. 在大類別下，點選 **新增**。
	- 新增類別選項：選擇 **小類別**，輸入 **類別標題**、**類別連結**。
	- 群組類型可選擇「商品自訂分類」、「商品條件分類」、「商品類型」，點選後往下選擇指定群組。
    > 小類別即為分類群組。分類設定請參考 [設定自訂分類群組](建立商品自訂分類群組.md)、[設定條件分類群組](設定條件分類群組)、[商品類型](#)。

	![](https://www.cyberbiz.io/support/wp-content/uploads/商品多層級分類06.png){ .screenshot }

4. 設定完成畫面

	![](https://www.cyberbiz.io/support/wp-content/uploads/商品多層級分類11.png){ .screenshot }
	
	![](https://www.cyberbiz.io/support/wp-content/uploads/商品多層級分類12.png){ .screenshot }

### 三層結構
適用於商家產品複雜多樣，需要在一層資料夾內再創建資料夾進行細分。

- 底層（小類別）：來源可包含自訂分類、條件分類，或商品設定頁中的商品類型。  
- 中類別：可建立於大類別之下。  
- 大類別：最上層分類。  

!!! note "設定多層級分類不會改變群組內的商品名單。"

1. 登入 CYBERBIZ 管理後台，前往 **商品 > 商品多層級分類**。
2. 點選 **新增類別**，於新增分類頁面輸入 **類別標題**、**類別連結**。
	
	![](https://www.cyberbiz.io/support/wp-content/uploads/商品多層級分類01.png){ .screenshot }

3. 在大類別下，點選 **新增**。
	- 「新增類別選項」：選擇 **類別**，輸入「類別標題」、「類別連結」。
	
	![](https://www.cyberbiz.io/support/wp-content/uploads/商品多層級分類02.png){ .screenshot }

4. 在中類別下，點選「新增」。
	- 新增類別選項：選擇 **小類別**，輸入 **類別標題**、**類別連結**。
	- 群組類型可選擇「商品自訂分類」、「商品條件分類」、「商品類型」，點選後往下選擇指定群組。
	> 小類別即為分類群組。分類設定請參考 [設定自訂分類群組](建立商品自訂分類群組.md)、[設定條件分類群組](設定條件分類群組)、[商品類型](#)。
	
	![](https://www.cyberbiz.io/support/wp-content/uploads/商品多層級分類09.png){ .screenshot }

5. 設定完成畫面

	![](https://www.cyberbiz.io/support/wp-content/uploads/商品多層級分類10.png){ .screenshot }
	
	![](https://www.cyberbiz.io/support/wp-content/uploads/商品多層級分類13.jpeg){ .screenshot }

!!! info "分類移動說明"
    想讓中分類移動到其他大分類下，可藉由拖拉的方式，移動到其他分類下。  
    > 注意：小分類無法移動到其他中分類下，請在中分類下直接新增小分類即可。
    
    ![](https://www.cyberbiz.io/support/wp-content/uploads/商品多層級分類08.png){ .screenshot }

## 常見問題

??? quote "商品多層級分類會影響商品名單嗎？"
    不會。商品多層級分類僅用於整理分類群組，不會影響群組內的商品名單。

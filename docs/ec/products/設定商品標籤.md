# 設定商品標籤
為商品建立與管理自訂標籤，以便分類、篩選、行銷應用及控制第三方平台同步。
{ .subtitle } 

![](../../assets/images/ec-product-tag-management.zh-tw.png){  title="管理商品標籤：商品 > 商品標籤管理"  .hero-page }

!!! tip "應用情境"
	- 商品分類管理：為不同類型的商品設定標籤，方便快速篩選與管理。
	- 行銷活動應用：將特定商品批次加入標籤群組，以便進行促銷活動或組合銷售。
	- 提升搜尋效率：透過標籤快速定位商品，優化後台操作效率。

## 操作流程
### 新增商品標籤

1. 登入 CYBERBIZ 管理後台，前往 **商品 > 商品標籤管理**。
2. 點擊 **新增標籤** 按鈕，輸入標籤名稱，例如：「衣服」。
3. 點擊 **儲存** 以套用變更。

![標籤群組位置](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組位置.png){ .screenshot }

### 將商品加入標籤群組

您可以透過兩種方式將商品加入標籤群組：

**手動加入**

 1. 在 **商品標籤管理** 頁面，找到您要編輯的標籤群組，點擊 **新增商品**。
2. 搜尋並勾選欲加入的商品，例如：「襯衫」。
3. 點擊 **加入** 完成。

![標籤群組加入](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組加入.png){ .screenshot }

**透過 Excel 匯入**

1. 匯出商品 Excel 表格，並在 **商品標籤** 欄位中填入欲加入的標籤名稱，例如：「衣服」。
2. 登入 CYBERBIZ 管理後台，前往 **商品 > Excel 大量匯入商品**。
3. 上傳已編輯完成的 Excel 檔案。

!!! warning "Excel 匯入商品限制"
	-   透過 Excel 匯入商品時，無法新增商品到不存在的標籤中。
	-   透過 Excel 匯入已在群組內的商品，系統將會略過，不會提醒。

### 從標籤群組移出商品

1. 在 **商品標籤管理** 頁面，找到您要編輯的標籤群組。
2. 在商品列表中，點擊欲移出商品旁的 **移出** 按鈕或 :lucide-x: 圖示。

![標籤群組移出](https://www.cyberbiz.co/support/wp-content/uploads/2019/08/標籤群組移出.png){ .screenshot }

### 刪除商品標籤

1. 在 **商品標籤管理** 頁面，找到欲刪除的標籤群組。
2. 點擊標籤群組旁的 **刪除** 按鈕。

!!! warning "刪除標籤後，所有已綁定此標籤的商品將會自動取消綁定。"


### 排除上傳至第三方平台標籤
[:lucide-lock:{ title="適用方案" }](../../resources/conventions#適用方案) | 企業

為商品設定特定標籤，將商品排除於系統的第三方平台資料同步機制之外；套用排除標籤後，該商品的資料將不會被上傳至下列第三方平台：

- Google Merchant Center
- Facebook 動態產品目錄 DPA
- LINE 購物
- 美安 (僅支援 `贈品` 標籤)

---

1. 登入 CYBERBIZ 管理後台，前往 **商品 > 所有商品**。
2. 在商品列表中，點擊欲排除商品的 **商品名稱**，進入商品編輯頁面。
3. 在商品標籤欄位中，輸入以下任一排除標籤：
	 - `排除product feed`：適用於 GMC、Facebook 動態產品目錄 DPA、LINE 購物平台。
	 > :lucide-triangle-alert: *排除* 與 *product* 之間請勿添加空格。
	 - `贈品`：適用於所有第三方平台，包括美安。

4. 點擊 **儲存** 以套用設定。

## 常見問題

??? quote "如果我同時設定了 `排除product feed` 和 `贈品` 標籤，會有什麼影響？"
    若同時設定兩個標籤，系統將優先依據 `贈品` 標籤的規則進行排除，確保商品不會上傳至所有支援的第三方平台。

??? quote "排除商品後，顧客還能在我的 CYBERBIZ 商店中找到並購買嗎？"
    可以。此排除設定僅影響商品是否上傳至第三方平台，不影響商品在您 CYBERBIZ 商店中的可見性與購買功能。顧客仍可透過商店內搜尋或直接連結找到並購買商品。

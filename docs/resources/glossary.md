## 商品

`商品資訊`
:   商品在系統中的所有設定屬性與內容，用於前台展示、銷售與行銷應用。

`商品名稱`
:   商品建立時的必要欄位，用於識別並展示商品。

`商品標語`
:   顯示於商品頁面的簡短文字，用於傳達商品賣點或活動訊息。

`商品簡述`
:   商品頁面的說明文字，用於快速介紹商品重點與特色。

`商品頁面`
:   顧客於前台瀏覽單一商品詳細資訊的頁面。

`商品網址`
:   商品頁面的專屬網址，用於前台存取與搜尋引擎索引。

`商品連結`
:   系統自動產生、可直接導向商品前台頁面的網址。

`圖床`
:   用於儲存與提供圖片連結的外部服務。


## 商品結構與規格

`組合商品`
:   由兩個或以上子商品組成，並以單一商品形式銷售。

`子商品`
:   構成組合商品的單一商品項目。

`指定組合商品`
:   組合內容固定、不可更換的組合商品。

`任選組合商品`
:   顧客可自由選擇子商品的組合商品。

`任選組合總數`
:   任選組合商品中需選購的子商品總數。

`組合品價差`
:   組合商品售價與子商品原價總和的差額。

`規格`
:   用於區分商品屬性的分類標籤。

`規格項目`
:   特定規格下的可選屬性值。

`款式`
:   由多個規格項目組合而成的實際販售單位。

`SKU`
:   用於識別與管理商品庫存的唯一代碼。


## 訂單、付款與物流

`信用卡一次付清`
:   信用卡交易金額於單一帳單週期內全額請款。

`3DS`
:   線上信用卡交易的身分驗證機制。

`OTP`
:   一次性使用的驗證密碼。

`COD`
:   貨到付款，於收貨時完成付款。

`配送屬性`
:   決定商品出貨與運費規則的屬性組合。

`WMS`
:   倉儲管理系統，用於管理倉儲作業。

`串倉`
:   電商平台與倉儲管理系統的整合模式。

`POS`
:   門市銷售作業系統。


## 行銷與成長

`優惠券`
:   指派給特定會員帳戶使用的優惠憑證。

`優惠碼`
:   結帳時輸入即可套用的折扣序號。

`EDM`
:   透過電子郵件發送行銷或促銷資訊。

`SEO`
:   搜尋引擎最佳化，用於提升網站可見度。

`GA`
:   Google Analytics，網站流量分析工具。

`GMC`
:   Google Merchant Center，商品資料管理平台。

`MBE`
:   Meta Business Extension。

`分潤`
:   商品或服務銷售後的收益分配機制。


## 平台、方案與其他

`一般版`
:   專業、高手、進階方案版本。

`Regular Plans`
:   Boss、Advance、Master 方案。

`PLUS Plans`
:   Boss PLUS、Advance PLUS、Master PLUS 方案。

`Enterprise Plan`
:   企業級訂閱方案。

`EC`
:   電子商務，透過網路進行交易的商業模式。

`B2C`
:   企業對消費者的銷售模式。

`C2C`
:   消費者對消費者的交易模式。

`分票`
:   將單一 QR Code 拆分為多個可獨立使用的票券。

`註解程式碼`
:   使被註解的程式碼不會被執行，用於測試或暫時停用功能。

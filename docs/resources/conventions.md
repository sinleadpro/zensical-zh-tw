# 圖示慣例

本節說明本文件中使用的幾個慣例。

## 符號
本文件使用一些符號以供說明之用。在您繼續閱讀之前，請確保您已熟悉以下慣例列表：

### :lucide-: 適用產品
表示內容適用 CYBERBIZ 相關的產品線。如 EC、POS、WM 等。 

### :lucide-tag: 適用方案
表示內容僅適用特定方案用戶。  

### :lucide-bolt: 適用功能
表示內容僅適用開啟某些功能。  

### :lucide-grid-2x2-plus: 適用擴充
表示內容適用特定的 CYBERBIZ 擴充功能。

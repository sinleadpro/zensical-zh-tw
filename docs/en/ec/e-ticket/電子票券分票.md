# 電子票券分票

分票功能可將同一票種的多張票券拆分為多個獨立核銷碼。

## 使用條件

- 僅適用於購買多張相同票種的情境
- 分票後每張票券可獨立核銷

## 注意事項

- 分票完成後無法復原
- 原 QR code 將失效

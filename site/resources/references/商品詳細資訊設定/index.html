商品詳細資訊設定¶

商品資訊的完整相關欄位說明，包含「基本資訊、款式設定、物流、SEO、圖片與規格」等，並可搭配以下操作文件：

  • [[新增單一商品]]
  • [[新增大量商品]]
  • [[編輯商品資訊與設定]]

Excel 匯入檔案欄位¶

此為通用版說明，實際欄位請以您後台下載的 Excel 範本為主。

欄位 說明 備註 / 限制
商品名稱 設定商品名稱 必填。請勿使用特殊符號或 HTML。
商品隸屬商店 指定商品所屬商店 必填。若未啟用 POS 功能請忽略此欄。
商品簡述類型 設定商品簡述類型 必填。填寫 html樣板文字 或 純文字。
商品款式售價 設定商品實際售價 必填。
欄位 說明 備註 / 限制
C. 商品名稱 設定商品名稱 必填。請勿使用特殊符號（如 | 或 ”）或 HTML，以免儲存失敗。
D. 商品隸屬商店 指定商品所屬的官網或 POS 商店名稱 建立商品時可指定多個商店，請用逗號 , 隔開；更新商品時僅能指定一家商店。未啟用 POS 功能請忽略此欄，啟用後為**必填**。
E. 商品上架時間 設定商品上架的日期與時間 選填。請注意時間格式。
F. 商品下架時間 設定商品下架的日期與時間 選填。請注意時間格式。
G. 商品是否公開 設定商品是否在前台顯示 選填。填寫 是 或 否。
H. 商品標語 顯示於商品頁的簡短文字 選填。可參考 [[客製化商品頁面標語與簡述文字]] 文件。
I. 商品簡述類型 設定商品簡述的內容類型 必填。填寫 html樣板文字 或 純文字。
J. 商品簡述 簡短說明商品重點賣點 選填。依照「商品簡述類型」輸入內容。
K. 商品網址 設定商品頁面 URL 選填。格式為 products/ 之後的字串。若留空，預設為商品名稱。
L. 商品稅別 設定商品稅務類型 選填。填寫 應稅 或 免稅。
欄位 說明 備註 / 限制
W. 商品款式選項/內容 設定商品款式（如顏色、尺寸） 選填。
AC. 商品款式售價 商品實際銷售金額 必填。
AD. 商品款式定價 商品建議售價 選填。
AE. 商品款式成本 商品成本價 選填。不對客戶顯示。
AF. 商品款式紅利最高折抵 設定商品紅利最高折抵金額 選填。
AG. 商品款式重量 商品重量 選填。填寫公斤 (kg)。
AH. 商品款式 SKU 碼 商品編號 (Stock Keeping Unit) EC 選填， POS 必填。
AI. 商品款式收貨地址是否需要填寫 設定購買此款式是否需填寫收貨地址 選填。填寫 是 或 否。
AJ. 商品款式管理庫存 設定是否啟用庫存管理 選填。使用 POS 功能請填 是。
AK. 商品款式庫存量 商品可販售數量 選填。留空表示無限。
AL. 商品款式安全庫存水位 設定庫存低於此數量時的通知水位 選填。
AN. 商品款式庫存不足時 設定庫存不足時的銷售行為 選填。填寫 繼續銷售 或 停止銷售。
AO. 商品款式排序 設定商品款式顯示順序 選填。
欄位 說明 備註 / 範例
AI. 商品款式收貨地址是否需要填寫 設定購買此款式是否需填寫收貨地址 選填。一般網購商品建議填寫，可選擇 不需填寫。
AP. 商品款式材積 商品包裹材積 選填。包裹長 + 寬 + 高 ≤ 上限值：
- 材積 >105 cm → 僅可宅配
- 材積 >150 cm → 可加印託運單分第二箱
- 詳細進階設定請參考 超商排除材積
AG. 商品款式重量 商品重量 選填。
- 超商物流 ≤ 5 kg
- 宅配物流 ≤ 20 kg
- 海外物流依重量計價
欄位 說明 備註 / 限制
AQ. SEO 設定網頁標題 設定商品頁面的 SEO 標題 選填。請參考 SEO 優化教學 文件。
AR. SEO 設定網頁描述 設定商品頁面的 SEO 描述 選填。請參考 SEO 優化教學 文件。
AS. SEO 網頁設定關鍵字 設定商品頁面的 SEO 關鍵字 選填。請參考 SEO 優化教學 文件。
欄位 說明 備註 / 限制
AT. 商品圖片 商品圖片連結 選填。圖片尺寸需小於 10MB，支援 jpg jpeg png gif webp 格式。此欄位為商品主圖。
BC. 規格說明 商品規格說明 選填。
BD. 運送方式 商品運送方式 選填。

物流與材積¶

欄位 說明 備註 / 範例
商品款式收貨地址是否需要填寫 是否需填寫收貨地址 選填，一般建議填寫，可選擇 不需填寫。
商品款式材積 商品包裹材積 選填，包裹長 + 寬 + 高 ≤ 上限值：
材積 >105 cm → 僅可宅配
材積 >150 cm → 可加印託運單分第二箱
詳見 超商排除材積
商品款式重量 商品重量 選填，超商 ≤5 kg，宅配 ≤20 kg，海外依重量計價。
運送溫層設定 商品可接受的運送溫層 選填，詳情參考 [[商品上架管理：配送物流/溫層綁定]]。
物流設定 商品可用物流方式 選填，詳情參考 [[新版物流運費設定]]。

圖片與規格¶

欄位 說明 備註 / 限制
商品圖片 商品主圖或輔圖 選填，支援 jpeg png webp，小於 10MB。
規格說明 商品規格說明 選填。
運送方式 商品運送方式描述 選填。

進階設定¶

欄位 功能說明 備註 / 限制
商品類型 為商品添加類型標籤，便於管理篩選 新增後不可刪除
商品通路 設定商品銷售通路 (現貨/預購/常溫/冷凍) 新增後不可刪除，詳見 [[批次修改商品溫層/配送方式/通路]]
商品廠商 添加廠商標籤 新增後不可刪除
自訂群組 將商品加入已建立的群組 詳見 [[商品群組 ：自訂群組]]
任選折扣群組 將商品加入折扣群組 詳見 [[任選折扣 一般]]
標籤 行銷標籤，指定商品優惠券 詳見 [[商品標籤應用]]
Google 產品類別 選擇或覆蓋 Google 自動分類 詳見 [[Google 購物廣告設定 (GMC)]]

商品管理¶

輕鬆管理商品資訊、分類與庫存，提升上架效率。



  • 快速上架商品、分類與管理庫存，輕鬆掌控商店運營。

    [[新增單一商品|上架第一個商品 ]]
    [[快入上手|快速上手 ]]

  • ![[product-hero.png]]{ .screenshot-hero-page }

立即開始¶

  • [[認識商品管理介面]]
    在所有商品頁面管理和操作商品。

  • [[新增單一商品]]
    建立及上架商品、更新商品資訊。
    [[新增組合商品|新增組合商品]]

  • [[新增大量商品]]
    使用 Excel 一次新增大量商品。
    [[從蝦皮匯入|從蝦皮匯入商品]]

  • [[設定多購物車#設定預購商品|新增預購商品]]
    建立預購商品通路，套用至既有商品。

  • [[設定電子票券#新增電子票券|新增電子票券]]
    建立電子票券商品，供線上購買、門市或指定通路兌換使用。

  • [[後台搜尋商品|定位商品進行管理]]
    使用關鍵字搜尋、條件篩選或組合多重邏輯檢索商品。
  • [[批次修改商品資訊|更新大量商品]]
    使用 Excel 一次更新大量商品跟資訊。

  • [[設定商品排除搜尋|關閉商品搜尋]]
    將商品從網站與 Google 搜尋結果中排除，避免非目標顧客看到優惠價。

  • [[設定商品排除上傳至第三方平台|排除商品上傳至第三方平台]]
    設定排除標籤，讓商品不上傳至第三方平台。

  • [[設定商品秘密群組|建立專屬秘密商店]]
    為特定顧客群體建立專屬秘密商品群組

設定管理¶

  • [[設定商品配送屬性（一般宅配）]]
    綁定一般宅配商品的配送物流、溫層及出貨通路

  • [[設定商品配送屬性（宅配貨到付款）]]
    綁定宅配貨到付款商品的配送物流、溫層及出貨通路

  • [[設定商品標語與商品簡述文字]]
    設定商品的標語和簡述，吸引顧客目光。

  • [[設定商品影片|商品影片]]
    上傳商品介紹影片，豐富商品頁面內容。

  • [[設定商品超商物流限制與排除選項]]
    管理超商物流限制與排除規則。

  • [[自訂商品分類群組]]
    依需求建立商品分類群組

  • [[設定商品多層級分類]]
    建立多層級商品結構，套用至導覽列與行銷活動

  • [[設定條件分類群組|條件分類群組]]
    依商品條件建立動態分類

  • [[設定商品標籤]]
    為商品建立與管理自訂標籤，以便分類、篩選、行銷應用及控制第三方平台同步。

  • [[設定前台商品群組排序]]
    調整前台商品群組的顯示順序。

  • [[設定前台商品篩選器|前台商品篩選器]]
    讓顧客能自行設定篩選條件找到目標商品。

  • [[新增電子票券]]
    建立電子票券商品，供線上購買與兌換。

  • [[購買電子票券]]
    引導顧客完成電子票券的線上購買流程。

  • [[電子票券分票]]
    將單一電子票券分發給多位使用者。

  • [[核銷電子票券]]
    處理電子票券的兌換與核銷程序。

  • [[票券分潤自動結案]]
    設定電子票券訂單自動結案並計算分潤

  • [[電子票券退款]]
    為顧客辦理電子票券的全額或部分退款作業

  • [[設定電子票券門市權限|電子票券門市權限]]
    設定門市人員核銷電子票券的操作權限。

  • [[設定電子票券任選折扣|電子票券任選折扣]]
    設定電子票券的任選優惠折扣活動。

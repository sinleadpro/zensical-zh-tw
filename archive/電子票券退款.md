# 電子票券退款
:lucide-tag:{ title="適用方案" } | PLUS 企業

為顧客辦理電子票券的全額或部分退款作業

## 使用須知

- 電子票券退款需以一個電子票券代碼為單位進行，且一張訂單只能進行一次退票。
> 假設一個電子票券代碼中，內含十個可核銷之電子票券數量，現已兌換三個，另有七個尚未兌換。則必須將尚未兌換的七個數量一次全部退票，不得分割為特定數量退票。
- 票券若退款，票券手續費會[[電子票券對帳方法#對帳規則|按比例]]退回。

## 操作流程
### 顧客通知退票
若顧客欲退票，請引導其前往 **電子票券訂單查詢** 的 **詢問紀錄** 通知商家退票。

![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定16.png){ .screenshot }

### 商家確認退貨訊息
商家可在訂單列表頁收到消費者的退貨訊息。

![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券設定17.png){ .screenshot }

### 確認退款情境
退款前需確認訂單內的票券種類與核銷狀態，[單一種類票券](#訂單內僅含一種票券)與[多種票券種類](#訂單內含多種票券)有不同的退貨流程。

### 執行退款
#### 訂單內僅含一種票券
=== "票券尚未核銷"
	若此票券尚未核銷，可直接走取消訂單流程。
	
	1. 在訂單列表中，找到並點擊欲取消的訂單，進入訂單明細頁面。
	2. 點擊 **取消訂單**，確認相關資訊無誤，點擊 **取消此訂單**。
	
		<div class="grid cards" markdown>
		
		- ![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定20.png){ title="取消訂單" }
		- ![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定21.png){ title="取消此訂單" }
		
		</div>
	
	1. 確認完成畫面
	
	![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定22.png){ .screenshot }

=== "票券已核銷"
	若此票券已核銷過，須走部分退款流程。（消費者當日付款，隔日才能操作部分退款）
	
	1. 在 CYBERBIZ 後台，前往 **訂單 > 所有訂單**，在訂單列表中勾選該訂單，點擊 **更多操作**，選擇 **退貨中**。
	
		![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定23.png){ .screenshot }
	
	2. 勾選該訂單，勾選該訂單，點擊 **更多操作**，選擇 **退貨審查**。
	
		![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定24.png){ .screenshot }
	
	1. 進入訂單頁，在部分退款選取票券商品，點擊 **確認退款**。
	
		![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定25.png){ .screenshot }
	
	2. 再次確認退款資訊，點擊 **確認退款**。
	
		![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定26.png){ .screenshot }
	    
#### 訂單內含多種票券
=== "票券皆尚未核銷"
	若訂單內的票券皆尚未核銷，且消費者想要全部退款，則可以走[取消訂單流程](#票券尚未核銷)。

=== "部分票券已核銷"
	若訂單內已有核銷過的票券，或者消費者想要退某幾種特定票券，則須走部分退款流程。
	
	1. 在 CYBERBIZ 後台，前往 **訂單 > 所有訂單**，在訂單列表中勾選該訂單，點擊 **退貨中**。
	
		![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定27.png){ .screenshot }
	
	2. 勾選此訂單，點擊 **退貨審查**。
	
		![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定28.png){ .screenshot }
	
	1. 進入訂單頁，在部分退款選取欲退票的票券商品，點擊 **確認退款**。
    > 若消費者只想退「Cyber 音樂祭 - 單日票」，則只需勾選該項目即可。
	
		![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定29.png){ .screenshot }
	
	2. 點擊 **確認**，完成退款。
    > 「Cyber 音樂祭 - 單日票」剩餘3張全數退款，QR Code 作廢。「皮卡丘體驗券 - 早鳥體驗」沒有退款，則可繼續照常使用。
	
	![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定30.png){ .screenshot }

## 常見問題

??? quote "一個電子票券代碼包含多個數量，若已核銷部分數量，剩餘的可以退款嗎？"
    可以。剩餘未核銷的數量必須一次全部退票，無法分割特定數量退票。

??? quote "核銷過的票券可以退款嗎？"
    不行。票券一經核銷便無法退款。

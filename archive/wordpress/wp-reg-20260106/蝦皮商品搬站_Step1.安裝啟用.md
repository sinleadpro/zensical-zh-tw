**看了此文件，您可以：**  

* **啟用 CYBERBIZ「全通路管理助手」** ，快速導入蝦皮商品至官網。
* **查閱與管理蝦皮商店授權狀態與到期日** ，並依提醒重新授權，避免搬站功能中斷。
* **支援多個蝦皮帳號搬站作業** ，透過授權切換進行多商店商品導入。

**操作目錄：**

* 功能說明
* 適用條件
* 啟用全通路管理助手
* 授權蝦皮帳號
* 多個蝦皮帳號如何搬站
* 管理蝦皮商店授權

注意事項:  

* **如需使用此功能，請洽 CYBERBIZ 客服，會有專員為您服務。**
* 當您授權蝦皮帳號時，系統僅透過官方 API 取得您的串聯許可，我們不會取得您的蝦皮帳密，也無法登入您的蝦皮後台，敬請放心。

## 📌 功能說明


CYBERBIZ 推出「全通路管理助手(CYBERBIZ CHANNEL BRIDGE)」，協助商家將蝦皮商品快速搬移至 CYBERBIZ
官網，完成商品建立與同步管理。  

當您已在蝦皮建立商品，但尚未於 CYBERBIZ 官網建立時，你可以：  
✅ 透過「全通路管理助手」批次匯入蝦皮商品。  
✅ **依需求挑選建立為官網商品。**  
✅ 可選擇**將蝦皮商品類別，轉換為官網的自訂分類。**  

⭕系統會**自動將蝦皮欄位對應填入官網欄位** （如品名、價格、圖片、庫存等）。  
⭕系統可依商家需求，**比對蝦皮商品的分類欄位，並依對應結果新增官網自訂分類，或套用至既有分類** 。  
⭕系統會自動將蝦皮商品關聯至官網商品，以方便商家檢視與管理。

## 📌 啟用全通路管理助手



1. 透過 CYBERBIZ 專員取得啟用入口連結，輸入您的官網前台網址，點擊「前往」。   
![蝦皮搬站](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-APPMARKET-蝦皮商品-授權商店01.png)

2. 勾選授權選項並確認安裝。   
![蝦皮搬站](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-APPMARKET-蝦皮商品-授權商店02.png)

3. 確認已安裝「CYBERBIZ CHANNEL BRIDGE」。   
![蝦皮搬站](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-APPMARKET-蝦皮商品-授權商店03.png)

4. 點擊「設定」可繼續蝦皮商店授權設定。   
![蝦皮搬站](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-APPMARKET-蝦皮商品-授權商店04.png)

* * *

## 📌 授權蝦皮帳號



1. 確認已安裝「CYBERBIZ CHANNEL BRIDGE」，點擊「設定」。  
![蝦皮搬站](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-APPMARKET-蝦皮商品-授權商店04.png)

2. 點選「申請授權」。  
![蝦皮搬站](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-APPMARKET-蝦皮商品-授權商店05.png)

3. 於蝦皮授權頁面中，帳號別選擇「TW」，登入您的蝦皮帳號。 
* **僅支援台灣帳號。**
![蝦皮搬站](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-APPMARKET-蝦皮商品-授權商店06.png)

4. 點擊「Confirm Authorization」。   
![蝦皮搬站](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-APPMARKET-蝦皮商品-授權商店07.png)

* 建議選擇授權期限為「365 days」。
![蝦皮搬站](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-APPMARKET-蝦皮商品-授權商店08.png)

5. 授權成功後，系統會顯示「商店狀態」為「已連結」。可進行下一步 [蝦皮商品搬站_Step2.導入商品](https://www.cyberbiz.io/helpcenter/?p=5923)。  
![蝦皮搬站](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-APPMARKET-蝦皮商品-授權商店09.png)

* * *

## 📌 多個蝦皮帳號如何搬站



1. 點選「取消授權」。  
![蝦皮搬站](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-APPMARKET-蝦皮商品-多蝦皮帳號搬站01.png)

2. 重新授權新的蝦皮帳號。授權步驟與 授權蝦皮帳號 相同。
3. 即可進行下一步 [蝦皮商品搬站_Step2.導入商品](https://www.cyberbiz.io/helpcenter/?p=5923)。

* * *

## 📌 管理蝦皮商店授權



* 可於「商店設定」頁籤，查看「商店狀態」與「到期日期」。
* 系統將於蝦皮商店授權到期日前30日、前15日、前7日、前3日、前1日寄送授權到期通知。
* 點選「重新授權」，方可繼續使用匯入商品功能。

![蝦皮搬站](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-APPMARKET-蝦皮商品-管理蝦皮商店授權01.png)

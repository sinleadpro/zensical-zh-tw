## 📌 如何製作購物車連結


您可生成含指定商品的購物車連結，只要指定商品款式 ID 與數量，即可快速建立「導入購物車」的分享連結，可應用於行銷導購、分眾導流等情境。  

您可透過網址參數 `products=[款式ID]:[數量]` 的形式，直接建立預設商品與數量的購物車連結。

* 網址格式：  
`_[您的網址]_ /cart/replace?products=_[款式ID]_ :_[數量]_ ,_[款式ID]_ :_[數量]_`



* 底線處請填入您的資訊： 
* 輸入您的官網網址。
* 輸入您想新增至購物車連結的商品款式ID，並輸入數量。
* 每項商品以`[款式ID]:[數量]`格式表示。如需建立多樣商品，請以逗號`,`或分號`;`隔開。
* 如需為同一商品的多個款式設定不同數量，請分別填寫每一款式的商品款式 ID 與對應數量。


* 範例網址：  
`_https://www.goodthing.com_ /cart/replace?products=_001234_ :_1_ ,_001235_
:_1_`  
打開連結後，購物車將自動載入款式 ID 為`001234`、`001235`兩項商品，每項數量為 1。

## 📌 如何取得商品款式ID


後台路徑 :  「商品」→「所有商品」  


1. 勾選欲導入的商品，點擊右上角「匯出商品」。  
![生成含指定商品的購物車連結](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-商品-所有商品-匯出商品01.png)  

2. 開啟下載的 Excel 檔，展開隱藏的B 欄：**[2] 商品款式id** 欄。  
![生成含指定商品的購物車連結](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-商品-所有商品-取得商品款式ID01.png)  

3. 欄位中顯示的數字即為商品款式 ID，即可填入網址中。

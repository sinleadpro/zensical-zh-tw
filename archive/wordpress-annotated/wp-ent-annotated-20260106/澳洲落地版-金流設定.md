[[圖片說明：澳洲站]](https://www.cyberbiz.io/support/wp-content/uploads/澳洲站.png) **看了此文件，您可以：**

- **完成 Airwallex 帳戶申請，** 了解帳戶方案與開戶文件需求。
- **激活 Airwallex 收單功能，** 依指引完成官網評估與功能開啟。
- **將 Airwallex 串接至 CYBERBIZ 官網，** 設定 API 密鑰與啟用金流選項。
**操作目錄：**

- 申請 Airwallex 帳戶
- 激活 Airwallex 帳戶功能
- 激活 Airwallex 收單功能
- 串接 CYBERBIZ 官網
注意事項:

- 有關方案費用、撥款機制及金流手續費等詳細資訊，請參閱 Airwallex 官方網站。如有任何疑問，請聯繫 Airwallex 客服進一步諮詢。
📌 申請 Airwallex 帳戶

1. 請聯繫 CYBERBIZ 開店顧問團隊，取得 Airwallex 帳戶申請連結。

2. 線上申請注意事項
- 請依貴司設立地點，選擇國家/地區。
[[圖片說明：文件名]](https://www.cyberbiz.io/support/wp-content/uploads/Airwallex-前台-會員註冊頁-建立會員01.png)

- 帳戶方案請選擇「Explore」。
- 帳戶月費依貴司設立地點而定，詳細資訊可參考 [Airwallex官方定價說明](https://www.airwallex.com/au/pricing)。
[[圖片說明：文件名]](https://www.cyberbiz.io/support/wp-content/uploads/Airwallex-前台-會員註冊頁-選擇公司所在地01.png)

- 申請帳戶時需同步提交申請文件，申請文件依貴司設立地點而異，詳細資訊可參考Airwallex官網 [各國家/地區開戶所需文件](https://help.airwallex.com/hc/zh-cn/sections/8352632554767)。

3. 申請完成後即可開通後台，請接著激活 Airwallex 帳戶。
📌 激活 Airwallex 帳戶功能

1. 開通 Airwallex 後台後，請於後台主頁激活帳戶。
[[圖片說明：澳洲到台-金流設定]](https://www.cyberbiz.io/support/wp-content/uploads/Airwallex-前台-首頁-激活帳戶01.png)

2. 請依後台指示激活帳戶，並驗證 Email。
[[圖片說明：澳洲到台-金流設定]](https://www.cyberbiz.io/support/wp-content/uploads/Airwallex-前台-首頁-激活帳戶02.png)

📌 激活 Airwallex 收單功能

1. 請提供您的官網連結給 Airwallex 窗口，並告知您的 Airwallex 後台已開通。

提供官網連結前，請先建立網站初步結構，並清楚建立以下資訊，以供 Airwallex 審核：

- 網站 Logo
- 建立至少一個商品，並標有正確售價
- 建立頁腳的「聯絡資訊」頁面
- 建立頁腳的「關於我們」頁面，「關於我們」頁面中需有公司名稱和統編
- 建立頁腳的「隱私政策」頁面
- 建立頁腳的「服務條款」頁面

2. 待您的官網經 Airwallex 窗口評估完成，請繼續激活 Airwallex 收單功能。
- Airwallex 【後台路徑】收單
[[圖片說明：澳洲到台-金流設定]](https://www.cyberbiz.io/support/wp-content/uploads/Airwallex-前台-會員頁-收單-開啟收單功能01.png)

3. 選擇 CYBERBIZ 平台支援的金流。
[[圖片說明：澳洲到台-金流設定]](https://www.cyberbiz.io/support/wp-content/uploads/Airwallex-前台-會員頁-收單-選擇金流01.png)

4. 請依 Airwallex 頁面指示，完成激活流程。
📌 串接 CYBERBIZ 官網

1. 激活完成後，即可開通 Airwallex 收單功能。請於 Airwallex 後台取得API密鑰。
- Airwallex 【後台路徑】帳戶→「開發者」→「API密鑰」
[[圖片說明：澳洲到台-金流設定]](https://www.cyberbiz.io/support/wp-content/uploads/Airwallex-前台-帳戶-開發者-複製API密鑰01.png)

2. 進入 CYBERBIZ 後台，編輯 Airwallex 金流選項。
- CYBERBIZ 【後台路徑】金物流→「金流設定」
[[圖片說明：澳洲到台-金流設定]](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-金物流-金流設定-澳洲站啟用金流01.png)

3. 找到「啟用設定」區塊，將 Airwallex 「CLIENT ID」與「API密鑰」填入對應欄位。
[[圖片說明：澳洲到台-金流設定]](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-金物流-金流設定-澳洲站串接API密鑰01.png)

4. 依需求開啟官網的付款選項。
- 詳細金流手續費可參考 [Airwallex官方文件](https://www.airwallex.com/au/pricing)。
[[圖片說明：澳洲到台-金流設定]](https://www.cyberbiz.io/support/wp-content/uploads/EC-後台-金物流-金流設定-澳洲站開啟付款方式01.png)

## 📌 DNS 設定

1. 進入 [GoDaddy 後台](https://tw.godaddy.com/) ，登入您自己購買自有網域的帳號。 [[圖片說明：登入godaddy]](https://www.cyberbiz.io/support/wp-content/uploads/網域設定-Godaddy01.png)

2. 點選下方網域的「DNS」文字。
[[圖片說明：dns畫面]](https://www.cyberbiz.io/support/wp-content/uploads/網域設定-Godaddy02.png)

3. 分頁 「DNS 記錄」，並查看下方 CNAME 自有網域的設定，點選編輯。
[[圖片說明：dns管理]](https://www.cyberbiz.io/support/wp-content/uploads/網域設定-Godaddy03.png)

4. 設定 CNAME
【類型】: CNAME
【名稱】: www
【內容值】: 您在 CYBERBIZ 的網域(例 : xxx.cyberbiz.co. )
【TTL】: 1天
[[圖片說明：cname設定]](https://www.cyberbiz.io/support/wp-content/uploads/網域設定-Godaddy04.png)

5. 完成 DNS 設定。

- * *


---

## 📌 轉址設定

1. 選擇分頁「轉址」，並點選「網域」→「新增轉址」。 [[圖片說明：轉址]](https://www.cyberbiz.io/support/wp-content/uploads/網域設定-Godaddy05.png)

2. 填入您的網域並按下儲存
【輸入網址】: 填入您新增的網域(例 : www.網域名稱.com)
[[圖片說明：轉址網域]](https://www.cyberbiz.io/support/wp-content/uploads/網域設定-Godaddy06.png)

3. 完成設定畫面。
[[圖片說明：設定網域]](https://www.cyberbiz.io/support/wp-content/uploads/網域設定-Godaddy07.png)

4. 最後，進入 CYBERBIZ 後台，新增設定完的 www 網域。(例 : www.網域名稱.com)，並點選「總是將顧客重新導向到這裡」。
【後台路徑】管理中心→「網域管理」
[[圖片說明：完成設定]](https://www.cyberbiz.io/support/wp-content/uploads/網域設定-HiNet12.png)

# 購買電子票券
顧客購買電子票券的結帳流程說明。
{ .subtitle }

[:lucide-lock:{ title="適用方案" }](../../resources/conventions#適用方案) | PLUS 企業

## 操作流程

### 票券商品頁面結帳
電子票券商品頁面會顯示票券注意事項，提醒顧客使用辦法。

![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定05.png){ .screenshot }

### 購物車分拆
若購物車內同時包含票券商品與一般商品，系統會要求拆分不同購物車進行結帳。

![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定06.png){ .screenshot }

### 勾選同意
消費者需勾選 **同意電子票券條款** 選項，才能進行結帳。（購物車內的 **配送時段** 可忽略）。

![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定07.png){ .screenshot }

### 查看電子票券
消費者付款完成後，可在以下兩個地方查看電子票券 QR Code（核銷碼）。
#### 電子票券訂單查詢 (前台)

1. 網頁前台點選 **我的帳戶 > 電子票券訂單查詢**。    
2. 點擊 **顯示票券 QR code**，即可獲得 QR Code。

#### 電子票券確認信 (信箱)

!!! note "需設定會員電子郵件"
    為確保電子票券能成功寄送至會員信箱，請先確認 *顧客註冊設定* 中的「電子郵件」已設為必填。若會員未提供 Email，系統將無法寄送 QR Code。

1. 登入 CYBERBIZ 後台，前往 **管理中心 > 顧客註冊設定**。  
2.  **電子郵件**：請勾選「必填」。若會員未填寫 Email，系統將無法寄送 QR Code。

 ![](https://www.cyberbiz.io/support/wp-content/uploads/電子票券設定39.png){ .screenshot }

![](https://www.cyberbiz.io/support/wp-content/uploads/2021/12/電子票券設定10.png){ title="信箱畫面範例" }
